title: 3、非齐次线性方程组

## 一、求法解的判定


$\boldsymbol{A}_{m \times n} \boldsymbol{x}=\boldsymbol{b}$,$(A, \boldsymbol{b})$,称为增广矩阵:


$$\left \{  
\begin{align*}
 & 有解  \Leftrightarrow r(\boldsymbol{A}, \mathbf{b})=r(\boldsymbol{A})=r , 且  \left\{\begin{array}{l}r=n \Leftrightarrow \text { 唯一解} \\ r<n \Leftrightarrow \text { 无穷多解}\end{array}\right.  \\
 & 无解  \Leftrightarrow r(\boldsymbol{A}, \mathbf{b}) \neq r(\boldsymbol{A}) \Leftrightarrow r(\boldsymbol{A}, \mathbf{b})=r(\boldsymbol{A})+1 
\end{align*}\right.
$$




$$\left \{  
\begin{align*}
& \boldsymbol{A x}=\mathbf{b} \text { 有解，即 } x_{1} \boldsymbol{\alpha}_{1}+x_{2} \boldsymbol{\alpha}_{2}+\cdots+x_{n} \boldsymbol{\alpha}_{n}=\mathbf{b} \text { 有解 } \\
& \Leftrightarrow \text { 存在 } x_{1}, x_{2}, \cdots, x_{n} \text {, 使 } x_{1} \boldsymbol{\alpha}_{1}+x_{2} \boldsymbol{\alpha}_{2}+\cdots+x_{n} \boldsymbol{\alpha}_{n}=\mathbf{b} \\
& \Leftrightarrow \boldsymbol{b} \text { 可由 } \boldsymbol{\alpha}_{1}, \boldsymbol{\alpha}_{2}, \cdots, \boldsymbol{\alpha}_{n} \text { 线性表示 } \Leftrightarrow r\left(\boldsymbol{\alpha}_{1}, \boldsymbol{\alpha}_{2}, \cdots, \boldsymbol{\alpha}_{n}, \mathbf{b}\right)=r\left(\boldsymbol{\alpha}_{1}, \boldsymbol{\alpha}_{2}, \cdots, \boldsymbol{\alpha}_{n}\right)=r \\
& \text { 且 }\left\{\begin{array}{l}
r=n \text { 时，} \mathbf{b} \text { 可由 } \boldsymbol{\alpha}_{1}, \boldsymbol{\alpha}_{2}, \cdots, \boldsymbol{\alpha}_{n} \text { 唯一线性表示．} \\
r<n \text { 时，} \mathbf{b} \text { 可由 } \boldsymbol{\alpha}_{1}, \boldsymbol{\alpha}_{2}, \cdots, \boldsymbol{\alpha}_{n} \text { 的表示形式有无穷多种}
\end{array}\right. \\
& \Leftrightarrow r(\boldsymbol{A}, \mathbf{b})=r(\boldsymbol{A})=r \text {, 且 }\left\{\begin{array}{ll}
r=n \text { 时, } & \boldsymbol{A} \boldsymbol{x}=\mathbf{b} \text { 有唯一解 } \\
r<n \text { 时, } & \boldsymbol{A} \boldsymbol{x}=\mathbf{b} \text { 有无穷多解 }
\end{array}\right. \\
&  \\ \\
& \boldsymbol{A} \boldsymbol{x}=\mathbf{b} \text { 无解，即 } x_{1} \boldsymbol{\alpha}_{1}+x_{2} \boldsymbol{\alpha}_{2}+\cdots+x_{n} \boldsymbol{\alpha}_{n}=\mathbf{b} \text { 无解 } \\
& \begin{array}{l}
\Leftrightarrow \text { 不存在 } x_{1}, x_{2}, \cdots, x_{n}, \text { 使 } x_{1} \boldsymbol{\alpha}_{1}+x_{2} \boldsymbol{\alpha}_{2}+\cdots+x_{n} \boldsymbol{\alpha}_{n}=\mathbf{b} \\
\Leftrightarrow \mathbf{b} \text { 不可由 } \boldsymbol{\alpha}_{1}, \boldsymbol{\alpha}_{2}, \cdots, \boldsymbol{\alpha}_{n} \text { 线性表示 } \\
\Leftrightarrow r\left(\boldsymbol{\alpha}_{1}, \boldsymbol{\alpha}_{2}, \cdots, \boldsymbol{\alpha}_{n}, \mathbf{b}\right) \neq r\left(\boldsymbol{\alpha}_{1}, \boldsymbol{\alpha}_{2}, \cdots, \boldsymbol{\alpha}_{n}\right) \text { 即 } r\left(\boldsymbol{\alpha}_{1}, \boldsymbol{\alpha}_{2}, \cdots, \boldsymbol{\alpha}_{n}, \mathbf{b}\right)=r\left(\boldsymbol{\alpha}_{1}, \boldsymbol{\alpha}_{2}, \cdots, \boldsymbol{\alpha}_{n}\right)+1 \\
\Leftrightarrow r(\boldsymbol{A}, \mathbf{b}) \neq r(\boldsymbol{A}) \text { 即 } r(\boldsymbol{A}, \boldsymbol{b})=r(\boldsymbol{A})+1
\end{array} \\
\end{align*}\right.
$$


## 二、求法解的性质与结构


- 解的性质:


$$\begin{align*}
&\text { 性质1:设 } \boldsymbol{A} \boldsymbol{\eta}_{1}=\boldsymbol{b}, \boldsymbol{A} \boldsymbol{\eta}_{2}=\boldsymbol{b} \text { ，则 } \boldsymbol{\eta}_{1}-\boldsymbol{\eta}_{2} \text { 是 } \boldsymbol{A} \boldsymbol{x}=\boldsymbol{0} \text { 的解} (非齐次解－非齐次解=齐次解)\\
& \text { 性质2:设 } \boldsymbol{A} \boldsymbol{\eta}=\boldsymbol{b}, \boldsymbol{A} \boldsymbol{\xi}=\mathbf{0} \text { ，则 } \boldsymbol{\eta}+\boldsymbol{\xi} \text { 是 } \boldsymbol{A} \boldsymbol{x}=\boldsymbol{b} \text { 的解}(非齐次解+齐次解=非齐次解)
\end{align*}$$


- 通解:若$Ax=b$有无穷多解(非齐次通解=齐次通解+一个非齐次特解)。

$$
\begin{align*}
 & \text { 设 } \boldsymbol{\xi}_{1}, \boldsymbol{\xi}_{2}, \cdots, \boldsymbol{\xi}_{n-r(4)} \text { 是 } \boldsymbol{A x}=\mathbf{0} \text {（导出组）的任意一个基础解系 }\\
 & \boldsymbol{\eta} \text { 是 } \boldsymbol{A x}=\mathrm{b} \text { 的一个特解 }(特解：不含有任意常数的解)\\
 & \text { 则其通解为 } \boldsymbol{x}=k_{1} \boldsymbol{\xi}_{1}+k_{2} \boldsymbol{\xi}_{2}+\cdots+k_{n-r(A)} \boldsymbol{\xi}_{n-r(A)}+\boldsymbol{\eta}   \\
 & \text { 其中 } k_{1}, k_{2}, \cdots, k_{n-r(A)} \text { 是任意常数 } \\
\end{align*}
$$

## 三、求法



已知非齐次线性方程组 A x=b  ，求解步骤(两步)：

$$(\boldsymbol{A}, \mathbf{b}) \xrightarrow{\text { 初等行变换 }} \text { 行阶梯形矩阵 } \xrightarrow{\text { 初等行变换 }} 行最简形矩阵$$

$$
\begin{align*}
& 若  r(\boldsymbol{A}, \mathbf{b}) \neq r(\boldsymbol{A})  ，则  \boldsymbol{A} \boldsymbol{x}=\boldsymbol{b}  无解\\
& 若  r(\boldsymbol{A}, \mathbf{b})=r(\boldsymbol{A})=n  ，则  \boldsymbol{A} \boldsymbol{x}=\boldsymbol{b}  有唯一解，求唯一解\\
&若  r(\boldsymbol{A}, \mathbf{b})=r(\boldsymbol{A})<n  ，则  \boldsymbol{A} \boldsymbol{x}=\boldsymbol{b}  有无穷多解,解\boldsymbol{A} \boldsymbol{x}=\mathbf{0} （导出组）的基础解系\\
&\Longrightarrow 得  \boldsymbol{A} \boldsymbol{x}=\mathbf{0}  的通解,找Ax=b的一个特解,求非齐通=齐通十一个非齐特
\end{align*}
$$
