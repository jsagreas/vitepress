---
title: 2、相似矩阵与矩阵的对角化
---
- 相似矩阵概念：设$A_{n \times n}, B_{n \times n}$，如果存在可逆矩阵$P_{n \times n}$，使得$B=P^{-1} A P$，则称$A$和$B$相似，记作$A \sim B$，称$P^{-1} A P$是对$A$作相似变换。


- 相似矩阵性质：若$A \sim B$，则$r(A)=r(B) ;|A|=|B| ;\left|\lambda E_{n}-A\right|=\left|\lambda E_{n}-B\right| ; \operatorname{tr}(A)=\operatorname{tr}(B)$即方阵相似，则有相同的秩，行列式，特征值，迹。



- 矩阵的相似对角化概念：若矩阵$A_{n \times n}$与对角矩阵$\Lambda$相似，则称矩阵$A$可相似对角化；记作$A \sim \Lambda$，称$\Lambda$是$A$的相似标准形。


- 矩阵的相似对角化条件：

$$\left \{ \begin{align*}
& n 阶方阵 A  可对角化 \Leftrightarrow A  有 n  个线性无关的特征向量 \\
& 若 A有 n  个不同的特征值，则 A  一定可以相似对角化 \\
& A_{n \times n} \text { 与对角矩阵 } \Lambda \text { 相似 },\Leftrightarrow \text { 对应每一个 } n_{i} \text { 重特征值 } \lambda_{i}, \text { 均存在 } n_{i}  个线性无关的特征向量 \\
\end{align*}\right.$$
