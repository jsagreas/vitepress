## 一、曲面积分的概念与性质


求质量不均匀分布的曲面形物体质量的问题，可将曲面$\Sigma$分割成$n$小块： $\Delta S_{1}, \Delta S_{2}, \cdots, \Delta S_{i}, \cdots, \Delta S_{n}$，在每一小块曲面$\Delta S_{i}(i=1,2, \cdots, n)$上任取一点$\left(\xi_{i}, \eta_{i}, \zeta_{i}\right)$，做乘积$\rho(\xi_{i}, \eta_{i}, \zeta_{i}) \Delta S_{i}$ ，（$\Delta S_{i}$在这里表示第$i$小块曲面的面积），再做和式$\sum_{i=1}^{n} \rho\left(\xi_{i}, \eta_{i}, \zeta_{i}\right) \Delta S_{i}$，得曲面形物体质量的近似值。令分割无限变细，即小块曲面$\Delta S_{1}, \Delta S_{2}, \cdots, \Delta S_{n}$的直径（曲面的直径，是指曲面上任意两点间距离的最大值）的最大值$\lambda \rightarrow 0$，若上述和式的极限存在，则定义此极限值为该曲面形物体的质量：

$$\lim _{\lambda \rightarrow 0} \sum_{i=1}^{n} \rho\left(\xi_{i}, \eta_{i}, \zeta_{i}\right) \Delta S_{i}$$

**对面积的曲面积分定义**

定义：设$\Sigma$是一块光滑曲面（光滑曲面，是指曲面上各点处都有切平面，且当点在曲面上连续移动时，切平面也在连续移动），函数 $f(x, y, z)$在$\Sigma$上有界。把  $\Sigma$任意分成 $n$小块：$\Delta S_{1}, \Delta S_{2}, \cdots, \Delta S_{i}, \cdots, \Delta S_{n}$，（$\Delta S_{i}$ 既表示第  $i$  块小曲面，同时又代表这块小曲面的面积），在$\Delta S_{i}$ 上任取一点$\left(\xi_{i}, \eta_{i}, \zeta_{i}\right)$，做乘积 $f\left(\xi_{i}, \eta_{i}, \zeta_{i}\right) \Delta S_{i}(i=1,2, \cdots, n)$，并做如下和式，其中 $f(x, y, z)$叫作被积函数， $\Sigma$叫作积分曲面。

$$\sum_{i=1}^{n} f\left(\xi_{i}, \eta_{i}, \zeta_{i}\right) \Delta S_{i}$$


如果当各小块曲面$\Delta S_{i}(i=1,2, \cdots, n)$的直径的最大值$\lambda \rightarrow 0$时，这个和式的极限存在，则称此极限值为函数$f(x, y, z)$在曲面  $\Sigma$上对面积的曲面积分或第一类曲面积分，记作$\iint_{\Sigma} f(x, y, z) \mathrm{d} S$，即：

$$\iint_{\Sigma} f(x, y, z) \mathrm{d} S=\lim _{\lambda \rightarrow 0} \sum_{i=1}^{n} f\left(\xi_{i}, \eta_{i}, \zeta_{i}\right) \Delta S_{i}$$

如果$\Sigma$是分片光滑的，我们规定函数$f(x, y, z)$在$\Sigma$上对面积的曲面积分等于该函数在各块光滑曲面上对面积的曲面积分之和。例如，设$\Sigma$可以分成两个光滑曲面$\Sigma_{1}, \Sigma_{2}$（记作$\Sigma=\Sigma_{1}+\Sigma_{2} ）$，则规定：
$$\iint_{\Sigma} f(x, y, z) \mathrm{d} S=\iint_{\Sigma_{1}} f(x, y, z) \mathrm{d} S+\iint_{\Sigma_{2}} f(x, y, z) \mathrm{d} S$$


**对面积的曲面积分有下列性质**

与对弧长的曲线积分类似，对面积的曲面积分有下列性质（假设讨论的曲面积分存在）：

$$\iint_{\Sigma}(f(x, y, z) \pm g(x, y, z)) \mathrm{d} S=\iint_{\Sigma_1} f(x, y, z) \mathrm{d} S \pm \iint_{\Sigma_2} g(x, y, z) \mathrm{d} S $$

如果$k$ 为常数，则：
$$\iint_{\Sigma} k f(x, y, z) \mathrm{d} S=k \iint_{\Sigma} f(x, y, z) \mathrm{d} S$$


如果 $k_{1}, k_{2}$是任意常数，则有：

$$\iint_{\Sigma}\left(k_{1} f(x, y, z)+k_{2} g(x, y, z)\right) \mathrm{d} S=k_{1} \iint_{\Sigma_1} f(x, y, z) \mathrm{d} S+k_{2} \iint_{\Sigma_2} g(x, y, z) \mathrm{d} S$$


若积分曲面 $\Sigma$被分成两块曲面$\Sigma_{1}, \Sigma_{2}\left(\Sigma=\Sigma_{1}+\Sigma_{2}\right)$，则有：

$$\iint_{\Sigma} f(x, y, z) \mathrm{d} S=\iint_{\Sigma_{1}} f(x, y, z) \mathrm{d} S+\iint_{\Sigma_{2}} f(x, y, z) \mathrm{d} S $$

当 $f(x, y, z)=1$时，其中$|\Sigma|$为曲面$\Sigma$的面积，则有：

$$\iint_{\Sigma} \mathrm{d} S=|\Sigma|$$





## 二、对面积的曲面积分的计算


设积分曲面 $\Sigma$由方程 $z=z(x, y)$确定，曲面$\Sigma$在$O x y$平面上的投影区域为$D_{x y}$，函数 $z(x, y)$在$D_{x y}$上具有连续偏导数，被积函数$f(x, y, z)$在曲面 $\Sigma$上连续，注意，在曲面积分$\iint_{\Sigma} f(x, y, z) \mathrm{d} S$中，积分变量 $x, y, z$在曲面 $\Sigma$ 上，所以满足曲面$\Sigma$的方程，即$z=z(x, y) ; \mathrm{d} S$是由小曲面 $\Delta S_{i}$ 的面积转化来的，可知曲面 $\Sigma$的面积微元为：

$$\mathrm{d} S=\sqrt{1+z_{x}^{2}+z_{y}^{2}} \mathrm{~d} x \mathrm{~d} y$$


于是有下面的计算公式：

$$\iint_{\Sigma} f(x, y, z) \mathrm{d} S=\iint_{D_{x y}} f(x, y, z(x, y)) \sqrt{1+z_{x}^{2}+z_{y}^{2}} \mathrm{~d} x \mathrm{~d} y $$
