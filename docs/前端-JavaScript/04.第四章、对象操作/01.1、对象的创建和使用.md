## 1. 🎯 什么是对象


### 1.1 对象的概念


**🔸 通俗理解**
```
对象就是一个"信息包"，里面可以放各种相关的数据和功能
就像一个人：有姓名、年龄、身高等属性，还会说话、走路等方法

现实例子 → JavaScript对象：
人 → person对象
- 姓名 → name属性  
- 年龄 → age属性
- 说话 → speak方法
```

### 1.2 对象的组成


**🔧 核心组成部分**
```javascript
// 对象 = 属性 + 方法
let person = {
    // 属性：存储数据
    name: "张三",
    age: 25,
    
    // 方法：存储功能
    sayHello: function() {
        console.log("你好！");
    }
};
```

**💡 为什么需要对象**
```
没有对象的时候：
let userName = "张三";
let userAge = 25;
let userCity = "北京";
// 数据分散，不好管理

有了对象之后：
let user = {
    name: "张三",
    age: 25,
    city: "北京"
};
// 数据集中，便于管理
```

## 2. 🏗️ 对象的创建方法


### 2.1 对象字面量（最常用）


**🔸 基本语法**
```javascript
// 创建空对象
let obj = {};

// 创建带属性的对象
let student = {
    name: "小明",
    age: 18,
    grade: "高三"
};
```

**📋 实际应用示例**
```javascript
// 用户信息对象
let user = {
    id: 1001,
    username: "admin",
    email: "admin@example.com",
    isActive: true
};

// 商品信息对象
let product = {
    title: "iPhone 15",
    price: 5999,
    category: "手机",
    inStock: true
};
```

### 2.2 构造函数创建（了解即可）


```javascript
// 使用 new Object()
let person = new Object();
person.name = "李四";
person.age = 30;

// 等价于对象字面量，但不推荐
let person2 = {
    name: "李四",
    age: 30
};
```

## 3. 🔍 访问对象属性


### 3.1 点语法访问（推荐）


**🔸 基本用法**
```javascript
let user = {
    name: "张三",
    age: 25,
    city: "北京"
};

// 读取属性
console.log(user.name);    // "张三"
console.log(user.age);     // 25

// 调用方法
user.sayHello();
```

### 3.2 括号语法访问


**🔸 特殊情况使用**
```javascript
let user = {
    name: "张三",
    "user-age": 25,    // 属性名有特殊字符
    "first name": "张"  // 属性名有空格
};

// 只能用括号语法访问
console.log(user["user-age"]);     // 25
console.log(user["first name"]);   // "张"

// 变量作为属性名
let prop = "name";
console.log(user[prop]);           // "张三"
```

### 3.3 两种访问方式对比


| 访问方式 | **适用场景** | **优缺点** |
|---------|-------------|-----------|
| 🔸 **点语法** | `普通属性名` | `简洁易读，推荐使用` |
| 🔸 **括号语法** | `特殊属性名、动态属性` | `灵活但略显复杂` |

```javascript
let obj = { name: "测试", count: 100 };

// 推荐：点语法
console.log(obj.name);

// 必须：括号语法
let key = "count";
console.log(obj[key]);  // 动态访问
```

## 4. ⚙️ 操作对象属性


### 4.1 添加和修改属性


**🔸 动态添加属性**
```javascript
let user = {};

// 添加属性
user.name = "王五";
user.age = 28;
user["email"] = "wang@example.com";

console.log(user);
// 结果：{ name: "王五", age: 28, email: "wang@example.com" }
```

**🔸 修改现有属性**
```javascript
let product = {
    name: "手机",
    price: 2999
};

// 修改属性值
product.price = 2599;    // 降价了
product.discount = 0.8;  // 添加折扣属性

console.log(product);
// 结果：{ name: "手机", price: 2599, discount: 0.8 }
```

### 4.2 删除属性


**🔸 使用 delete 关键字**
```javascript
let user = {
    name: "赵六",
    age: 30,
    password: "123456"
};

// 删除敏感信息
delete user.password;

console.log(user);
// 结果：{ name: "赵六", age: 30 }
```

### 4.3 检查属性是否存在


**🔸 常用检查方法**
```javascript
let user = {
    name: "孙七",
    age: 25
};

// 方法1：in 操作符
console.log("name" in user);     // true
console.log("email" in user);    // false

// 方法2：hasOwnProperty()
console.log(user.hasOwnProperty("name"));  // true
console.log(user.hasOwnProperty("email")); // false

// 方法3：直接判断 undefined
console.log(user.name !== undefined);      // true
console.log(user.email !== undefined);     // false
```

## 5. 🔄 对象的遍历方法


### 5.1 for...in 循环（最常用）


**🔸 遍历所有属性**
```javascript
let student = {
    name: "小红",
    age: 20,
    major: "计算机",
    gpa: 3.8
};

// 遍历对象的所有属性
for (let key in student) {
    console.log(key + ": " + student[key]);
}

/*
输出：
name: 小红
age: 20
major: 计算机
gpa: 3.8
*/
```

### 5.2 Object 方法遍历


**🔸 获取键名数组**
```javascript
let user = {
    id: 1,
    name: "admin",
    role: "管理员"
};

// 获取所有键名
let keys = Object.keys(user);
console.log(keys);  // ["id", "name", "role"]

// 遍历键名
keys.forEach(key => {
    console.log(key + ": " + user[key]);
});
```

**🔸 获取值数组**
```javascript
// 获取所有值
let values = Object.values(user);
console.log(values);  // [1, "admin", "管理员"]
```

**🔸 获取键值对数组**
```javascript
// 获取键值对
let entries = Object.entries(user);
console.log(entries);  
// [["id", 1], ["name", "admin"], ["role", "管理员"]]

// 遍历键值对
entries.forEach(([key, value]) => {
    console.log(`${key}: ${value}`);
});
```

## 6. 💡 实用对象技巧


### 6.1 对象属性简写


**🔸 ES6 简写语法**
```javascript
let name = "张三";
let age = 25;

// 传统写法
let user1 = {
    name: name,
    age: age
};

// 简写（属性名和变量名相同时）
let user2 = {
    name,  // 等价于 name: name
    age    // 等价于 age: age
};
```

### 6.2 动态属性名


**🔸 计算属性名**
```javascript
let fieldName = "userEmail";
let fieldValue = "test@example.com";

let user = {
    name: "测试用户",
    [fieldName]: fieldValue  // 动态属性名
};

console.log(user);
// 结果：{ name: "测试用户", userEmail: "test@example.com" }
```

### 6.3 对象方法简写


**🔸 方法定义简写**
```javascript
let calculator = {
    // 传统方法定义
    add: function(a, b) {
        return a + b;
    },
    
    // ES6 简写方法
    subtract(a, b) {
        return a - b;
    },
    
    // 箭头函数（注意 this 指向问题）
    multiply: (a, b) => a * b
};

console.log(calculator.add(5, 3));      // 8
console.log(calculator.subtract(5, 3)); // 2
console.log(calculator.multiply(5, 3));  // 15
```

### 6.4 对象复制


**🔸 浅拷贝方法**
```javascript
let original = {
    name: "原始对象",
    age: 25
};

// 方法1：Object.assign()
let copy1 = Object.assign({}, original);

// 方法2：扩展运算符（推荐）
let copy2 = { ...original };

// 方法3：手动复制
let copy3 = {};
for (let key in original) {
    copy3[key] = original[key];
}
```

## 7. 📋 核心要点总结


### 7.1 必须掌握的基本概念


```
🔸 对象本质：用来存储相关数据和功能的容器
🔸 创建对象：优先使用对象字面量 {}
🔸 访问属性：一般用点语法，特殊情况用括号语法
🔸 动态操作：可以随时增删改属性
🔸 遍历对象：for...in 是最常用的方法
```

### 7.2 关键语法对比


| 操作类型 | **语法** | **适用场景** |
|---------|---------|-------------|
| 🔸 **创建** | `let obj = {}` | `最常用的创建方式` |
| 🔸 **访问** | `obj.prop` 或 `obj["prop"]` | `读取属性值` |
| 🔸 **修改** | `obj.prop = value` | `设置属性值` |
| 🔸 **删除** | `delete obj.prop` | `移除不需要的属性` |
| 🔸 **检查** | `"prop" in obj` | `判断属性是否存在` |
| 🔸 **遍历** | `for (let key in obj)` | `处理所有属性` |

### 7.3 实际开发中的应用


**🎯 常见使用场景**
```javascript
// 1. 用户数据管理
let currentUser = {
    id: 1001,
    username: "admin",
    permissions: ["read", "write"],
    lastLogin: "2025-07-28"
};

// 2. 配置信息
let appConfig = {
    theme: "dark",
    language: "zh-CN",
    autoSave: true,
    maxFileSize: 5 * 1024 * 1024  // 5MB
};

// 3. API 响应数据
let apiResponse = {
    code: 200,
    message: "success",
    data: {
        users: [...],
        total: 150
    }
};
```

### 7.4 最佳实践建议


**🔧 开发建议**
```
属性命名：
✅ 使用有意义的名字 (userName 而不是 u)
✅ 采用小驼峰命名 (firstName 而不是 first_name)
✅ 避免使用保留字

访问方式：
✅ 优先使用点语法 (obj.prop)
✅ 动态属性时使用括号语法 (obj[key])

对象操作：
✅ 检查属性存在性再使用
✅ 使用 Object.keys() 等方法遍历
✅ 注意对象是引用类型
```

**核心记忆**：
- 对象是存放相关数据的容器
- 用对象字面量 {} 创建最简单
- 点语法访问最常用，括号语法处理特殊情况
- for...in 遍历是必备技能
- 对象让数据管理更清晰有序
