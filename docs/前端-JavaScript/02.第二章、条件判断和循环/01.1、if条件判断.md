## 1. 🎯 if语句基础


### 1.1 基本概念

**if语句**就是让程序会"思考"，根据条件决定要不要执行某段代码。

```
简单理解：
如果(条件成立) {
  执行这里的代码
}
```

### 1.2 基本语法

```javascript
if (条件) {
  // 条件为真时执行的代码
}
```

**💡 实际例子**
```javascript
let age = 18;

if (age >= 18) {
  console.log("可以投票了！");
}

// 输出：可以投票了！
```

### 1.3 条件判断示意图

```
程序执行流程：
    开始
     ↓
  [条件判断]
   /     \
  是     否
  ↓      ↓
执行代码  跳过
  ↓      ↓
    继续
```

## 2. 🔄 if...else语句


### 2.1 核心概念

**if...else**提供了"二选一"的逻辑：要么执行A，要么执行B。

### 2.2 基本语法

```javascript
if (条件) {
  // 条件为真时执行
} else {
  // 条件为假时执行
}
```

**💡 实际例子**
```javascript
let weather = "晴天";

if (weather === "下雨") {
  console.log("带伞出门");
} else {
  console.log("正常出门");
}

// 输出：正常出门
```

### 2.3 常用判断场景

| 场景 | **条件** | **真时执行** | **假时执行** |
|------|---------|-------------|-------------|
| 🎂 **年龄判断** | `age >= 18` | `"成年人"` | `"未成年"` |
| 🌡️ **温度判断** | `temp > 30` | `"太热了"` | `"温度正常"` |
| 💰 **价格判断** | `price > 100` | `"太贵了"` | `"可以买"` |
| 🔒 **登录判断** | `isLoggedIn` | `"进入主页"` | `"跳转登录"` |

## 3. 🎛️ 多重条件判断


### 3.1 if...else if...else结构

当有多个条件需要判断时，用这个结构最清晰。

```javascript
if (条件1) {
  // 条件1为真
} else if (条件2) {
  // 条件2为真  
} else if (条件3) {
  // 条件3为真
} else {
  // 所有条件都为假
}
```

### 3.2 成绩等级判断示例

```javascript
let score = 85;

if (score >= 90) {
  console.log("优秀");
} else if (score >= 80) {
  console.log("良好");
} else if (score >= 60) {
  console.log("及格");
} else {
  console.log("不及格");
}

// 输出：良好
```

### 3.3 判断流程图

```
     score >= 90？
        /    \
      是      否
      ↓       ↓
    优秀   score >= 80？
              /    \
            是      否
            ↓       ↓
          良好   score >= 60？
                    /    \
                  是      否
                  ↓       ↓
                及格    不及格
```

### 3.4 实用判断场景

```javascript
// 🌤️ 天气建议
let temp = 25;

if (temp > 30) {
  console.log("太热了，开空调");
} else if (temp > 20) {
  console.log("温度刚好，很舒适");
} else if (temp > 10) {
  console.log("有点凉，加件衣服");
} else {
  console.log("太冷了，多穿点");
}

// 🕐 时间问候
let hour = 14;

if (hour < 6) {
  console.log("夜深了，早点休息");
} else if (hour < 12) {
  console.log("早上好！");
} else if (hour < 18) {
  console.log("下午好！");
} else {
  console.log("晚上好！");
}
```

## 4. ⚡ 三元运算符


### 4.1 简化的条件判断

**三元运算符**是if...else的简化写法，适合简单的二选一场景。

```javascript
条件 ? 真时的值 : 假时的值
```

### 4.2 基本使用

```javascript
// 传统if...else写法
let age = 20;
let status;
if (age >= 18) {
  status = "成年人";
} else {
  status = "未成年";
}

// 三元运算符写法（更简洁）
let status = age >= 18 ? "成年人" : "未成年";
console.log(status); // 成年人
```

### 4.3 实用场景

```javascript
// 🛒 购物折扣
let totalPrice = 150;
let finalPrice = totalPrice > 100 ? totalPrice * 0.9 : totalPrice;
console.log(`最终价格：${finalPrice}`); // 最终价格：135

// 🎮 游戏状态
let score = 85;
let result = score >= 60 ? "过关" : "失败";
console.log(result); // 过关

// 🌟 用户等级
let points = 1200;
let level = points >= 1000 ? "VIP" : "普通用户";
console.log(level); // VIP
```

### 4.4 何时使用三元运算符


| 场景 | **推荐用法** | **原因** |
|------|-------------|---------|
| ✅ **简单二选一** | `三元运算符` | `代码简洁，一目了然` |
| ✅ **赋值操作** | `三元运算符` | `直接赋值，不需要中间变量` |
| ❌ **复杂逻辑** | `if...else` | `可读性更好，便于调试` |
| ❌ **多重判断** | `if...else if` | `结构清晰，易于维护` |

## 5. 🔍 真值和假值


### 5.1 JavaScript中的真假判断

在JavaScript中，条件判断不只是`true`和`false`，任何值都可以被判断为真或假。

### 5.2 假值（Falsy）- 记住这6个

这些值在条件判断中会被当作`false`：
```javascript
// 🚫 6个假值（重要！）
false      // 布尔假值
0          // 数字零
""         // 空字符串
null       // 空值
undefined  // 未定义
NaN        // 非数字
```

### 5.3 真值（Truthy）- 除假值外都是真值

```javascript
// ✅ 常见真值
true       // 布尔真值
1          // 非零数字
"hello"    // 非空字符串
[]         // 空数组（重要！）
{}         // 空对象（重要！）
"0"        // 字符串"0"
" "        // 空格字符串
```

### 5.4 实际应用示例

```javascript
// 🔍 检查用户输入
let username = "";

if (username) {
  console.log(`欢迎，${username}！`);
} else {
  console.log("请输入用户名");
}
// 输出：请输入用户名

// 🔍 检查数组是否有内容
let items = [];

if (items.length) {
  console.log("有商品");
} else {
  console.log("购物车为空");
}
// 输出：购物车为空

// 🔍 检查对象属性
let user = { name: "小明" };

if (user.age) {
  console.log(`年龄：${user.age}`);
} else {
  console.log("年龄未填写");
}
// 输出：年龄未填写
```

### 5.5 常见陷阱避免

```javascript
// ⚠️ 注意：这些都是真值！
if ([]) {
  console.log("空数组是真值！"); // 会执行
}

if ({}) {
  console.log("空对象是真值！"); // 会执行
}

if ("0") {
  console.log("字符串'0'是真值！"); // 会执行
}

// ✅ 正确的检查方式
let arr = [];
if (arr.length > 0) {
  console.log("数组有内容");
}

let obj = {};
if (Object.keys(obj).length > 0) {
  console.log("对象有属性");
}
```

## 6. 📋 实用技巧总结


### 6.1 必须掌握的核心要点

```
🔸 if语句：最基础的条件判断，必须熟练
🔸 if...else：二选一逻辑，日常开发最常用
🔸 多重if：复杂条件判断，要注意顺序
🔸 三元运算符：简化代码，适合简单场景
🔸 真假值：理解JavaScript的真假判断规则
```

### 6.2 实战开发建议


**🔹 代码可读性优先**
```javascript
// ❌ 不推荐：过于复杂
let result = age >= 18 ? (score >= 60 ? (level > 5 ? "A" : "B") : "C") : "D";

// ✅ 推荐：清晰易懂
let result;
if (age >= 18) {
  if (score >= 60) {
    result = level > 5 ? "A" : "B";
  } else {
    result = "C";
  }
} else {
  result = "D";
}
```

**🔹 条件判断的最佳实践**
```javascript
// ✅ 提前返回，减少嵌套
function checkUser(user) {
  if (!user) {
    return "用户不存在";
  }
  
  if (!user.name) {
    return "用户名不能为空";
  }
  
  if (user.age < 18) {
    return "年龄不足";
  }
  
  return "验证通过";
}

// ✅ 使用常量提高可读性
const ADULT_AGE = 18;
const PASS_SCORE = 60;

if (age >= ADULT_AGE && score >= PASS_SCORE) {
  console.log("符合条件");
}
```

### 6.3 常见应用场景总结


| 应用场景 | **推荐写法** | **注意要点** |
|---------|-------------|-------------|
| 🔐 **登录验证** | `if (username && password)` | `利用真假值特性` |
| 🛒 **购物判断** | `if...else if` | `多个价格区间判断` |
| 🎮 **游戏状态** | `三元运算符` | `简单的成功失败判断` |
| 📝 **表单验证** | `提前返回模式` | `减少嵌套，提高可读性` |

**核心记忆**：
- if是程序的"大脑"，让代码会思考
- 简单用三元，复杂用if...else  
- 记住6个假值，其他都是真值
- 代码清晰比简洁更重要
