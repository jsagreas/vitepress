## 1. 🚀 高等数学方法概述


### 1.1 什么是高等数学与应用类方法


**🔸 现代数学的工具箱**
高等数学与应用类方法是处理**复杂现实问题**的高级数学工具，就像从简单的螺丝刀升级到多功能电动工具。

```
工具进化类比：
🔧 基础工具：加减乘除 = 手工螺丝刀
📐 中级工具：函数分析 = 电动螺丝刀  
🚀 高级工具：矩阵线代 = 多功能电动工具箱

解决问题的复杂度：
单变量 → 多变量 → 向量空间 → 抽象空间
```

**🔸 应用导向的特点**
这些方法不仅仅是纯数学理论，更重要的是它们的**实际应用价值**，能够解决工程、科学、经济等领域的实际问题。

### 1.2 高等方法的核心特征


**⚡ 主要特点**

| **计算密集** | `通常需要计算机辅助` | `处理大规模实际问题` |
| **跨学科性** | `广泛应用于各个领域` | `推动科技进步` |

## 2. 📊 矩阵与线性代数法


### 2.1 矩阵方法的基本思想


**🔸 什么是矩阵方法**
矩阵方法将**多个变量的线性关系**用矩阵形式表示，就像用表格整理复杂的数据关系。

```
矩阵的直观理解：
📋 数据表格：学生成绩表、财务报表
🗺️ 网络关系：社交网络、交通网络
🎮 图像处理：像素矩阵、变换矩阵
🧮 方程组：多元线性方程组

数学表达：
线性方程组：{ a₁₁x₁ + a₁₂x₂ + ... = b₁
            { a₂₁x₁ + a₂₂x₂ + ... = b₂
            { ...
矩阵形式：AX = B
```

### 2.2 基本矩阵运算


**🔢 核心运算规则**

```
矩阵加法：对应元素相加
[a b] + [e f] = [a+e b+f]
[c d]   [g h]   [c+g d+h]

矩阵乘法：行乘列求和
[a b] × [e f] = [ae+bg af+bh]
[c d]   [g h]   [ce+dg cf+dh]

逆矩阵：AA⁻¹ = I（单位矩阵）
对于2×2矩阵：A⁻¹ = (1/det(A)) × [d -b]
                                [-c  a]
```

### 2.3 线性方程组求解


**⚖️ 高斯消元法**

```
例：解方程组 { 2x + y - z = 1
             { x - y + 2z = 4  
             { 3x + 2y + z = 7

增广矩阵：
[2  1 -1 | 1]
[1 -1  2 | 4]
[3  2  1 | 7]

消元过程：
第一步：第1行÷2，得主元1
[1  0.5 -0.5 | 0.5]
[1  -1    2  | 4  ]
[3   2    1  | 7  ]

第二步：消除第1列
[1  0.5 -0.5 | 0.5]
[0 -1.5  2.5 | 3.5]
[0  0.5  2.5 | 5.5]

继续消元得到：
x = 1, y = -1, z = 2
```

### 2.4 特征值与特征向量


**🎯 特征值分析的意义**

```
特征值问题：Av = λv
其中 λ 是特征值，v 是特征向量

物理意义：
- 特征向量：系统的"天然振动方向"
- 特征值：对应方向的"放大倍数"

应用实例：
🏗️ 结构分析：建筑物的固有频率
📊 数据分析：主成分分析（PCA）
🌐 网络分析：PageRank算法
🎵 信号处理：频谱分析
```

### 2.5 矩阵方法的现代应用


**🤖 机器学习中的线性代数**

```
线性回归：找到最佳拟合直线
问题：给定数据点 (x₁,y₁), (x₂,y₂), ..., (xₙ,yₙ)
目标：找到 y = ax + b 使误差最小

矩阵表示：
X = [x₁ 1]    Y = [y₁]    θ = [a]
    [x₂ 1]        [y₂]        [b]
    [... ]        [..]
    [xₙ 1]        [yₙ]

最小二乘解：θ = (X^T X)⁻¹ X^T Y

这个公式背后是线性代数的完整理论！
```

**📊 图像处理中的矩阵变换**

```
图像变换矩阵：
旋转：[cos θ  -sin θ]
     [sin θ   cos θ]

缩放：[sx  0 ]
     [0   sy]

平移：[1  0  tx]
     [0  1  ty]  
     [0  0  1 ]

复合变换：将多个变换矩阵相乘
结果变换 = 变换3 × 变换2 × 变换1
```

## 3. 🎯 拉格朗日乘数法


### 3.1 拉格朗日乘数法的核心思想


**🔸 约束优化问题**
拉格朗日乘数法解决的是在**约束条件下寻找最优解**的问题，就像在有限的预算下买到最满意的商品。

```
约束优化的生活例子：
💰 预算限制：在有限预算下最大化效用
⏰ 时间限制：在有限时间内完成最多任务
🌍 资源限制：在有限资源下最大化产出
🏠 空间限制：在有限空间内最优化布局

数学表达：
目标函数：max/min f(x,y)
约束条件：g(x,y) = 0
```

### 3.2 拉格朗日函数的构造


**📐 基本原理**

```
拉格朗日函数：
L(x,y,λ) = f(x,y) + λg(x,y)

其中：
- f(x,y)：目标函数
- g(x,y) = 0：约束条件
- λ：拉格朗日乘数

最优条件：
∂L/∂x = ∂f/∂x + λ∂g/∂x = 0
∂L/∂y = ∂f/∂y + λ∂g/∂y = 0  
∂L/∂λ = g(x,y) = 0

解这个方程组得到最优解
```

### 3.3 经典应用案例


**📦 几何优化问题**

```
问题：在周长固定的所有矩形中，哪个面积最大？

设定：
目标函数：面积 S = xy（要最大化）
约束条件：周长 2x + 2y = P（固定值）

求解：
约束条件改写：g(x,y) = 2x + 2y - P = 0
拉格朗日函数：L = xy + λ(2x + 2y - P)

求偏导：
∂L/∂x = y + 2λ = 0  →  y = -2λ
∂L/∂y = x + 2λ = 0  →  x = -2λ  
∂L/∂λ = 2x + 2y - P = 0

结论：x = y = P/4
即：正方形的面积最大！
```

**🏭 生产优化问题**

```
问题：工厂用资金K和劳动L生产产品，已知生产函数 Q = L^a K^b
在预算约束下如何配置资源使产量最大？

设定：
目标函数：Q = L^a K^b（最大化产量）
约束条件：wL + rK = B（预算约束）
其中w是工资率，r是资本成本率，B是总预算

拉格朗日函数：
L = L^a K^b + λ(B - wL - rK)

最优条件导出：
L/K = (a/b) × (r/w)

这就是著名的"要素替代率"公式！
```

### 3.4 多约束条件的推广


**🔗 多个约束的处理**

```
一般形式：
目标函数：f(x₁,x₂,...,xₙ)
约束条件：g₁(x₁,x₂,...,xₙ) = 0
         g₂(x₁,x₂,...,xₙ) = 0
         ...
         gₘ(x₁,x₂,...,xₙ) = 0

拉格朗日函数：
L = f(x) + λ₁g₁(x) + λ₂g₂(x) + ... + λₘgₘ(x)

最优条件：
∇f + λ₁∇g₁ + λ₂∇g₂ + ... + λₘ∇gₘ = 0
g₁(x) = 0, g₂(x) = 0, ..., gₘ(x) = 0

这给出 n+m 个方程，求解 n+m 个未知数
```

## 4. 🎨 变分法


### 4.1 变分法的基本概念


**🔸 函数空间中的优化**
变分法处理的是**函数的函数**的极值问题，就像在所有可能的路径中找到最优路径。

```
变分法的直观理解：
🛣️ 最短路径：所有连接两点的曲线中哪条最短？
💡 光的传播：光为什么选择特定的传播路径？
🏗️ 结构设计：什么形状的梁承载能力最强？
🌊 物理原理：自然界为什么遵循"最小作用量原理"？

数学抽象：
在所有满足条件的函数 y(x) 中，
找到使泛函 I[y] = ∫F(x,y,y')dx 达到极值的函数
```

### 4.2 欧拉-拉格朗日方程


**📐 变分法的核心工具**

```
标准变分问题：
求函数 y(x) 使泛函 I[y] = ∫[a,b] F(x,y,y')dx 达到极值
边界条件：y(a) = A, y(b) = B

欧拉-拉格朗日方程：
∂F/∂y - d/dx(∂F/∂y') = 0

这个微分方程的解就是使泛函达到极值的函数！
```

### 4.3 经典变分问题


**🔗 最速降线问题**

```
问题：一个质点从A点滑到B点，什么路径用时最短？

分析：
设路径方程为 y = y(x)
根据能量守恒：v = √(2gy)
弧长微元：ds = √(1 + y'²)dx
时间微元：dt = ds/v = √(1 + y'²)/(√(2gy))dx

总时间：T = ∫√(1 + y'²)/(√(2gy))dx

应用欧拉-拉格朗日方程求解得到：
最优路径是摆线（cycloid）！

令人惊奇的是：摆线比直线更快！
```

**⚡ 费马原理的数学表述**

```
费马原理：光总是选择光程最短的路径

数学表达：
光程：S = ∫n(s)ds，其中n(s)是折射率
变分问题：δS = 0

应用：
- 均匀介质：光走直线
- 分层介质：光走曲线（如海市蜃楼）
- 光纤：光被约束在纤芯中传播

这解释了斯涅耳定律：n₁sinθ₁ = n₂sinθ₂
```

### 4.4 现代变分法应用


**🤖 机器学习中的变分方法**

```
变分自编码器（VAE）：
目标：学习数据的概率分布

变分下界：
L = E[log p(x|z)] - KL(q(z|x)||p(z))

其中：
- 第一项：重构误差
- 第二项：KL散度（正则化项）

这是经典变分法在深度学习中的应用！
```

**🏗️ 工程结构优化**

```
结构拓扑优化：
目标：在给定材料量下设计最强结构

变分表述：
最小化：柔顺性 ∫u^T f dV
约束：材料体积 ∫ρ dV ≤ V₀

解决方法：
拉格朗日乘数 + 有限元方法
得到最优的材料分布ρ(x,y,z)

结果：类似骨骼结构的仿生设计！
```

## 5. 🌐 图论方法


### 5.1 图论的基本概念


**🔸 什么是图论**
图论研究的是**节点（顶点）**和**边**构成的网络结构，就像研究朋友圈的社交网络。

```
图的基本组成：
🔵 顶点（Vertex）：网络中的节点
➡️ 边（Edge）：连接两个顶点的线
🔢 权重（Weight）：边上的数值（距离、成本等）

现实中的图：
🗺️ 交通网络：城市是顶点，道路是边
👥 社交网络：人是顶点，关系是边
🌐 互联网：网页是顶点，链接是边
🧬 分子结构：原子是顶点，化学键是边
```

### 5.2 图的基本性质


**📊 图的分类与特征**

| 图的类型 | 特征 | 应用场景 |
|----------|------|----------|
| **无向图** | `边没有方向` | `朋友关系、道路网络` |
| **有向图** | `边有方向` | `网页链接、食物链` |
| **加权图** | `边有权重` | `距离网络、成本网络` |
| **完全图** | `任意两点都相连` | `理论分析、最坏情况` |
| **树** | `连通无环图` | `家族关系、决策树` |
| **平面图** | `边不相交` | `电路设计、地图着色` |

### 5.3 最短路径算法


**🚗 Dijkstra算法**

```
问题：在加权图中找从起点到所有其他点的最短路径

算法思想：
1. 从起点开始，距离设为0
2. 选择当前距离最小的未访问节点
3. 更新其所有邻居的距离
4. 标记该节点为已访问
5. 重复直到所有节点都访问过

伪代码：
function Dijkstra(graph, start):
    distance[start] = 0
    for all other vertices v:
        distance[v] = ∞
    
    while there are unvisited vertices:
        u = vertex with minimum distance
        mark u as visited
        for each neighbor v of u:
            if distance[u] + weight(u,v) < distance[v]:
                distance[v] = distance[u] + weight(u,v)
    
    return distance
```

**🗺️ 实际应用案例**

```
GPS导航系统：
输入：起点、终点、道路网络（带路况权重）
处理：Dijkstra算法或A*算法
输出：最短时间路径

网络路由：
路由器使用最短路径算法选择数据包传输路径
考虑因素：带宽、延迟、可靠性

供应链优化：
从工厂到客户的最低成本配送路径
节点：工厂、仓库、客户
边权：运输成本、时间
```

### 5.4 网络流问题


**💧 最大流问题**

```
问题：在容量网络中，从源点到汇点的最大流量是多少？

应用场景：
🚰 水管网络：最大供水量
🚗 交通网络：最大车流量
📡 通信网络：最大数据传输量
⚡ 电力网络：最大输电容量

Ford-Fulkerson算法：
1. 初始化：所有边的流量为0
2. 寻找增广路径：从源点到汇点的可增流路径
3. 沿路径增加流量：增加量为路径上最小剩余容量
4. 重复直到没有增广路径

最大流 = 最小割（最大流最小割定理）
```

### 5.5 图论在现代技术中的应用


**🔍 PageRank算法**

```
Google搜索的核心算法：
思想：重要的网页被更多重要网页链接

数学模型：
PR(A) = (1-d)/N + d × Σ[PR(Ti)/C(Ti)]

其中：
- PR(A)：网页A的PageRank值
- d：阻尼系数（通常0.85）
- N：网页总数
- Ti：链接到A的网页i
- C(Ti)：网页Ti的出链数量

这是一个特征向量问题！
主特征向量就是PageRank向量
```

```
蛋白质相互作用网络：

分析方法：

🔍 社区检测：发现功能模块
📊 拓扑分析：理解网络结构
```

## 6. 🎯 动态规划法


### 6.1 动态规划的基本思想


**🔸 什么是动态规划**
动态规划是将**复杂问题分解**为**子问题**，通过**存储子问题的解**来避免重复计算的方法。

```
动态规划的生活类比：
📚 考试复习：先掌握基础知识，再学高级内容
🏗️ 建房子：先打地基，再建一层层楼
💰 理财规划：每年的最佳投资依赖于前一年的结果
🎮 游戏升级：当前等级的最优策略依赖于之前的状态

核心思想：
"当前最优解 = 之前最优解 + 当前最优选择"
```

### 6.2 动态规划的基本要素


**📋 四个关键要素**

```
1. 最优子结构：
   大问题的最优解包含子问题的最优解

2. 重叠子问题：
   递归过程中会重复计算相同的子问题

3. 状态转移方程：
   描述状态之间转换关系的数学表达式

4. 边界条件：
   递归的终止条件，通常是最简单的情况
```

### 6.3 经典动态规划问题


**💰 背包问题**

```
0-1背包问题：
有n个物品，每个物品有重量wi和价值vi
背包容量为W，求最大价值

状态定义：
dp[i][w] = 前i个物品在容量w下的最大价值

状态转移：
dp[i][w] = max{
    dp[i-1][w],           // 不选第i个物品
    dp[i-1][w-wi] + vi    // 选第i个物品
}

边界条件：
dp[0][w] = 0  // 没有物品时价值为0
dp[i][0] = 0  // 容量为0时价值为0

时间复杂度：O(nW)
空间复杂度：O(nW)，可优化到O(W)
```

**🔢 最长公共子序列**

```
问题：求两个字符串的最长公共子序列

例：X = "ABCDGH", Y = "AEDFHR"
LCS = "ADH"（长度为3）

状态定义：
dp[i][j] = X[1..i]和Y[1..j]的LCS长度

状态转移：
if X[i] == Y[j]:
    dp[i][j] = dp[i-1][j-1] + 1
else:
    dp[i][j] = max(dp[i-1][j], dp[i][j-1])

边界条件：
dp[0][j] = 0, dp[i][0] = 0

应用：
📄 文本相似性比较
🧬 DNA序列分析  
📁 文件差异检测
```

### 6.4 动态规划的优化技巧


**🔄 空间优化**

```
滚动数组技巧：
很多DP问题只需要前一行的结果
可以用一维数组代替二维数组

背包问题的空间优化：
// 原始版本：dp[i][w]
// 优化版本：dp[w]

for i = 1 to n:
    for w = W to wi:  // 注意：逆序遍历
        dp[w] = max(dp[w], dp[w-wi] + vi)

为什么逆序？
避免在计算dp[w]时使用已更新的dp[w-wi]
```

### 6.5 动态规划的现代应用


**🤖 强化学习中的动态规划**

```
马尔可夫决策过程（MDP）：
状态：s ∈ S
动作：a ∈ A  
奖励：R(s,a,s')
转移概率：P(s'|s,a)

贝尔曼方程：
V(s) = max_a Σ_{s'} P(s'|s,a)[R(s,a,s') + γV(s')]

价值迭代算法：
V_{k+1}(s) = max_a Σ_{s'} P(s'|s,a)[R(s,a,s') + γV_k(s')]

这就是动态规划在AI中的应用！
```

**📊 金融工程中的应用**

```
期权定价（二叉树模型）：
每个时间步，股价有两种可能：上涨或下跌
期权价值：V = max(行权价值, 持有价值)

动态规划求解：
从到期日开始往前推算
V_t = max(行权价值, e^{-rΔt}[pV_{t+1}^u + (1-p)V_{t+1}^d])

其中：
- p：风险中性概率
- r：无风险利率
- Δt：时间步长
```

## 7. 🌊 傅里叶分析法


### 7.1 傅里叶分析的基本思想


**🔸 频域分析的威力**
傅里叶分析将**复杂的时间信号**分解为**简单的正弦波组合**，就像将复杂的音乐分解为不同频率的基本音调。

```
傅里叶分析的直观理解：
🎵 音乐：复杂音乐 = 不同频率音调的叠加
🌊 海浪：复杂波浪 = 不同周期波的合成
📡 信号：通信信号 = 载波频率的调制
🏗️ 振动：结构振动 = 各阶固有频率的组合

数学表达：
时域 ↔ 频域
f(t) ↔ F(ω)
```

### 7.2 傅里叶级数与变换


**📊 周期信号的傅里叶级数**

```
对于周期为T的函数f(t)：

三角形式：
f(t) = a₀/2 + Σ[aₙcos(nω₀t) + bₙsin(nω₀t)]

其中：
ω₀ = 2π/T（基频）
aₙ = (2/T)∫f(t)cos(nω₀t)dt
bₙ = (2/T)∫f(t)sin(nω₀t)dt

复指数形式：
f(t) = Σ cₙe^{jnω₀t}
cₙ = (1/T)∫f(t)e^{-jnω₀t}dt
```

**🔄 傅里叶变换**

```
对于非周期信号：

正变换：F(ω) = ∫f(t)e^{-jωt}dt
逆变换：f(t) = (1/2π)∫F(ω)e^{jωt}dω

物理意义：
- F(ω)：频率ω处的"频谱密度"
- |F(ω)|：振幅谱
- arg(F(ω))：相位谱
```

### 7.3 快速傅里叶变换（FFT）


**⚡ 计算效率的革命**

```
DFT（离散傅里叶变换）：
X[k] = Σ_{n=0}^{N-1} x[n]e^{-j2πkn/N}

直接计算：O(N²)复杂度
FFT算法：O(N log N)复杂度

Cooley-Tukey算法：
基本思想：分治法
将N点DFT分解为两个N/2点DFT

计算时间比较（N=1024）：
直接DFT：1,048,576次运算
FFT：10,240次运算
加速比：约100倍！
```

### 7.4 傅里叶分析的应用


**📡 信号处理**

```
数字滤波器设计：
低通滤波器：保留低频，去除高频
高通滤波器：保留高频，去除低频
带通滤波器：保留特定频段

设计步骤：
1. 在频域设计理想滤波器H(ω)
2. 通过逆FFT得到时域冲激响应h(t)
3. 加窗函数得到有限长滤波器
4. 实现数字滤波

应用：音频处理、图像处理、通信系统
```

**🏥 医学成像**

```
CT扫描的数学原理：
Radon变换：将图像投影到不同角度
逆Radon变换：从投影重建图像

关键：投影定理
图像f(x,y)的投影的傅里叶变换
= 图像傅里叶变换F(u,v)在对应直线上的值

重建算法：
1. 对各角度投影做一维FFT
2. 在频域进行滤波
3. 反变换得到滤波后的投影
4. 反投影重建图像

这就是"滤波反投影"算法！
```

**🎵 音频分析与合成**

```
音频频谱分析：
MP3压缩：去除人耳不敏感的频率成分
音效处理：在频域进行均衡、混响等处理
音乐识别：通过频谱特征识别歌曲

语音识别：
预处理：窗函数、预加重
特征提取：MFCC（梅尔频率倒谱系数）
基于傅里叶变换的频谱分析

音乐合成：
加法合成：叠加不同频率的正弦波
FM合成：频率调制合成
物理建模：模拟乐器的物理振动
```

## 8. 🔬 有限元方法


### 8.1 有限元方法的基本思想


**🔸 化整为零的智慧**
有限元方法将**复杂的连续体**分解为**有限个简单单元**，就像用积木搭建复杂的建筑。

```
有限元的直观理解：
🧩 拼图游戏：复杂图案 = 简单拼图块的组合
🏗️ 建筑施工：复杂建筑 = 标准构件的装配
🍰 切蛋糕：整个蛋糕 = 小块蛋糕的集合
🗺️ 地图网格：复杂地形 = 网格单元的拼接

数学原理：
连续问题 → 离散近似 → 数值求解
微分方程 → 代数方程组 → 计算机求解
```

### 8.2 有限元分析的基本步骤


**📋 标准流程**

```
第一步：几何建模与网格划分
- 建立几何模型
- 划分为有限个单元（三角形、四边形、四面体等）
- 定义节点和单元连接关系

第二步：单元分析
- 选择插值函数（形函数）
- 建立单元刚度矩阵
- 建立单元载荷向量

第三步：整体装配
- 将单元矩阵装配成整体矩阵
- 考虑边界条件
- 形成最终的线性方程组

第四步：求解与后处理
- 求解代数方程组
- 计算应力、应变等导出量
- 结果可视化与分析
```

### 8.3 一维有限元示例


**🔧 杆件拉伸问题**

```
问题：受轴向力的杆件，求位移分布

控制方程：-d/dx(EA du/dx) = f(x)
边界条件：u(0) = 0, EA du/dx|_{x=L} = P

有限元离散：
将杆分为n个单元，每个单元长度为h = L/n

形函数（线性插值）：
N₁(ξ) = (1-ξ)/2, N₂(ξ) = (1+ξ)/2
其中ξ ∈ [-1,1]是局部坐标

单元刚度矩阵：
K^e = (EA/h)[1  -1]
              [-1  1]

装配后的整体方程：
KU = F
其中K是整体刚度矩阵，U是位移向量，F是载荷向量
```

### 8.4 二维有限元应用


**🏗️ 平面应力分析**

```
问题：平板在面内载荷作用下的应力分析

控制方程：
∂σₓ/∂x + ∂τₓᵧ/∂y + fₓ = 0
∂τₓᵧ/∂x + ∂σᵧ/∂y + fᵧ = 0

应力-应变关系：
{σ} = [D]{ε}
其中[D]是弹性矩阵

三角形单元：
- 3个节点，6个自由度（u₁,v₁,u₂,v₂,u₃,v₃）
- 线性位移插值
- 常应变单元

形函数：
u = N₁u₁ + N₂u₂ + N₃u₃
v = N₁v₁ + N₂v₂ + N₃v₃

其中Nᵢ是面积坐标
```

```
飞机机翼分析：
```

## 9. 💻 数值计算法


### 9.1 数值计算的必要性


**🔸 为什么需要数值计算**
现实世界的很多问题无法得到**解析解**，必须通过数值方法求**近似解**。

```
需要数值计算的典型情况：
🔢 非线性方程：e^x = x² + 2x + 1
📊 高维积分：∫∫∫f(x,y,z)dxdydz
🌊 偏微分方程：∂u/∂t = ∇²u + f(x,y,t)
🎯 优化问题：min f(x) subject to g(x) ≤ 0

数值计算的特点：
✅ 能处理复杂实际问题
✅ 计算结果可控精度
⚠️ 存在数值误差
⚠️ 需要算法设计技巧
```

### 9.2 数值误差分析


**📊 误差的来源与控制**

```
误差类型：

建模误差：
现实问题 → 数学模型的简化误差
无法通过数值方法消除

截断误差：
无限过程 → 有限近似的误差
如：泰勒级数的有限项近似

舍入误差：
计算机有限精度造成的误差
累积效应可能很严重

误差传播：
输入误差 → 输出误差的放大
条件数衡量问题的敏感性

条件数：cond(A) = ||A|| × ||A⁻¹||
- 条件数大：问题病态，小误差被放大
- 条件数小：问题良态，误差控制较好
```

### 9.3 线性方程组的数值解法


**📐 直接法 vs 迭代法**

```
高斯消元法（直接法）：
优点：有限步骤得到精确解
缺点：O(n³)复杂度，大型稀疏矩阵不经济

雅可比迭代法：
x_i^{(k+1)} = (b_i - Σ_{j≠i} a_{ij}x_j^{(k)})/a_{ii}

高斯-赛德尔迭代法：
x_i^{(k+1)} = (b_i - Σ_{j<i} a_{ij}x_j^{(k+1)} - Σ_{j>i} a_{ij}x_j^{(k)})/a_{ii}

共轭梯度法（CG）：
- 适用于对称正定矩阵
- 理论上n步收敛
- 实际中通常远少于n步

预条件技术：
解 M⁻¹Ax = M⁻¹b 代替 Ax = b
选择M使得M⁻¹A条件数更好
```

### 9.4 非线性方程求解


**🎯 牛顿法的推广**

```
多元牛顿法：
问题：求解F(x) = 0，其中F: Rⁿ → Rⁿ

迭代公式：
x^{(k+1)} = x^{(k)} - J⁻¹(x^{(k)})F(x^{(k)})

其中J是雅可比矩阵：
J_{ij} = ∂F_i/∂x_j

算法步骤：
1. 选择初始点x⁰
2. 计算雅可比矩阵J(x^k)
3. 解线性方程组：J(x^k)s = -F(x^k)
4. 更新：x^{k+1} = x^k + s
5. 检验收敛性，否则返回步骤2

收敛性：
- 局部二次收敛
- 需要好的初始点
- 雅可比矩阵必须可逆
```

### 9.5 大规模科学计算


**🚀 高性能计算技术**

```
并行计算策略：

数据并行：
- 将大数据分割到多个处理器
- 每个处理器执行相同操作
- 适合矩阵运算、图像处理

任务并行：
- 将不同任务分配到不同处理器
- 处理器间可能需要通信
- 适合复杂算法流程

混合并行：
MPI + OpenMP
- MPI：进程间通信（分布式内存）

- 适合大型集群计算

GPU计算：
- CUDA、OpenCL编程
- 大规模并行线程
- 适合密集数值计算
```
