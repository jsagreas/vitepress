## 1. 📝 文本对齐工具


### 1.1 基础对齐方式

Bootstrap提供了简单直观的文本对齐类，就像在Word里设置对齐方式一样简单。

```html
<!-- 基础对齐 -->
<p class="text-start">左对齐文本（默认）</p>
<p class="text-center">居中对齐文本</p>
<p class="text-end">右对齐文本</p>
<p class="text-justify">两端对齐文本，会自动调整单词间距让每行左右都对齐</p>
```

### 1.2 响应式对齐

**💡 核心概念**：不同屏幕尺寸使用不同对齐方式

```html
<!-- 手机左对齐，平板以上居中 -->
<h2 class="text-start text-md-center">响应式标题</h2>

<!-- 手机居中，桌面右对齐 -->
<p class="text-center text-lg-end">联系方式信息</p>
```

**📱 实际应用场景**：
- 标题：手机居中显示，桌面左对齐
- 导航：手机居中，桌面右对齐
- 按钮：手机居中，桌面根据位置调整

## 2. 🎨 文本颜色系统


### 2.1 语义化颜色

Bootstrap的颜色不是随便起名的，每种颜色都有明确的使用场景。

```html
<!-- 主要内容 -->
<p class="text-primary">品牌主色调文本 - 通常是蓝色</p>
<p class="text-secondary">次要信息文本 - 通常是灰色</p>

<!-- 状态提示 -->
<p class="text-success">成功提示 - 绿色</p>
<p class="text-danger">错误警告 - 红色</p>
<p class="text-warning">注意提醒 - 黄色</p>
<p class="text-info">普通信息 - 浅蓝色</p>

<!-- 基础颜色 -->
<p class="text-light">浅色文本（用于深色背景）</p>
<p class="text-dark">深色文本（用于浅色背景）</p>
<p class="text-muted">静音文本 - 灰色淡化效果</p>
```

### 2.2 颜色使用原则

```
🔸 primary: 按钮、链接等主要操作
🔸 success: "保存成功"、"操作完成"
🔸 danger: "删除"、"错误信息"
🔸 warning: "注意事项"、"即将过期"
🔸 info: "提示信息"、"帮助说明"
🔸 muted: 次要信息、时间戳、说明文字
```

## 3. ⚖️ 字体粗细控制


### 3.1 三种粗细等级

Bootstrap提供了三个最常用的字体粗细等级：

```html
<p class="fw-light">细体文本 - 用于大标题或装饰性文字</p>
<p class="fw-normal">正常粗细 - 默认状态</p>
<p class="fw-bold">粗体文本 - 强调重要内容</p>
```

### 3.2 实际应用场景

```html
<!-- 卡片标题用粗体突出 -->
<div class="card">
  <div class="card-body">
    <h5 class="card-title fw-bold">产品名称</h5>
    <p class="card-text fw-normal">产品描述信息</p>
    <small class="text-muted fw-light">上市时间</small>
  </div>
</div>
```

## 4. 📏 字体大小等级


### 4.1 六级字体系统

Bootstrap提供了6个预设的字体大小，从大到小排列：

```html
<p class="fs-1">最大字体 - 适合主标题</p>
<p class="fs-2">大字体 - 适合副标题</p>
<p class="fs-3">中大字体 - 适合小标题</p>
<p class="fs-4">中等字体 - 适合强调文本</p>
<p class="fs-5">小字体 - 适合正文</p>
<p class="fs-6">最小字体 - 适合说明文字</p>
```

### 4.2 字体大小对应值

```
fs-1 ≈ 2.5rem (40px)  ← 主标题
fs-2 ≈ 2rem   (32px)  ← 副标题  
fs-3 ≈ 1.75rem(28px)  ← 小标题
fs-4 ≈ 1.5rem (24px)  ← 强调文本
fs-5 ≈ 1.25rem(20px)  ← 正文
fs-6 ≈ 1rem   (16px)  ← 说明文字
```

### 4.3 实际页面应用

```html
<article>
  <h1 class="fs-1 fw-bold text-primary">文章标题</h1>
  <p class="fs-6 text-muted">发布于 2025年1月</p>
  <p class="fs-5">这是文章的正文内容，使用fs-5保证良好的阅读体验...</p>
  <h2 class="fs-3 fw-bold">小节标题</h2>
</article>
```

## 5. ✨ 文本装饰效果


### 5.1 下划线控制

主要用于链接的样式控制：

```html
<!-- 移除链接下划线 -->
<a href="#" class="text-decoration-none">无下划线链接</a>

<!-- 添加下划线 -->
<span class="text-decoration-underline">带下划线文本</span>

<!-- 删除线效果 -->
<p class="text-decoration-line-through">原价 ¥199</p>
```

### 5.2 实际应用场景

```html
<!-- 导航菜单 - 去掉下划线 -->
<nav>
  <a href="#" class="text-decoration-none text-dark">首页</a>
  <a href="#" class="text-decoration-none text-dark">产品</a>
</nav>

<!-- 价格展示 -->
<div class="price">
  <span class="text-decoration-line-through text-muted">原价: ¥299</span>
  <span class="text-danger fw-bold">现价: ¥199</span>
</div>
```

## 6. ✂️ 文本截断与换行


### 6.1 文本截断

当文本太长时，用省略号代替：

```html
<!-- 单行截断 -->
<div style="width: 200px;">
  <p class="text-truncate">
    这是一段很长很长的文本，超出容器宽度会被截断显示为省略号
  </p>
</div>
```

**截断效果演示**：
```
正常显示: 这是一段很长很长的文本，超出容器宽度会被截断显示为省略号
截断效果: 这是一段很长很长的文本，超出容器...
```

### 6.2 换行控制

```html
<!-- 强制不换行 -->
<p class="text-nowrap">
  这段文本不会自动换行，即使容器很窄也会溢出显示
</p>

<!-- 允许换行（默认） -->
<p class="text-wrap">
  这段文本会正常换行，适应容器宽度进行自动换行处理
</p>
```

### 6.3 实际应用场景

```html
<!-- 商品列表 - 标题截断 -->
<div class="card" style="width: 250px;">
  <div class="card-body">
    <h5 class="card-title text-truncate">
      iPhone 15 Pro Max 256GB 深空黑色 移动联通电信5G手机
    </h5>
    <p class="card-text">¥9999</p>
  </div>
</div>

<!-- 用户名显示 - 不换行 -->
<div class="user-info">
  <span class="text-nowrap">用户名: very_long_username_123</span>
</div>
```

## 7. 📐 行高控制技巧


### 7.1 行高等级系统

控制文本行与行之间的距离：

```html
<p class="lh-1">紧密行高 - 行距很小，适合标题</p>
<p class="lh-sm">小行高 - 稍微紧凑</p>
<p class="lh-base">基础行高 - 默认正常行距</p>
<p class="lh-lg">大行高 - 行距较大，阅读舒适</p>
```

### 7.2 行高数值对应

```
lh-1    = 1      ← 最紧密（100%）
lh-sm   = 1.25   ← 紧凑（125%）
lh-base = 1.5    ← 正常（150%）
lh-lg   = 1.75   ← 宽松（175%）
```

### 7.3 行高使用场景

```html
<!-- 标题用紧密行高 -->
<h1 class="lh-1 fw-bold">
  多行标题<br>
  第二行标题
</h1>

<!-- 正文用舒适行高 -->
<p class="lh-lg">
  这是一段需要仔细阅读的文章内容，使用较大的行高
  可以让用户阅读起来更加舒适，不会感到拥挤。
</p>

<!-- 代码或数据用紧凑行高 -->
<code class="lh-sm">
  function hello() {<br>
  &nbsp;&nbsp;console.log("Hello");<br>
  }
</code>
```

## 8. 🔤 文本大小写转换


### 8.1 三种转换方式

```html
<p class="text-lowercase">UPPERCASE TEXT → lowercase text</p>
<p class="text-uppercase">lowercase text → UPPERCASE TEXT</p>
<p class="text-capitalize">hello world → Hello World</p>
```

### 8.2 实际应用场景

```html
<!-- 用户输入标准化 -->
<div class="user-profile">
  <h3 class="text-capitalize">john doe</h3>  <!-- 显示为: John Doe -->
  <p class="text-uppercase">member</p>        <!-- 显示为: MEMBER -->
</div>

<!-- 按钮文字 -->
<button class="btn btn-primary text-uppercase">
  submit form  <!-- 显示为: SUBMIT FORM -->
</button>

<!-- 邮箱显示 -->
<p class="text-lowercase">USER@EXAMPLE.COM</p>  <!-- 显示为: user@example.com -->
```

## 9. 🎯 实用技巧总结


### 9.1 常用组合模式


**卡片标题组合**：
```html
<h5 class="fw-bold fs-4 text-primary text-truncate">
  完美的卡片标题
</h5>
```

**副标题组合**：
```html
<p class="fs-6 text-muted fw-light lh-sm">
  发布于 2025年1月31日
</p>
```

**按钮文字组合**：
```html
<button class="btn btn-success text-uppercase fw-bold">
  save changes
</button>
```

### 9.2 响应式文本技巧

```html
<!-- 移动端小字体，桌面端大字体 -->
<h1 class="fs-3 fs-md-1 text-center text-md-start">
  响应式标题
</h1>

<!-- 移动端截断，桌面端完整显示 -->
<p class="text-truncate text-lg-wrap">
  长文本内容...
</p>
```

### 9.3 实战案例：文章页面

```html
<article class="container">
  <!-- 文章标题 -->
  <h1 class="fs-1 fw-bold text-primary lh-1 text-center">
    Bootstrap实用技巧大全
  </h1>
  
  <!-- 文章信息 -->
  <div class="text-center">
    <p class="fs-6 text-muted fw-light">
      发布于 <span class="text-capitalize">january 31, 2025</span>
    </p>
  </div>
  
  <!-- 文章内容 -->
  <div class="mt-4">
    <h2 class="fs-3 fw-bold text-dark lh-sm">小节标题</h2>
    <p class="lh-lg fs-5">
      这里是文章的正文内容，使用合适的行高和字体大小
      确保良好的阅读体验。
    </p>
    
    <!-- 重要提示 -->
    <div class="alert alert-warning">
      <p class="fw-bold text-warning text-uppercase mb-2">Important Note</p>
      <p class="mb-0 lh-base">
        这是一个重要的提示信息，需要用户特别注意。
      </p>
    </div>
  </div>
</article>
```

### 9.4 性能与兼容性提示


**✅ 推荐做法**：
- 优先使用语义化颜色（primary、success等）
- 响应式文本大小结合断点使用
- 截断文本时设置固定宽度

**❌ 避免做法**：
- 不要过度使用文本装饰
- 避免在小屏幕使用过大字体
- 不要混用太多字体粗细

**核心记忆**：
- **对齐**：start/center/end + 响应式
- **颜色**：语义化使用，muted做辅助
- **字体**：fs-1到6 + fw-bold/normal/light
- **装饰**：text-decoration-none最常用
- **截断**：text-truncate + 固定宽度
- **行高**：lh-lg让阅读更舒适
