## 1. 📱 响应式图片基础


### 1.1 什么是响应式图片


**🔸 核心概念**
```
响应式图片：能够根据屏幕大小自动调整尺寸的图片
目标：在不同设备上都能完美显示
原理：图片宽度自适应容器，高度等比缩放
```

**💡 为什么需要响应式图片？**
```
问题场景：
手机屏幕：320px宽度
电脑屏幕：1920px宽度
如果图片固定1000px宽度 → 手机上会溢出

解决方案：
使用 img-fluid 类让图片自适应
```

### 1.2 img-fluid 的使用


**🔧 基本用法**
```html
<!-- 普通图片（固定尺寸） -->
<img src="photo.jpg" alt="固定尺寸图片">

<!-- 响应式图片（自适应尺寸） -->
<img src="photo.jpg" class="img-fluid" alt="响应式图片">
```

**⚡ 实际效果对比**
```
普通图片：
┌─────────────┐  手机屏幕(320px)
│ [========= │  图片溢出 ❌
│            │
└─────────────┘

响应式图片：
┌─────────────┐  手机屏幕(320px)  
│ [=======]   │  图片适应 ✅
│             │
└─────────────┘
```

### 1.3 picture响应式图片


**🔸 高级响应式方案**
```html
<!-- 不同屏幕使用不同图片 -->
<picture>
  <source media="(min-width: 768px)" srcset="large.jpg">
  <source media="(min-width: 480px)" srcset="medium.jpg">
  <img src="small.jpg" class="img-fluid" alt="响应式图片">
</picture>
```

**📊 使用场景**
- **大屏幕**：显示高清大图
- **中等屏幕**：显示中等画质
- **小屏幕**：显示压缩小图，节省流量

## 2. 🎨 图片样式美化


### 2.1 缩略图效果


**🔸 img-thumbnail 缩略图**
```html
<!-- 普通图片 -->
<img src="avatar.jpg" alt="头像">

<!-- 缩略图效果（带边框） -->
<img src="avatar.jpg" class="img-thumbnail" alt="头像缩略图">
```

**💡 视觉效果**
```
普通图片：       缩略图：
┌─────────┐     ┌───────────┐
│ [图片]  │     │┌─────────┐│
│         │ →   ││ [图片]  ││ ← 带浅色边框
│         │     │└─────────┘│
└─────────┘     └───────────┘
```

### 2.2 圆角图片


**🔸 rounded 圆角系列**
```html
<!-- 小圆角 -->
<img src="photo.jpg" class="rounded">

<!-- 大圆角 -->
<img src="photo.jpg" class="rounded-lg">

<!-- 圆形图片（常用于头像） -->
<img src="avatar.jpg" class="rounded-circle">

<!-- 胶囊形状 -->
<img src="banner.jpg" class="rounded-pill">
```

**🎭 效果展示**
```
rounded：        rounded-circle：    rounded-pill：
┌─────────┐     ○─────────○        ╭─────────╮
│ [图片]  │     │ [头像]  │        │ [横幅]  │  
│         │     │         │        │         │
└─────────┘     ○─────────○        ╰─────────╯
```

### 2.3 图片适应方式


**🔸 object-fit 控制图片适应**
```html
<!-- 图片完全填充容器（可能变形） -->
<img src="photo.jpg" style="object-fit: fill; width: 200px; height: 150px;">

<!-- 图片等比缩放（保持比例） -->
<img src="photo.jpg" style="object-fit: cover; width: 200px; height: 150px;">

<!-- 图片完整显示（可能有空白） -->
<img src="photo.jpg" style="object-fit: contain; width: 200px; height: 150px;">
```

## 3. 🖼️ 图片说明组件


### 3.1 figure 图片说明容器


**🔸 基本结构**
```html
<figure class="figure">
  <img src="landscape.jpg" class="figure-img img-fluid" alt="风景图">
  <figcaption class="figure-caption">这是一张美丽的风景图片</figcaption>
</figure>
```

**💡 组件说明**
- `figure`：图片和说明的容器
- `figure-img`：图片样式
- `figure-caption`：说明文字样式

### 3.2 说明文字位置


**🔧 文字对齐方式**
```html
<!-- 左对齐说明（默认） -->
<figcaption class="figure-caption">左对齐说明文字</figcaption>

<!-- 右对齐说明 -->
<figcaption class="figure-caption text-end">右对齐说明文字</figcaption>

<!-- 居中说明 -->
<figcaption class="figure-caption text-center">居中说明文字</figcaption>
```

### 3.3 完整示例


**📱 实际应用**
```html
<div class="row">
  <div class="col-md-6">
    <figure class="figure">
      <img src="product1.jpg" class="figure-img img-fluid rounded" alt="产品图">
      <figcaption class="figure-caption text-center">
        <strong>产品名称</strong><br>
        产品详细描述信息
      </figcaption>
    </figure>
  </div>
</div>
```

## 4. 📺 媒体对象布局


### 4.1 什么是媒体对象


**🔸 核心概念**
```
媒体对象：图片 + 文字内容的组合布局
常见场景：
- 用户评论（头像 + 评论内容）
- 产品列表（产品图 + 产品信息）
- 新闻列表（缩略图 + 新闻标题）
```

### 4.2 基本媒体对象


**🔧 基础结构**
```html
<div class="d-flex">
  <img src="avatar.jpg" class="me-3" alt="用户头像" width="64">
  <div class="media-body">
    <h5>张三</h5>
    <p>这是用户发表的评论内容，可以很长很长...</p>
  </div>
</div>
```

**📊 布局效果**
```
[头像图片] 用户姓名
           评论内容第一行
           评论内容第二行
           ...
```

### 4.3 媒体对象变化


**🔸 图片位置变化**
```html
<!-- 图片在右侧 -->
<div class="d-flex">
  <div class="media-body me-3">
    <h5>标题内容</h5>
    <p>文字描述...</p>
  </div>
  <img src="thumbnail.jpg" alt="缩略图" width="64">
</div>

<!-- 图片在上方 -->
<div class="text-center">
  <img src="product.jpg" class="mb-3" alt="产品图">
  <div>
    <h5>产品名称</h5>
    <p>产品描述...</p>
  </div>
</div>
```

### 4.4 嵌套媒体对象


**🔧 评论回复场景**
```html
<div class="d-flex">
  <!-- 主评论 -->
  <img src="user1.jpg" class="me-3" alt="用户1" width="64">
  <div class="media-body">
    <h5>用户1</h5>
    <p>这是主评论内容...</p>
    
    <!-- 回复评论 -->
    <div class="d-flex mt-3">
      <img src="user2.jpg" class="me-3" alt="用户2" width="48">
      <div class="media-body">
        <h6>用户2</h6>
        <p>这是回复内容...</p>
      </div>
    </div>
  </div>
</div>
```

## 5. 🎬 响应式嵌入内容


### 5.1 什么是响应式嵌入


**🔸 核心概念**
```
响应式嵌入：让视频、地图等嵌入内容自适应屏幕
常见内容：
- YouTube视频
- 百度地图
- PDF文档
- 其他iframe内容
```

### 5.2 embed响应式嵌入


**🔧 基本用法**
```html
<!-- 16:9比例的响应式嵌入 -->
<div class="ratio ratio-16x9">
  <iframe src="https://www.youtube.com/embed/视频ID" 
          title="视频标题" allowfullscreen></iframe>
</div>

<!-- 4:3比例的嵌入 -->
<div class="ratio ratio-4x3">
  <iframe src="地图链接" title="地图"></iframe>
</div>
```

### 5.3 常用比例


**📊 预设比例选项**
```html
<!-- 21:9 超宽屏 -->
<div class="ratio ratio-21x9">
  <iframe src="..."></iframe>
</div>

<!-- 16:9 标准宽屏 -->
<div class="ratio ratio-16x9">
  <iframe src="..."></iframe>
</div>

<!-- 4:3 传统屏幕 -->
<div class="ratio ratio-4x3">
  <iframe src="..."></iframe>
</div>

<!-- 1:1 正方形 -->
<div class="ratio ratio-1x1">
  <iframe src="..."></iframe>
</div>
```

### 5.4 自定义比例


**🔧 CSS自定义**
```html
<!-- 自定义3:2比例 -->
<div class="ratio" style="--bs-aspect-ratio: 66.67%;">
  <iframe src="..." title="自定义比例内容"></iframe>
</div>
```

**💡 计算方法**
```
比例计算公式：
高度/宽度 × 100% = 百分比值

3:2比例 = 2/3 × 100% = 66.67%
5:4比例 = 4/5 × 100% = 80%
```

## 6. 📋 核心要点总结


### 6.1 必须掌握的基本概念


```
🔸 响应式图片：img-fluid 让图片自适应容器大小
🔸 图片美化：rounded系列创造不同圆角效果
🔸 图片说明：figure + figcaption 组合使用
🔸 媒体对象：图片与文字内容的灵活布局
🔸 响应式嵌入：ratio 让视频地图等内容自适应
```

### 6.2 关键类名速查


| 功能类别 | **类名** | **作用** |
|---------|---------|---------|
| 📱 **响应式** | `img-fluid` | `图片自适应容器` |
| 🖼️ **缩略图** | `img-thumbnail` | `带边框的缩略图效果` |
| ⭕ **圆角** | `rounded` | `小圆角` |
| 🔵 **圆形** | `rounded-circle` | `完全圆形（头像常用）` |
| 💊 **胶囊** | `rounded-pill` | `胶囊形状` |
| 📝 **说明** | `figure` | `图片说明容器` |
| 📝 **说明** | `figure-caption` | `说明文字样式` |
| 📺 **嵌入** | `ratio ratio-16x9` | `16:9响应式嵌入` |

### 6.3 实际应用场景


**🎯 常见使用场景**
```
用户头像显示：
<img src="avatar.jpg" class="rounded-circle" width="50">

产品图片展示：
<img src="product.jpg" class="img-fluid img-thumbnail">

视频嵌入：
<div class="ratio ratio-16x9">
  <iframe src="视频链接"></iframe>
</div>

评论列表布局：
<div class="d-flex">
  <img src="头像" class="me-3 rounded-circle" width="48">
  <div class="media-body">
    <h6>用户名</h6>
    <p>评论内容</p>
  </div>
</div>
```

**🔧 组合使用技巧**
```
多个类组合：
class="img-fluid rounded shadow"
↑ 响应式 + 圆角 + 阴影

媒体对象 + 响应式：
<div class="d-flex align-items-center">
  <img class="me-3 rounded-circle img-thumbnail">
  <div class="flex-grow-1">...</div>
</div>
```

### 6.4 注意事项


**⚠️ 常见问题**
- `img-fluid` 只控制最大宽度，不控制最小宽度
- `rounded-circle` 需要图片是正方形才能显示为完美圆形
- 响应式嵌入的 `iframe` 必须放在 `ratio` 容器内
- 媒体对象使用 `d-flex` 而不是专门的 media 类（Bootstrap 5已移除）

**💡 最佳实践**
- 图片加载建议使用 `loading="lazy"` 延迟加载
- 重要图片添加 `alt` 属性提升可访问性
- 大图片考虑使用 `picture` 元素提供多种尺寸
- 嵌入内容注意隐私和安全问题

**核心记忆**：
- 图片响应用 fluid，美化样式选 rounded
- 说明组合 figure 配 caption，媒体布局 flex 最灵活
- 嵌入内容 ratio 包装，比例选择要合适
