## 1. 📋 表格基础结构


### 1.1 HTML表格基本构成


表格就像是网页中的Excel表格，用来整齐地展示数据。Bootstrap让原本单调的HTML表格变得美观实用。

```html
<!-- 基础表格结构 -->
<table class="table">
  <!-- 表格标题 -->
  <caption>学生成绩表</caption>
  
  <!-- 表头部分 -->
  <thead>
    <tr>
      <th scope="col">姓名</th>
      <th scope="col">语文</th>
      <th scope="col">数学</th>
      <th scope="col">总分</th>
    </tr>
  </thead>
  
  <!-- 表格主体 -->
  <tbody>
    <tr>
      <td>张三</td>
      <td>85</td>
      <td>92</td>
      <td>177</td>
    </tr>
    <tr>
      <td>李四</td>
      <td>78</td>
      <td>88</td>
      <td>166</td>
    </tr>
  </tbody>
  
  <!-- 表格底部（可选） -->
  <tfoot>
    <tr>
      <td>平均分</td>
      <td>81.5</td>
      <td>90</td>
      <td>171.5</td>
    </tr>
  </tfoot>
</table>
```

### 1.2 表格结构详解


```
表格结构层次：
┌─────────────────────────────────┐
│ <table>  表格容器               │
│ ├── <caption>  表格标题         │
│ ├── <thead>    表头区域         │
│ │   └── <tr><th>...             │
│ ├── <tbody>    数据区域         │
│ │   └── <tr><td>...             │
│ └── <tfoot>    底部汇总（可选） │
│     └── <tr><td>...             │
└─────────────────────────────────┘
```

**核心概念理解**：
- **`<table>`**：整个表格的容器，就像画表格的纸
- **`<thead>`**：表头区域，放列标题的地方
- **`<tbody>`**：表格主体，放具体数据的地方
- **`<tfoot>`**：表格底部，通常放合计、平均值等
- **`<th>`**：表头单元格，**自动加粗居中**
- **`<td>`**：普通数据单元格

## 2. 🎨 Bootstrap表格样式


### 2.1 基础表格样式


Bootstrap的 `.table` 类是所有表格样式的基础，就像给表格穿上了一套基础"衣服"。

```html
<!-- 最基础的Bootstrap表格 -->
<table class="table">
  <thead>
    <tr>
      <th>产品名称</th>
      <th>价格</th>
      <th>库存</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>iPhone 15</td>
      <td>¥6999</td>
      <td>50</td>
    </tr>
    <tr>
      <td>MacBook Pro</td>
      <td>¥12999</td>
      <td>20</td>
    </tr>
  </tbody>
</table>
```

### 2.2 斑马线表格（table-striped）


**作用**：让表格行交替显示不同背景色，像斑马纹一样，更容易区分每一行。

```html
<table class="table table-striped">
  <thead>
    <tr>
      <th>员工姓名</th>
      <th>部门</th>
      <th>薪资</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>王小明</td>  <!-- 白色背景 -->
      <td>技术部</td>
      <td>15000</td>
    </tr>
    <tr>
      <td>李小花</td>  <!-- 灰色背景 -->
      <td>市场部</td>
      <td>12000</td>
    </tr>
    <tr>
      <td>张小强</td>  <!-- 白色背景 -->
      <td>财务部</td>
      <td>13000</td>
    </tr>
  </tbody>
</table>
```

### 2.3 边框表格（table-bordered）


**作用**：给表格添加边框，让数据区域更清晰分明。

```html
<table class="table table-bordered">
  <thead>
    <tr>
      <th>城市</th>
      <th>温度</th>
      <th>天气</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>北京</td>
      <td>25°C</td>
      <td>晴天</td>
    </tr>
    <tr>
      <td>上海</td>
      <td>28°C</td>
      <td>多云</td>
    </tr>
  </tbody>
</table>
```

### 2.4 悬停效果（table-hover）


**作用**：鼠标悬停时高亮显示当前行，提升用户体验。

```html
<table class="table table-hover">
  <thead>
    <tr>
      <th>课程名称</th>
      <th>学分</th>
      <th>教师</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Java程序设计</td>  <!-- 鼠标悬停时会高亮 -->
      <td>4</td>
      <td>张教授</td>
    </tr>
    <tr>
      <td>数据库原理</td>
      <td>3</td>
      <td>李教授</td>
    </tr>
  </tbody>
</table>
```

### 2.5 紧凑表格（table-sm）


**作用**：减少单元格内边距，让表格更紧凑，适合显示大量数据。

```html
<table class="table table-sm">
  <thead>
    <tr>
      <th>ID</th>
      <th>用户名</th>
      <th>邮箱</th>
      <th>状态</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>001</td>
      <td>admin</td>
      <td>admin@example.com</td>
      <td>在线</td>
    </tr>
    <tr>
      <td>002</td>
      <td>user01</td>
      <td>user01@example.com</td>
      <td>离线</td>
    </tr>
  </tbody>
</table>
```

### 2.6 组合样式使用


**实战技巧**：多个样式可以组合使用，创造出更丰富的视觉效果。

```html
<!-- 组合使用：斑马线 + 悬停 + 边框 -->
<table class="table table-striped table-hover table-bordered">
  <thead>
    <tr>
      <th>订单号</th>
      <th>商品</th>
      <th>数量</th>
      <th>金额</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>20240731001</td>
      <td>笔记本电脑</td>
      <td>1</td>
      <td>¥8999</td>
    </tr>
    <tr>
      <td>20240731002</td>
      <td>无线鼠标</td>
      <td>2</td>
      <td>¥198</td>
    </tr>
  </tbody>
</table>
```

## 3. 🌈 表格颜色变体


### 3.1 表格整体颜色


Bootstrap提供了多种预定义颜色，让表格更有语义化和视觉冲击力。

| 颜色类名 | **效果说明** | **使用场景** |
|---------|-------------|-------------|
| `table-primary` | 主要蓝色调 | 重要数据表格 |
| `table-success` | 成功绿色调 | 成功状态数据 |
| `table-danger` | 危险红色调 | 错误或警告数据 |
| `table-warning` | 警告黄色调 | 需要注意的数据 |
| `table-info` | 信息蓝色调 | 提示性信息 |
| `table-light` | 浅色调 | 普通数据展示 |
| `table-dark` | 深色调 | 深色主题应用 |

```html
<!-- 成功状态的表格 -->
<table class="table table-success">
  <thead>
    <tr>
      <th>任务名称</th>
      <th>完成状态</th>
      <th>完成时间</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>项目开发</td>
      <td>已完成</td>
      <td>2024-07-30</td>
    </tr>
  </tbody>
</table>
```

### 3.2 行级颜色控制


**更精细的控制**：可以给单独的行设置不同颜色，突出重要信息。

```html
<table class="table">
  <thead>
    <tr>
      <th>学生姓名</th>
      <th>考试成绩</th>
      <th>等级</th>
    </tr>
  </thead>
  <tbody>
    <tr class="table-success">  <!-- 成功：绿色背景 -->
      <td>张三</td>
      <td>95分</td>
      <td>优秀</td>
    </tr>
    <tr class="table-warning">  <!-- 警告：黄色背景 -->
      <td>李四</td>
      <td>75分</td>
      <td>良好</td>
    </tr>
    <tr class="table-danger">   <!-- 危险：红色背景 -->
      <td>王五</td>
      <td>55分</td>
      <td>不及格</td>
    </tr>
  </tbody>
</table>
```

### 3.3 单元格颜色控制


**最精细的控制**：甚至可以给单个单元格设置颜色。

```html
<table class="table">
  <thead>
    <tr>
      <th>指标名称</th>
      <th>当前值</th>
      <th>状态</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>CPU使用率</td>
      <td class="table-warning">85%</td>  <!-- 单独设置警告色 -->
      <td>偏高</td>
    </tr>
    <tr>
      <td>内存使用率</td>
      <td class="table-success">45%</td>  <!-- 单独设置成功色 -->
      <td>正常</td>
    </tr>
  </tbody>
</table>
```

## 4. 📱 响应式表格


### 4.1 基础响应式表格


**问题**：在手机等小屏设备上，表格内容可能会被挤压变形。  
**解决方案**：使用 `table-responsive` 让表格在小屏幕上可以水平滚动。

```html
<!-- 基础响应式表格 -->
<div class="table-responsive">
  <table class="table">
    <thead>
      <tr>
        <th>商品ID</th>
        <th>商品名称</th>
        <th>分类</th>
        <th>价格</th>
        <th>库存</th>
        <th>上架时间</th>
        <th>操作</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>P001</td>
        <td>iPhone 15 Pro Max</td>
        <td>数码产品</td>
        <td>¥9999</td>
        <td>50</td>
        <td>2024-07-01</td>
        <td>
          <button class="btn btn-sm btn-primary">编辑</button>
          <button class="btn btn-sm btn-danger">删除</button>
        </td>
      </tr>
    </tbody>
  </table>
</div>
```

### 4.2 响应式断点控制


**更精确的控制**：可以指定在哪个屏幕尺寸开始显示滚动条。

```html
<!-- 只在小屏幕(sm)以下显示滚动条 -->
<div class="table-responsive-sm">
  <table class="table">
    <!-- 表格内容 -->
  </table>
</div>

<!-- 在中等屏幕(md)以下显示滚动条 -->
<div class="table-responsive-md">
  <table class="table">
    <!-- 表格内容 -->
  </table>
</div>
```

**断点说明**：
```
响应式断点对照表：
┌────────────────┬─────────────┬──────────────┐
│ 类名           │ 屏幕尺寸     │ 何时出现滚动条 │
├────────────────┼─────────────┼──────────────┤
│ table-responsive-sm │ <576px  │ 手机竖屏时    │
│ table-responsive-md │ <768px  │ 手机横屏时    │
│ table-responsive-lg │ <992px  │ 平板设备时    │
│ table-responsive-xl │ <1200px │ 小笔记本时    │
└────────────────┴─────────────┴──────────────┘
```

## 5. ♿ 表格可访问性


### 5.1 caption表格标题


**作用**：为表格提供标题说明，帮助屏幕阅读器用户理解表格内容。

```html
<table class="table">
  <caption>2024年第二季度销售数据统计表</caption>
  <thead>
    <tr>
      <th scope="col">月份</th>
      <th scope="col">销售额</th>
      <th scope="col">增长率</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>4月</td>
      <td>¥158万</td>
      <td>+12%</td>
    </tr>
    <tr>
      <td>5月</td>
      <td>¥173万</td>
      <td>+18%</td>
    </tr>
  </tbody>
</table>
```

### 5.2 scope属性详解


**关键概念**：`scope` 属性告诉屏幕阅读器这个表头是管理哪个范围的数据。

```html
<table class="table">
  <thead>
    <tr>
      <!-- scope="col" 表示这是列标题 -->
      <th scope="col">姓名</th>
      <th scope="col">语文</th>
      <th scope="col">数学</th>
      <th scope="col">英语</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <!-- scope="row" 表示这是行标题 -->
      <th scope="row">张三</th>
      <td>85</td>
      <td>90</td>
      <td>88</td>
    </tr>
    <tr>
      <th scope="row">李四</th>
      <td>78</td>
      <td>85</td>
      <td>92</td>
    </tr>
  </tbody>
</table>
```

**scope属性值说明**：
- **`scope="col"`**：这个 `<th>` 是列标题，管理下面一整列的数据
- **`scope="row"`**：这个 `<th>` 是行标题，管理右边一整行的数据

## 6. 📋 核心要点总结


### 6.1 必须掌握的基本概念


```
🔸 表格基础：table基础结构和语义化标签的正确使用
🔸 Bootstrap样式：table、table-striped、table-bordered、table-hover等
🔸 颜色系统：table-*颜色变体的语义化使用
🔸 响应式设计：table-responsive在不同设备上的表现
🔸 可访问性：caption和scope属性的重要性
```

### 6.2 关键记忆要点


**🔹 表格结构层次**
```
基础记忆：
- table → 表格容器
- thead → 表头区域（用th）
- tbody → 数据区域（用td）
- tfoot → 底部汇总（可选）
```

**🔹 样式组合技巧**
```
常用组合：
- table + table-striped + table-hover → 斑马线悬停效果
- table + table-bordered + table-sm → 紧凑边框表格
- table-responsive + 颜色类 → 响应式彩色表格
```

**🔹 响应式策略**
```
移动端优化：
- 数据量大 → 使用 table-responsive
- 屏幕特别小 → 考虑改用卡片布局
- 重要数据优先 → 隐藏次要列
```

### 6.3 实际应用指导


**适用场景判断**：
- ✅ **数据展示**：成绩单、商品列表、用户信息
- ✅ **状态监控**：系统状态、任务进度、服务器监控
- ✅ **对比分析**：价格对比、性能对比、历史数据
- ✅ **管理后台**：用户管理、订单管理、内容管理

**不适用场景**：
- ❌ **复杂布局**：不规则的数据展示
- ❌ **少量数据**：1-2行数据用列表更合适
- ❌ **移动优先**：手机应用优先考虑卡片布局

### 6.4 开发实践技巧


**性能优化**：
```html
<!-- 大数据表格：添加固定高度和滚动 -->
<div class="table-responsive" style="max-height: 400px; overflow-y: auto;">
  <table class="table table-sm">
    <!-- 大量数据 -->
  </table>
</div>
```

**用户体验**：
```html
<!-- 加载状态提示 -->
<table class="table">
  <tbody>
    <tr>
      <td colspan="4" class="text-center">
        <div class="spinner-border" role="status">
          <span class="visually-hidden">加载中...</span>
        </div>
      </td>
    </tr>
  </tbody>
</table>
```

**核心记忆口诀**：
- 表格结构要清晰，thead tbody要分明
- Bootstrap样式很丰富，striped hover加边框
- 颜色变体有语义，success danger要用对
- 移动设备用响应式，caption scope助无障碍
