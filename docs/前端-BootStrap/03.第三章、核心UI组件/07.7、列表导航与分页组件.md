## 1. 📋 列表组件


### 1.1 基础列表组（list-group）


**🔸 什么是列表组**
列表组就是把相关的内容项目整齐地排列在一起，像微信聊天列表、设置页面的选项列表一样。

```html
<!-- 基础列表组 -->
<ul class="list-group">
  <li class="list-group-item">首页</li>
  <li class="list-group-item">产品介绍</li>
  <li class="list-group-item">关于我们</li>
  <li class="list-group-item">联系方式</li>
</ul>
```

**💡 核心概念**
- `list-group`：给列表容器加上统一的样式
- `list-group-item`：每个列表项的样式类

### 1.2 可点击列表项（list-group-item-action）


**🔸 交互式列表**
当列表项需要点击时，添加 `list-group-item-action` 类，会有鼠标悬停效果。

```html
<!-- 可点击的列表 -->
<div class="list-group">
  <a href="#" class="list-group-item list-group-item-action">
    📧 收件箱 <span class="badge bg-primary">12</span>
  </a>
  <a href="#" class="list-group-item list-group-item-action">
    📤 已发送
  </a>
  <a href="#" class="list-group-item list-group-item-action active">
    📝 草稿箱
  </a>
</div>
```

**✨ 状态说明**
- `active`：当前选中项（蓝色背景）
- `disabled`：禁用项（灰色，不可点击）

### 1.3 列表组的实用技巧


```html
<!-- 带图标和说明的列表 -->
<div class="list-group">
  <div class="list-group-item">
    <div class="d-flex w-100 justify-content-between">
      <h5 class="mb-1">🎵 音乐播放</h5>
      <small>3分钟前</small>
    </div>
    <p class="mb-1">正在播放：周杰伦 - 青花瓷</p>
  </div>
  <div class="list-group-item">
    <div class="d-flex w-100 justify-content-between">
      <h5 class="mb-1">📺 视频播放</h5>
      <small>10分钟前</small>
    </div>
    <p class="mb-1">正在观看：前端开发教程</p>
  </div>
</div>
```

## 2. 🧭 导航组件


### 2.1 基础导航（nav）


**🔸 导航的本质**
导航就像网站的"目录"，帮用户快速跳转到不同页面或内容区域。

```html
<!-- 基础导航 -->
<nav class="nav">
  <a class="nav-link active" href="#">首页</a>
  <a class="nav-link" href="#">产品</a>
  <a class="nav-link" href="#">服务</a>
  <a class="nav-link" href="#">关于</a>
</nav>
```

### 2.2 标签页导航（nav-tabs）


**🔸 像文件夹标签一样的导航**
标签页让用户在同一页面内切换不同内容，就像浏览器的多个标签页。

```html
<!-- 标签页导航 -->
<ul class="nav nav-tabs">
  <li class="nav-item">
    <a class="nav-link active" href="#home">🏠 首页</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#profile">👤 个人资料</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#settings">⚙️ 设置</a>
  </li>
</ul>

<!-- 对应的内容区域 -->
<div class="tab-content mt-3">
  <div class="tab-pane active" id="home">
    <h4>欢迎来到首页</h4>
    <p>这里是首页内容...</p>
  </div>
</div>
```

### 2.3 胶囊导航（nav-pills）


**🔸 圆角按钮式的导航**
胶囊导航看起来像药丸，更适合侧边栏或者空间较小的地方。

```html
<!-- 胶囊导航 -->
<ul class="nav nav-pills">
  <li class="nav-item">
    <a class="nav-link active" href="#">📊 数据统计</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">📈 销售报表</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">💰 财务分析</a>
  </li>
</ul>
```

### 2.4 导航布局优化


**🔸 填充和对齐选项**

```html
<!-- 填充整个宽度 -->
<ul class="nav nav-pills nav-fill">
  <li class="nav-item">
    <a class="nav-link" href="#">短</a>
  </li>
  <li class="nav-item">
    <a class="nav-link active" href="#">中等长度</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">很长的导航项目</a>
  </li>
</ul>

<!-- 等宽布局 -->
<ul class="nav nav-pills nav-justified">
  <li class="nav-item">
    <a class="nav-link" href="#">项目1</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">项目2</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">项目3</a>
  </li>
</ul>
```

**💡 填充布局说明**
- `nav-fill`：按内容比例分配空间（内容多的占地大）
- `nav-justified`：平均分配空间（每项等宽）

## 3. 📄 分页组件


### 3.1 基础分页（pagination）


**🔸 分页的作用**
当内容太多时，分页帮助用户一页页浏览，就像翻书一样。

```html
<!-- 基础分页 -->
<nav>
  <ul class="pagination">
    <li class="page-item">
      <a class="page-link" href="#">上一页</a>
    </li>
    <li class="page-item"><a class="page-link" href="#">1</a></li>
    <li class="page-item active">
      <a class="page-link" href="#">2</a>
    </li>
    <li class="page-item"><a class="page-link" href="#">3</a></li>
    <li class="page-item">
      <a class="page-link" href="#">下一页</a>
    </li>
  </ul>
</nav>
```

### 3.2 分页尺寸调整


**🔸 不同尺寸的分页**

```html
<!-- 大尺寸分页 -->
<ul class="pagination pagination-lg">
  <li class="page-item">
    <a class="page-link" href="#">1</a>
  </li>
  <li class="page-item active">
    <a class="page-link" href="#">2</a>
  </li>
  <li class="page-item">
    <a class="page-link" href="#">3</a>
  </li>
</ul>

<!-- 默认尺寸 -->
<ul class="pagination">
  <li class="page-item"><a class="page-link" href="#">1</a></li>
  <li class="page-item"><a class="page-link" href="#">2</a></li>
  <li class="page-item"><a class="page-link" href="#">3</a></li>
</ul>

<!-- 小尺寸分页 -->
<ul class="pagination pagination-sm">
  <li class="page-item"><a class="page-link" href="#">1</a></li>
  <li class="page-item"><a class="page-link" href="#">2</a></li>
  <li class="page-item"><a class="page-link" href="#">3</a></li>
</ul>
```

### 3.3 实用分页样式


```html
<!-- 完整的分页组件 -->
<nav aria-label="页面导航">
  <ul class="pagination justify-content-center">
    <li class="page-item disabled">
      <a class="page-link" href="#" tabindex="-1">
        « 首页
      </a>
    </li>
    <li class="page-item">
      <a class="page-link" href="#">‹ 上一页</a>
    </li>
    
    <li class="page-item"><a class="page-link" href="#">1</a></li>
    <li class="page-item active">
      <a class="page-link" href="#">2 <span class="sr-only">(当前)</span></a>
    </li>
    <li class="page-item"><a class="page-link" href="#">3</a></li>
    <li class="page-item">
      <span class="page-link">...</span>
    </li>
    <li class="page-item"><a class="page-link" href="#">10</a></li>
    
    <li class="page-item">
      <a class="page-link" href="#">下一页 ›</a>
    </li>
    <li class="page-item">
      <a class="page-link" href="#">末页 »</a>
    </li>
  </ul>
</nav>
```

## 4. 🎯 实际应用场景


### 4.1 常见组合应用


**📱 移动端菜单**
```html
<!-- 手机端侧边栏菜单 -->
<div class="list-group">
  <a href="#" class="list-group-item list-group-item-action active">
    🏠 首页
  </a>
  <a href="#" class="list-group-item list-group-item-action">
    🛍️ 商城
  </a>
  <a href="#" class="list-group-item list-group-item-action">
    👤 个人中心
  </a>
  <a href="#" class="list-group-item list-group-item-action">
    ⚙️ 设置
  </a>
</div>
```

**💻 后台管理页面**
```html
<!-- Tab切换 + 分页 -->
<ul class="nav nav-tabs mb-3">
  <li class="nav-item">
    <a class="nav-link active" href="#users">用户管理</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#orders">订单管理</a>
  </li>
</ul>

<!-- 内容区域 -->
<div class="tab-content">
  <div class="tab-pane active" id="users">
    <!-- 用户列表 -->
    <div class="table-responsive">
      <!-- 表格内容... -->
    </div>
    
    <!-- 分页 -->
    <nav class="mt-3">
      <ul class="pagination justify-content-end">
        <li class="page-item"><a class="page-link" href="#">1</a></li>
        <li class="page-item active"><a class="page-link" href="#">2</a></li>
        <li class="page-item"><a class="page-link" href="#">3</a></li>
      </ul>
    </nav>
  </div>
</div>
```

### 4.2 响应式设计技巧


```html
<!-- 响应式导航 -->
<ul class="nav nav-pills flex-column flex-md-row">
  <li class="nav-item">
    <a class="nav-link" href="#">首页</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">产品</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">服务</a>
  </li>
</ul>
```

> **💡 响应式说明**：在小屏幕上垂直排列（`flex-column`），中等屏幕以上水平排列（`flex-md-row`）

## 5. 📋 核心要点总结


### 5.1 组件对比表格


| 组件类型 | **主要用途** | **核心类名** | **最佳场景** |
|---------|------------|-------------|-------------|
| 📋 **列表组** | `展示相关项目` | `list-group` | `设置选项、消息列表` |
| 🧭 **标签页** | `内容切换` | `nav-tabs` | `详情页面、表单分步` |
| 💊 **胶囊导航** | `页面导航` | `nav-pills` | `侧边栏、小空间导航` |
| 📄 **分页** | `数据分页` | `pagination` | `列表页面、搜索结果` |

### 5.2 记忆要点


**🔹 选择原则**
```
内容展示 → 用 list-group（列表组）
页面切换 → 用 nav-tabs（标签页）
菜单导航 → 用 nav-pills（胶囊导航）
数据很多 → 用 pagination（分页）
```

**🔹 常用类名速记**
```
list-group-item          列表项
list-group-item-action   可点击列表项
nav-link                 导航链接
page-item                分页项
page-link                分页链接
nav-fill                 填充布局
nav-justified            等宽布局
pagination-lg/sm         分页大小
```

**🔹 实用技巧**
- 列表组可以放任何内容，不只是文字
- 导航要配合JavaScript实现真正的切换效果
- 分页通常配合后端数据一起使用
- 响应式设计时考虑在小屏幕上的显示效果

**核心记忆**：
- **列表组**：整理相关内容，像手机设置页面
- **导航**：帮用户跳转，tabs像文件夹，pills像按钮
- **分页**：数据太多时分页显示，提升用户体验
- **布局**：fill按比例分配，justified平均分配
