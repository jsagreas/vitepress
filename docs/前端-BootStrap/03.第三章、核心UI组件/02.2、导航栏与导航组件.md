## 1. 🧭 导航栏基础概念


### 1.1 什么是导航栏


**简单理解**：导航栏就是网站顶部的"菜单条"，用户通过它快速跳转到不同页面。

```
典型网站导航栏样子：
┌─────────────────────────────────────────────────────┐
│ 🏠 网站Logo    首页  产品  服务  关于我们    登录 注册 │
└─────────────────────────────────────────────────────┘
```

**核心作用**：
- 🎯 **快速导航**：让用户知道"我在哪，能去哪"
- 🎨 **品牌展示**：放置Logo和网站名称
- 📱 **响应式体验**：手机上自动折叠成汉堡菜单

### 1.2 Bootstrap导航栏的优势


**传统做法 vs Bootstrap做法**：
```html
<!-- 传统方式：复杂CSS -->
<div class="header">
  <div class="logo">Logo</div>
  <ul class="menu">
    <li><a href="#">首页</a></li>
    <li><a href="#">产品</a></li>
  </ul>
</div>
<!-- 需要写很多CSS样式 -->

<!-- Bootstrap方式：简单快捷 -->
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">Logo</a>
  <ul class="navbar-nav">
    <li class="nav-item"><a class="nav-link" href="#">首页</a></li>
    <li class="nav-item"><a class="nav-link" href="#">产品</a></li>
  </ul>
</nav>
```

## 2. 🏗️ 导航栏基本结构


### 2.1 最简单的导航栏


```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container">
    <!-- 品牌Logo -->
    <a class="navbar-brand" href="#">我的网站</a>
    
    <!-- 导航菜单 -->
    <ul class="navbar-nav">
      <li class="nav-item">
        <a class="nav-link" href="#">首页</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">关于</a>
      </li>
    </ul>
  </div>
</nav>
```

### 2.2 结构解析


```
导航栏结构层次：
┌─ nav.navbar ────────────────────────────────┐
│  ┌─ .container ─────────────────────────┐   │
│  │  ┌─ .navbar-brand ─┐                │   │
│  │  │    Logo/网站名   │                │   │
│  │  └─────────────────┘                │   │
│  │                                     │   │
│  │  ┌─ .navbar-nav ──────────────────┐  │   │
│  │  │  ┌─ .nav-item ─┐  ┌─ .nav-item │  │   │
│  │  │  │  .nav-link  │  │  .nav-link │  │   │
│  │  │  └─────────────┘  └───────────  │  │   │
│  │  └────────────────────────────────┘  │   │
│  └─────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
```

**基础类名理解**：
- `navbar`：告诉浏览器"这是一个导航栏"
- `container`：内容居中对齐，左右留白
- `navbar-brand`：品牌Logo或网站名的位置
- `navbar-nav`：导航菜单的容器
- `nav-item`：每个菜单项的容器
- `nav-link`：菜单项里的链接

## 3. 🔧 导航栏核心组件


### 3.1 navbar-brand（品牌区域）


**作用**：放置Logo、网站名或品牌标识

```html
<!-- 纯文字品牌 -->
<a class="navbar-brand" href="#">我的网站</a>

<!-- 图片Logo -->
<a class="navbar-brand" href="#">
  <img src="logo.png" alt="Logo" width="30" height="30">
</a>

<!-- 图片+文字组合 -->
<a class="navbar-brand" href="#">
  <img src="logo.png" alt="Logo" width="30" height="30">
  我的网站
</a>
```

**显示效果**：
```
┌─ 品牌区域 ─────────────┐
│ 🏠 我的网站            │  ← 通常在导航栏左侧
└───────────────────────┘
```

### 3.2 navbar-nav（导航菜单）


**作用**：放置主要的导航链接

```html
<ul class="navbar-nav">
  <li class="nav-item">
    <a class="nav-link active" href="#">首页</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">产品</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">服务</a>
  </li>
</ul>
```

**常用状态**：
- `active`：当前活跃页面（高亮显示）
- `disabled`：禁用的链接（灰色不可点击）

```html
<a class="nav-link active" href="#">当前页面</a>
<a class="nav-link disabled" href="#">暂未开放</a>
```

### 3.3 navbar-toggler（折叠按钮）


**作用**：手机屏幕上显示的"汉堡菜单"按钮

```html
<!-- 折叠按钮 -->
<button class="navbar-toggler" type="button" 
        data-bs-toggle="collapse" 
        data-bs-target="#navbarNav">
  <span class="navbar-toggler-icon"></span>
</button>

<!-- 可折叠的内容 -->
<div class="collapse navbar-collapse" id="navbarNav">
  <ul class="navbar-nav">
    <li class="nav-item">
      <a class="nav-link" href="#">首页</a>
    </li>
  </ul>
</div>
```

**工作原理**：
```
桌面端：  Logo  首页  产品  服务  关于
              ↓ 屏幕变小
手机端：  Logo  ☰    (点击☰展开菜单)
```

## 4. 📱 响应式导航栏


### 4.1 navbar-expand控制展开


**核心概念**：控制在什么屏幕尺寸下导航栏自动展开

| 类名 | 展开断点 | 说明 |
|------|---------|------|
| `navbar-expand-sm` | ≥576px | 小屏幕及以上展开 |
| `navbar-expand-md` | ≥768px | 中等屏幕及以上展开 |
| `navbar-expand-lg` | ≥992px | **最常用**，大屏幕展开 |
| `navbar-expand-xl` | ≥1200px | 超大屏幕展开 |
| `navbar-expand` | 始终展开 | 所有尺寸都展开 |

```html
<!-- 在992px以上展开，以下折叠 -->
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <!-- 导航内容 -->
</nav>
```

### 4.2 完整响应式导航栏


```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container">
    <!-- 品牌 -->
    <a class="navbar-brand" href="#">我的网站</a>
    
    <!-- 手机端折叠按钮 -->
    <button class="navbar-toggler" type="button" 
            data-bs-toggle="collapse" 
            data-bs-target="#navbarNav">
      <span class="navbar-toggler-icon"></span>
    </button>
    
    <!-- 可折叠内容 -->
    <div class="collapse navbar-collapse" id="navbarNav">
      <ul class="navbar-nav ms-auto">
        <li class="nav-item">
          <a class="nav-link active" href="#">首页</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">产品</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">关于</a>
        </li>
      </ul>
    </div>
  </div>
</nav>
```

**响应式效果**：
```
大屏幕：┌─ Logo ─ 首页 ─ 产品 ─ 关于 ─┐
       
小屏幕：┌─ Logo ─────────────── ☰ ─┐
       └─ 首页                   ─┘
       └─ 产品                   ─┘  
       └─ 关于                   ─┘
```

## 5. 🎨 导航栏主题样式


### 5.1 主题组合搭配


**常用主题搭配**：

```html
<!-- 浅色主题 -->
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <!-- 白色背景，深色文字 -->
</nav>

<!-- 深色主题 -->
<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
  <!-- 深色背景，白色文字 -->
</nav>

<!-- 彩色主题 -->
<nav class="navbar navbar-expand-lg navbar-dark bg-primary">
  <!-- 蓝色背景，白色文字 -->
</nav>
```

### 5.2 主题选择指南


| 背景色 | 推荐主题 | 效果预览 |
|--------|---------|----------|
| `bg-light` | `navbar-light` | 白色背景 + 深色文字 |
| `bg-white` | `navbar-light` | 纯白背景 + 深色文字 |
| `bg-dark` | `navbar-dark` | 深色背景 + 白色文字 |
| `bg-primary` | `navbar-dark` | 蓝色背景 + 白色文字 |
| `bg-success` | `navbar-dark` | 绿色背景 + 白色文字 |

**记忆技巧**：
- **浅色背景** → 用 `navbar-light`（深色文字看得清）
- **深色背景** → 用 `navbar-dark`（白色文字看得清）

### 5.3 自定义颜色


```html
<!-- 使用自定义背景色 -->
<nav class="navbar navbar-expand-lg navbar-dark" 
     style="background-color: #e3f2fd;">
  <!-- 内容 -->
</nav>
```

## 6. 📍 导航栏位置设置


### 6.1 固定位置类型


**位置类型对比**：

```html
<!-- 固定在顶部 -->
<nav class="navbar fixed-top navbar-light bg-light">
  <!-- 始终在页面顶部，滚动时不消失 -->
</nav>

<!-- 固定在底部 -->
<nav class="navbar fixed-bottom navbar-light bg-light">
  <!-- 始终在页面底部 -->
</nav>

<!-- 粘性顶部 -->
<nav class="navbar sticky-top navbar-light bg-light">
  <!-- 滚动到导航栏时粘在顶部 -->
</nav>

<!-- 默认位置 -->
<nav class="navbar navbar-light bg-light">
  <!-- 正常文档流中，会随页面滚动 -->
</nav>
```

### 6.2 位置效果说明


```
fixed-top：     ┌─ 导航栏 ─┐ ← 始终在顶部
               │  内容区  │
               │  内容区  │   滚动时导航栏
               │  内容区  │   不会消失
               └─────────┘

sticky-top：    │  内容区  │
               ┌─ 导航栏 ─┐ ← 滚动到这里时
               │  内容区  │   粘在顶部
               │  内容区  │
               └─────────┘

fixed-bottom：  │  内容区  │
               │  内容区  │
               │  内容区  │
               └─ 导航栏 ─┘ ← 始终在底部
```

**使用建议**：
- `fixed-top`：**最常用**，适合主导航
- `sticky-top`：适合二级导航或子页面导航
- `fixed-bottom`：适合移动端的底部导航

## 7. 📝 导航栏内表单


### 7.1 搜索框示例


```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container">
    <a class="navbar-brand" href="#">网站</a>
    
    <!-- 导航栏内的搜索表单 -->
    <form class="d-flex">
      <input class="form-control me-2" type="search" 
             placeholder="搜索..." aria-label="Search">
      <button class="btn btn-outline-success" type="submit">
        搜索
      </button>
    </form>
  </div>
</nav>
```

### 7.2 登录表单示例


```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container">
    <a class="navbar-brand" href="#">网站</a>
    
    <!-- 导航栏内的登录表单 -->
    <form class="d-flex">
      <input class="form-control me-2" type="text" 
             placeholder="用户名">
      <input class="form-control me-2" type="password" 
             placeholder="密码">
      <button class="btn btn-primary" type="submit">
        登录
      </button>
    </form>
  </div>
</nav>
```

### 7.3 表单布局技巧


**关键类名**：
- `d-flex`：让表单元素水平排列
- `me-2`：给元素右侧加一点间距
- `ms-auto`：把表单推到导航栏右侧

```html
<!-- 表单居右显示 -->
<form class="d-flex ms-auto">
  <!-- 表单内容 -->
</form>
```

## 8. 🍞 面包屑导航


### 8.1 什么是面包屑导航


**简单理解**：面包屑就是告诉用户"你现在在网站的什么位置"，像童话里的面包屑路径。

```
面包屑示例：
首页 > 产品分类 > 手机 > iPhone > iPhone 15
```

**作用**：
- 🎯 **位置指示**：用户知道当前在哪里
- 🔙 **快速返回**：点击任一层级快速跳转
- 🔍 **SEO友好**：搜索引擎更好理解网站结构

### 8.2 基础面包屑


```html
<nav aria-label="breadcrumb">
  <ol class="breadcrumb">
    <li class="breadcrumb-item">
      <a href="#">首页</a>
    </li>
    <li class="breadcrumb-item">
      <a href="#">产品</a>
    </li>
    <li class="breadcrumb-item active" aria-current="page">
      iPhone 15
    </li>
  </ol>
</nav>
```

**显示效果**：
```
首页 > 产品 > iPhone 15
 ↑     ↑       ↑
链接  链接   当前页面(不可点击)
```

### 8.3 面包屑样式定制


```html
<!-- 自定义分隔符 -->
<style>
.breadcrumb-item + .breadcrumb-item::before {
  content: "→";  /* 使用箭头代替默认的 > */
}
</style>

<nav aria-label="breadcrumb">
  <ol class="breadcrumb">
    <li class="breadcrumb-item"><a href="#">首页</a></li>
    <li class="breadcrumb-item"><a href="#">产品</a></li>
    <li class="breadcrumb-item active">iPhone 15</li>
  </ol>
</nav>
```

### 8.4 实用面包屑模板


```html
<!-- 电商网站面包屑 -->
<nav aria-label="breadcrumb">
  <ol class="breadcrumb">
    <li class="breadcrumb-item"><a href="/">首页</a></li>
    <li class="breadcrumb-item"><a href="/category">数码产品</a></li>
    <li class="breadcrumb-item"><a href="/category/phone">手机</a></li>
    <li class="breadcrumb-item active">苹果 iPhone 15</li>
  </ol>
</nav>

<!-- 企业网站面包屑 -->
<nav aria-label="breadcrumb">
  <ol class="breadcrumb">
    <li class="breadcrumb-item"><a href="/">首页</a></li>
    <li class="breadcrumb-item"><a href="/about">关于我们</a></li>
    <li class="breadcrumb-item active">公司历史</li>
  </ol>
</nav>
```

## 9. 💡 实用技巧总结


### 9.1 导航栏最佳实践


**📋 设计原则**：
```
✅ 简洁明了：导航项不超过7个
✅ 层次清晰：重要链接放左边，次要的放右边
✅ 响应式：确保手机端体验良好
✅ 一致性：全站导航保持统一风格
```

**🔧 常用布局模式**：
```html
<!-- 经典布局：Logo左，菜单右 -->
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container">
    <a class="navbar-brand" href="#">Logo</a>
    <div class="collapse navbar-collapse">
      <ul class="navbar-nav ms-auto">
        <li class="nav-item"><a class="nav-link" href="#">首页</a></li>
        <li class="nav-item"><a class="nav-link" href="#">关于</a></li>
      </ul>
    </div>
  </div>
</nav>
```

### 9.2 常见问题解决


**问题1：手机端菜单不展开**
```html
<!-- 确保引入Bootstrap的JS文件 -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

**问题2：导航栏遮挡内容**
```css
/* 使用fixed-top时，给body加上顶部边距 */
body {
  padding-top: 70px; /* 根据导航栏高度调整 */
}
```

**问题3：导航项对齐问题**
```html
<!-- 左对齐 -->
<ul class="navbar-nav me-auto">

<!-- 右对齐 -->
<ul class="navbar-nav ms-auto">

<!-- 居中对齐 -->
<ul class="navbar-nav mx-auto">
```

### 9.3 核心知识点回顾


**🎯 必记的类名**：
- `navbar`：导航栏容器
- `navbar-brand`：品牌Logo区域
- `navbar-nav`：导航菜单容器
- `navbar-toggler`：手机端折叠按钮
- `navbar-expand-lg`：响应式展开控制
- `navbar-light/dark`：主题控制

**📱 响应式要点**：
- 使用`navbar-expand-*`控制展开断点
- 添加`navbar-toggler`折叠按钮
- 用`collapse navbar-collapse`包装可折叠内容

**🎨 样式搭配**：
- 浅色背景 + `navbar-light`
- 深色背景 + `navbar-dark`
- 使用`fixed-top`创建固定导航栏

**记忆口诀**：
```
导航栏来navbar，品牌logo是brand
菜单容器nav结尾，响应展开expand
浅色light深色dark，固定位置加fixed
面包屑用breadcrumb，层级清晰好导航
```
