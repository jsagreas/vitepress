## 1. 🐍 语言基础与进阶


### 1.1 核心语言特性

**必须精通**：
```python
# 高级数据结构
collections: defaultdict, Counter, OrderedDict, deque
functools: lru_cache, partial, singledispatch, wraps
itertools: chain, combinations, groupby, islice

# 元编程
metaclass: 类的创建机制，自定义行为
descriptor: __get__, __set__, __delete__
decorator: 函数装饰器、类装饰器、参数化装饰器
```

**深度理解要求**：
- **内存管理**：引用计数、垃圾回收机制、内存池
- **GIL机制**：全局解释器锁的影响和规避策略  
- **字节码**：理解Python执行原理，性能优化基础
- **C扩展**：Cython、ctypes与底层交互

### 1.2 异步编程体系

```python
# asyncio核心
async/await: 协程语法
Event Loop: 事件循环机制
asyncio.gather(): 并发执行
asyncio.create_task(): 任务创建
asyncio.Queue: 异步队列

# 实际应用
aiohttp: 异步HTTP客户端/服务端
aioredis: 异步Redis客户端
asyncpg: 异步PostgreSQL客户端
uvloop: 高性能事件循环
```

**架构级应用**：
- **微服务通信**：异步HTTP调用，服务间解耦
- **高并发处理**：WebSocket长连接，实时数据推送
- **IO密集型任务**：文件处理，网络爬虫，数据同步

### 1.3 并发与并行

```python
# 多线程
threading: Thread, ThreadPoolExecutor, Lock, RLock
concurrent.futures: ThreadPoolExecutor, ProcessPoolExecutor
queue: Queue, PriorityQueue, LifoQueue

# 多进程  
multiprocessing: Process, Pool, Manager, Pipe
shared_memory: 共享内存(Python 3.8+)

# 协程
asyncio: 单线程并发
gevent: greenlet协程库
```

**选择策略**：
- **IO密集型**：asyncio > threading > multiprocessing
- **CPU密集型**：multiprocessing > threading > asyncio
- **混合场景**：asyncio + ProcessPoolExecutor
