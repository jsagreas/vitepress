## 1. 🛠️ 基础技能层


### 1.1 操作系统基础

**Linux系统核心**：
```
🔸 Linux发行版掌握
  • CentOS/RHEL：企业级主流选择
  • Ubuntu：云原生常用系统
  • Debian：稳定性优先场景

**存储架构设计**：
```

  • LVM管理：逻辑卷扩容、快照功能
  • RAID配置：硬RAID vs 软RAID选择

🔸 分布式存储系统

  • HDFS：大数据存储系统
  • MinIO：云原生对象存储
  • FastDFS：小文件存储系统

**数据备份与恢复**：
```
🔸 备份策略设计
  • 3-2-1备份原则：3份副本、2种介质、1处异地
  • 增量备份：rsync、数据库binlog
  • 快照备份：LVM快照、存储阵列快照
  • 云备份：对象存储、跨地域备份
  • 备份验证：定期恢复测试

🔸 灾难恢复规划

  • 异地容灾：主备站点、数据同步
  • 故障演练：定期灾难恢复演练
  • 文档维护：操作手册、联系方式

## 5. 📊 监控与可观测性


### 5.1 系统监控体系

**Prometheus监控生态**：
```
🔸 Prometheus核心组件
  • Prometheus Server：时序数据存储、查询引擎
  • Alertmanager：告警路由、告警抑制
  • Pushgateway：短期任务指标推送
  • Exporter：各类系统指标暴露
  • Grafana：数据可视化、仪表板

🔸 监控指标分类
  • 系统指标：CPU、内存、磁盘、网络
  • 应用指标：QPS、延迟、错误率、并发数
  • 基础设施：数据库、缓存、消息队列
  • 业务指标：用户活跃、交易量、转化率
  • 自定义指标：业务特定、性能指标

🔸 PromQL查询语言
  • 基础查询：指标名、标签筛选
  • 聚合函数：sum、avg、max、min
  • 时间序列：rate、increase、delta
  • 告警规则：阈值设定、持续时间
