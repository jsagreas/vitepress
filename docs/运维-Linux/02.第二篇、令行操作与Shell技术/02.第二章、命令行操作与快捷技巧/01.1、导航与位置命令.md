## 1. 🧭 基础导航命令


### 1.1 pwd - 显示当前工作目录


**🔸 核心概念**
```
pwd = Print Working Directory（打印工作目录）
作用：告诉你现在在文件系统的哪个位置
就像问"我现在在哪里？"
```

**💡 基本使用**
```bash
# 显示当前目录的完整路径
pwd
# 输出示例：/home/username/Documents

# 显示物理路径（解析符号链接）
pwd -P
# 如果当前目录是符号链接，显示真实路径
```

**🎯 实用场景**
- **确认位置**：执行命令前确认自己在正确目录
- **脚本编写**：获取脚本执行时的当前目录
- **路径复制**：快速获取当前目录路径用于其他操作

### 1.2 cd - 目录切换神器


**🔸 核心概念**
```
cd = Change Directory（更改目录）
作用：在文件系统中移动位置，就像走路一样
支持绝对路径和相对路径两种方式
```

**🚀 基本语法与示例**
```bash
# 绝对路径切换（从根目录开始的完整路径）
cd /home/username/Documents
cd /var/log
cd /usr/local/bin

# 相对路径切换（基于当前位置）
cd Documents          # 进入当前目录下的Documents
cd ../                # 返回上一级目录
cd ../../             # 返回上两级目录
cd ./subfolder        # 进入当前目录下的subfolder
```

**⚡ 快捷技巧**
```bash
# 快速跳转技巧
cd ~                  # 回到家目录（等同于cd $HOME）
cd                    # 不加参数也是回到家目录
cd -                  # 切换到上一次访问的目录
cd ~username          # 切换到指定用户的家目录

# 实用示例
cd ~/Documents        # 切换到家目录下的Documents
cd ~/.config          # 切换到配置文件目录
```

**📊 路径类型对比**

| 路径类型 | **特征** | **示例** | **使用场景** |
|---------|---------|----------|-------------|
| **绝对路径** | `以/开头，完整路径` | `/home/user/docs` | `精确定位，脚本中使用` |
| **相对路径** | `基于当前位置` | `docs/file.txt` | `日常操作，灵活方便` |
| **家目录** | `以~开头` | `~/Documents` | `个人文件快速访问` |

## 2. 📂 文件列表查看


### 2.1 ls - 文件列表显示命令


**🔸 核心概念**
```
ls = List（列出）
作用：显示目录中的文件和文件夹
就像打开抽屉看里面有什么东西
```

### 2.2 基础用法


**💻 简单列表**
```bash
# 基本列表显示
ls
# 输出：file1.txt  folder1  folder2  image.jpg

# 显示指定目录内容
ls /home/username
ls ~/Documents
```

### 2.3 重要参数详解


**📋 -l 参数：详细信息显示**
```bash
ls -l
# 显示详细信息，包括权限、所有者、大小、修改时间

# 输出格式解读：
# drwxr-xr-x  2  user  group  4096  Mar 15 10:30  folder1
# -rw-r--r--  1  user  group  1024  Mar 15 09:15  file1.txt
#
# d = 目录，- = 文件
# rwxr-xr-x = 权限信息
# 2 = 链接数
# user group = 所有者和组
# 4096 = 文件大小（字节）
# Mar 15 10:30 = 修改时间
```

**🔍 -a 参数：显示隐藏文件**
```bash
ls -a
# 显示所有文件，包括以.开头的隐藏文件

# 常见隐藏文件：
# .bashrc    # Bash配置文件
# .profile   # 用户配置文件
# .ssh/      # SSH密钥目录
# .gitignore # Git忽略文件
```

**📏 -h 参数：人性化大小显示**
```bash
ls -lh
# 文件大小用K, M, G等单位显示，而不是字节数

# 对比效果：
# 不用-h：  -rw-r--r--  1 user group     1048576 Mar 15 file.txt
# 使用-h：  -rw-r--r--  1 user group        1.0M Mar 15 file.txt
```

**⏰ -t 参数：按时间排序**
```bash
ls -lt
# 按修改时间排序，最新的文件在前面

ls -ltr
# 加上-r参数反向排序，最旧的在前面
```

### 2.4 常用组合参数


```bash
# 最常用的组合
ls -la          # 显示所有文件的详细信息
ls -lah         # 详细信息 + 隐藏文件 + 人性化大小
ls -lath        # 上述基础上按时间排序

# 实用示例
ls -la ~/       # 查看家目录所有文件详情
ls -lh /var/log # 查看日志目录文件大小
ls -lt *.txt    # 显示当前目录txt文件按时间排序
```

**🎯 实际应用场景**
```bash
# 场景1：检查脚本权限
ls -l script.sh
# 确认脚本是否有执行权限

# 场景2：查找最新日志
ls -lt /var/log/
# 找出最近修改的日志文件

# 场景3：查看隐藏配置
ls -la ~/
# 检查家目录中的配置文件
```

## 3. 🔍 命令位置查找


### 3.1 which - 查找命令的可执行文件位置


**🔸 核心概念**
```
which：在PATH环境变量中查找命令
作用：告诉你当前使用的命令来自哪个文件
就像问"这个工具放在哪个工具箱里？"
```

**💻 基本使用**
```bash
# 查找常用命令位置
which ls
# 输出：/bin/ls

which python3
# 输出：/usr/bin/python3

which gcc
# 输出：/usr/bin/gcc 或者 gcc not found
```

### 3.2 whereis - 全面位置搜索


**🔸 核心概念**
```
whereis：查找命令的多个相关文件位置
包括：可执行文件、源码、帮助文档
搜索范围更广，信息更全面
```

**📊 功能对比**
```bash
# whereis 搜索结果更全面
whereis ls
# 输出：ls: /bin/ls /usr/share/man/man1/ls.1.gz

whereis python3
# 输出：python3: /usr/bin/python3 /usr/bin/python3.8 /usr/lib/python3.8 /usr/share/man/man1/python3.1.gz

# 只查找可执行文件
whereis -b python3
# 只查找帮助文档
whereis -m python3
```

### 3.3 type - 命令类型识别


**🔸 核心概念**
```
type：识别命令的具体类型
能区分：内置命令、外部命令、别名、函数
最准确地告诉你命令的真实身份
```

**🎯 实用示例**
```bash
# 检查命令类型
type ls
# 输出：ls is aliased to `ls --color=auto`
# 说明ls实际上是一个别名

type cd
# 输出：cd is a shell builtin
# 说明cd是shell内置命令

type python3
# 输出：python3 is /usr/bin/python3
# 说明python3是外部可执行文件

# 显示所有匹配项
type -a python
# 会列出所有名为python的命令
```

**📋 命令类型总结**

| 命令工具 | **主要功能** | **搜索范围** | **适用场景** |
|---------|-------------|-------------|-------------|
| **which** | `查找PATH中的命令` | `PATH环境变量` | `快速定位当前使用的命令` |
| **whereis** | `全面搜索相关文件` | `标准系统路径` | `查找命令的完整信息` |
| **type** | `识别命令类型` | `当前shell环境` | `区分内置/外部/别名命令` |

## 4. 📚 目录栈操作


### 4.1 目录栈概念理解


**🔸 什么是目录栈**
```
目录栈就像一摞书：
- 最后放上去的书在最上面（后进先出）
- 可以快速在不同目录间跳转
- 记住你访问过的目录历史
```

**💡 生活类比**
```
就像浏览器的前进后退按钮：
pushd = 把当前网页加入历史记录，并打开新网页
popd  = 关闭当前网页，返回历史记录中的上一个
dirs  = 查看浏览历史记录
```

### 4.2 pushd - 压入目录栈


**🚀 基本使用**
```bash
# 当前在 /home/user
pushd /var/log
# 进入 /var/log，同时把 /home/user 保存到栈中

pushd ~/Documents  
# 进入 ~/Documents，把 /var/log 保存到栈中

# 栈的状态（从左到右，左边是栈顶）：
# ~/Documents /var/log /home/user
```

### 4.3 popd - 弹出目录栈


**⚡ 返回操作**
```bash
# 从当前目录返回到栈顶目录
popd
# 从 ~/Documents 返回到 /var/log

popd  
# 从 /var/log 返回到 /home/user
```

### 4.4 dirs - 查看目录栈


**📋 栈状态查看**
```bash
# 显示目录栈内容
dirs
# 输出：~/Documents /var/log /home/user

# 带行号显示
dirs -v
# 输出：
# 0  ~/Documents
# 1  /var/log  
# 2  /home/user

# 只显示路径，不显示~简写
dirs -l
```

**🎯 实际应用场景**
```bash
# 场景：在多个项目目录间切换
cd ~/projects/web-app
pushd ~/projects/mobile-app    # 切换到mobile项目
pushd ~/projects/api-server    # 切换到API项目

# 现在可以快速返回
popd    # 回到mobile-app
popd    # 回到web-app

# 查看当前栈状态
dirs -v
```

## 5. 🛠️ 路径处理工具


### 5.1 basename - 提取文件名


**🔸 核心功能**
```
basename：从完整路径中提取文件名
就像从地址中只取门牌号
```

**💻 实用示例**
```bash
# 提取文件名
basename /home/user/documents/file.txt
# 输出：file.txt

basename /var/log/system.log
# 输出：system.log

# 去掉文件扩展名
basename /home/user/photo.jpg .jpg
# 输出：photo

basename file.tar.gz .tar.gz
# 输出：file
```

### 5.2 dirname - 提取目录路径


**🔸 核心功能**
```
dirname：从完整路径中提取目录部分
就像从地址中只取街道名称
```

**💻 实用示例**
```bash
# 提取目录路径
dirname /home/user/documents/file.txt
# 输出：/home/user/documents

dirname /var/log/system.log  
# 输出：/var/log

# 脚本中获取脚本所在目录
script_dir=$(dirname "$0")
echo "脚本位于：$script_dir"
```

### 5.3 realpath - 解析真实路径


**🔸 核心功能**
```
realpath：解析符号链接，获取真实的绝对路径
处理相对路径，返回规范化的绝对路径
```

**💻 实用示例**
```bash
# 解析相对路径
realpath ../documents/file.txt
# 输出：/home/user/documents/file.txt

# 解析符号链接
realpath /usr/bin/python3
# 输出：/usr/bin/python3.8 (如果python3是符号链接)

# 脚本中获取真实路径
real_path=$(realpath "$0")
echo "脚本真实位置：$real_path"
```

### 5.4 readlink - 读取符号链接


**🔸 核心功能**
```
readlink：专门读取符号链接指向的目标
-f 参数可以递归解析多层链接
```

**💻 实用示例**
```bash
# 读取符号链接目标
readlink /usr/bin/python3
# 输出：python3.8

# 递归解析所有链接层级
readlink -f /usr/bin/python3
# 输出：/usr/bin/python3.8 (完整路径)
```

**🎯 路径工具对比**

| 工具 | **功能** | **输入** | **输出** |
|------|---------|----------|----------|
| **basename** | `提取文件名` | `/home/user/file.txt` | `file.txt` |
| **dirname** | `提取目录路径` | `/home/user/file.txt` | `/home/user` |
| **realpath** | `解析真实绝对路径` | `../file.txt` | `/home/user/file.txt` |
| **readlink** | `读取符号链接目标` | `link_file` | `target_file` |

## 6. 🌳 目录树显示


### 6.1 tree - 目录结构可视化


**🔸 核心概念**
```
tree：以树形结构显示目录和文件
就像画一张家族谱系图
让目录结构一目了然
```

**💻 基本使用**
```bash
# 显示当前目录树
tree

# 显示指定目录树
tree ~/Documents

# 输出示例：
# .
# ├── folder1/
# │   ├── file1.txt
# │   └── file2.txt
# ├── folder2/
# │   ├── subfolder/
# │   │   └── deep_file.txt
# │   └── image.jpg
# └── readme.txt
```

### 6.2 实用参数详解


**📏 控制显示深度**
```bash
# 只显示2层目录
tree -L 2

# 只显示目录，不显示文件
tree -d

# 显示隐藏文件
tree -a
```

**📊 文件信息显示**
```bash
# 显示文件大小
tree -s

# 显示文件权限
tree -p

# 显示最后修改时间
tree -D

# 组合使用
tree -psa  # 权限 + 大小 + 隐藏文件
```

**🎯 过滤和格式化**
```bash
# 只显示特定类型文件
tree -P "*.txt"    # 只显示txt文件
tree -I "*.log"    # 排除log文件

# 限制显示文件数量
tree | head -20    # 只显示前20行

# 保存树形结构到文件
tree > directory_structure.txt
```

### 6.3 实际应用场景


**📁 项目结构展示**
```bash
# 展示项目代码结构
tree -I "node_modules|.git|*.pyc" ~/my_project

# 输出类似：
# my_project/
# ├── src/
# │   ├── main.py
# │   ├── utils.py
# │   └── config/
# │       └── settings.py
# ├── tests/
# │   └── test_main.py
# ├── README.md
# └── requirements.txt
```

**🔍 系统目录分析**
```bash
# 查看系统重要目录结构
tree -L 2 /etc | head -30

# 分析日志目录
tree -L 3 /var/log
```

## 7. 📋 实用技巧总结


### 7.1 导航效率提升技巧


**⚡ 快速跳转组合**
```bash
# 技巧1：使用别名简化常用路径
alias docs='cd ~/Documents'
alias logs='cd /var/log'
alias proj='cd ~/projects'

# 技巧2：结合pushd/popd的工作流
pushd ~/projects/app1    # 进入项目1
# 工作一会...
pushd ~/projects/app2    # 切换到项目2  
# 工作一会...
popd                     # 快速回到项目1
popd                     # 回到最初位置
```

**🔍 信息查看组合技巧**
```bash
# 完整目录信息查看
pwd && ls -lah && echo "--- 目录栈状态 ---" && dirs -v

# 快速定位命令和查看帮助
which git && whereis git && type git

# 路径分析组合
echo "当前目录：$(pwd)"
echo "父目录：$(dirname $(pwd))"  
echo "目录名：$(basename $(pwd))"
```

### 7.2 必记快捷操作


```
🔸 导航快捷键：
  cd ~     # 回家目录
  cd -     # 上一个目录
  cd ..    # 上级目录
  
🔸 查看文件：
  ls -la   # 详细列表含隐藏文件
  ls -lh   # 人性化大小显示
  ls -lt   # 按时间排序
  
🔸 路径处理：
  pwd      # 当前位置
  which    # 命令位置
  tree -L 2 # 显示目录树
```

### 7.3 实际工作场景应用


**场景1：多项目开发**
```bash
# 设置项目目录栈
pushd ~/work/project-a
pushd ~/work/project-b  
pushd ~/work/project-c

# 快速在项目间切换
dirs -v    # 查看所有项目
popd       # 返回上一个项目
```

**场景2：系统维护**
```bash
# 检查系统关键目录
ls -lh /var/log/     # 查看日志大小
tree -L 2 /etc/      # 查看配置结构
which systemctl      # 确认命令位置
```

**场景3：脚本开发**
```bash
#!/bin/bash
# 获取脚本相关路径信息
script_path=$(realpath "$0")
script_dir=$(dirname "$script_path")  
script_name=$(basename "$script_path")

echo "脚本路径：$script_path"
echo "脚本目录：$script_dir"
echo "脚本名称：$script_name"
```
