## 1、🔤 标题(Headings)


```markdown
# 一级标题

## 二级标题

## 三级标题

```

## 2、📄 换行(Line Breaks)


```markdown
这是一个Paragraphs。

这是另一段落，  
并且有换行。
```

## 3、✨ 加粗、斜体(Bold, Italics)


```markdown
**Emphasis:加粗文本**  
*Emphasis:斜体文本*  
```

::: details  效果预览

**加粗文本**   

*斜体文本*   

:::

## 4、❌ 删除线（Strikethrough）


```markdown
horizontal :
---
***
___

~~删除线文本~~
```

::: details  效果预览

---

~~删除线文本~~

:::

## 5、📝 列表(Lists)


```markdown
- 第一项
- 第二项
  - 子项
1. 第一项
1. 第二项
   1. 子项
   2. 子项
```

::: details  效果预览

- 第一项
- 第二项
  - 子项

1. 第一项
1. 第二项
   1. 子项
   2. 子项

:::

## 6、💬 引用(Blockquotes)


```markdown
> 这是一个引用。
> > 二级引用
```

::: details  效果预览

> "一句名言可以激励读者。"

> > 二级引用

:::

## 7、💻 代码块(Code Blocks)


```markdown
这是 `行内代码`。
使用三个反引号 ``` 包裹代码块
```

::: details  效果预览

这是 `行内代码`。

```python
print("hello")
```

:::

## 8、📊 表格(Tables)


```markdown
| 表头1 | 表头2 |
| ----- | ----- |
| 内容1 | 内容2 |
```

::: details  效果预览

| 姓名  | 年龄 |
| ----- | ---- |
| Alice | 25   |
| Bob   | 30   |

:::

## 9、🛡️ 转义字符


```markdown
\*   转义星号，用于粗体和斜体
\_   转义下划线，也用于粗体和斜体
\`   转义反引号，用于行内代码块
\#   转义井号，用于标题
\+   转义加号，用于无序列表
\-   转义减号，用于无序列表
\.   转义点号，用于有序列表
\!   转义感叹号，用于图片标记
\[   转义左方括号，用于链接或图片的文字
\]   转义右方括号
\(   转义左圆括号，用于链接或图片的 URL
\)   转义右圆括号
\|   转义竖线，用于表格
```

::: details  效果预览

 \*   转义星号，用于粗体和斜体

\_   转义下划线，也用于粗体和斜体

\`   转义反引号，用于行内代码块

\#   转义井号，用于标题

\+   转义加号，用于无序列表

\-   转义减号，用于无序列表

\.   转义点号，用于有序列表

\!   转义感叹号，用于图片标记

\[   转义左方括号，用于链接或图片的文字

\]   转义右方括号

\(   转义左圆括号，用于链接或图片的 URL

\)   转义右圆括号

\|   转义竖线，用于表格

:::
