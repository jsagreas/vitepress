## 1. 🔗 超链接基础


### 1.1 基本语法


**🔸 `<a>` 标签结构**
```html
<a href="目标地址">链接文本</a>
```

**📋 核心特性**
```
元素类型：行内元素
必需属性：href
默认样式：蓝色下划线
语义：创建超链接
```

### 1.2 链接状态


**🎨 CSS伪类**
```css
a:link    { color: blue; }      /* 未访问 */
a:visited { color: purple; }    /* 已访问 */
a:hover   { color: red; }       /* 鼠标悬停 */
a:active  { color: darkred; }   /* 点击时 */
```

## 2. 🎯 链接类型


### 2.1 外部链接


```html
<!-- 其他网站 -->
<a href="https://www.example.com">外部网站</a>

<!-- 邮件链接 -->
<a href="mailto:user@example.com">发送邮件</a>

<!-- 电话链接 -->
<a href="tel:13800138000">拨打电话</a>
```

### 2.2 内部链接


```html
<!-- 相对路径 -->
<a href="about.html">关于我们</a>
<a href="../products/index.html">产品页面</a>

<!-- 绝对路径 -->
<a href="/contact">联系我们</a>
```

### 2.3 锚点链接


```html
<!-- 页面内导航 -->
<a href="#section1">跳转到第一节</a>
<a href="#top">返回顶部</a>

<!-- 对应的目标 -->
<div id="section1">第一节内容</div>
<div id="top">页面顶部</div>
```

### 2.4 下载链接


```html
<!-- 直接下载 -->
<a href="file.pdf" download>下载PDF</a>

<!-- 指定文件名 -->
<a href="report.pdf" download="年度报告.pdf">下载报告</a>
```

## 3. ⚙️ 重要属性


### 3.1 target属性


| **值** | **说明** | **示例** |
|-------|---------|---------|
| `_self` | `当前窗口（默认）` | `<a href="page.html">当前窗口</a>` |
| `_blank` | `新窗口/标签页` | `<a href="page.html" target="_blank">新窗口</a>` |

### 3.2 rel属性


**🔸 安全与SEO**
```html
<!-- 外部链接安全 -->
<a href="https://external.com" 
   target="_blank" 
   rel="noopener noreferrer">外部链接</a>

<!-- SEO控制 -->
<a href="https://ads.com" rel="nofollow">广告链接</a>
<a href="https://sponsor.com" rel="sponsored">赞助链接</a>
```

### 3.3 常用属性组合


```html
<!-- 完整外部链接 -->
<a href="https://example.com" 
   target="_blank" 
   rel="noopener noreferrer"
   title="打开外部网站">
   访问网站
</a>

<!-- 下载链接 -->
<a href="document.pdf" 
   download="说明书.pdf"
   title="下载说明书(2MB)">
   下载文档
</a>
```

### 3.4 属性说明


| **属性** | **作用** | **重要性** |
|---------|---------|-----------|
| `href` | `链接目标` | `必需` |
| `target` | `打开方式` | `常用` |
| `rel` | `链接关系` | `安全必需` |
| `download` | `下载文件` | `下载时必需` |
| `title` | `提示信息` | `可访问性` |

## 4. 📋 核心要点总结


### 4.1 必知要点


```
🔸 基本语法：<a href="地址">文本</a>
🔸 外部链接：必须添加 rel="noopener noreferrer"
🔸 新窗口：target="_blank"
🔸 下载：download属性
🔸 锚点：href="#id"页面内跳转
🔸 特殊协议：mailto:、tel:
```

### 4.2 快速选择


```
内部页面    → href="page.html"
外部网站    → href="https://" + target="_blank" + rel="noopener noreferrer"
下载文件    → href="file" + download
页面内跳转  → href="#id"
发送邮件    → href="mailto:email@domain.com"
拨打电话    → href="tel:13800138000"
```

### 4.3 常见模式


**✅ 正确写法**
```html
<!-- 安全的外部链接 -->
<a href="https://external.com" target="_blank" rel="noopener noreferrer">外部链接</a>

<!-- 清晰的链接文本 -->
<a href="/products">查看所有产品</a>

<!-- 下载链接 -->
<a href="file.pdf" download>下载PDF</a>
```

**❌ 常见错误**
```html
<!-- 不安全的外部链接 -->
<a href="https://external.com" target="_blank">外部链接</a>  <!-- 缺少rel -->

<!-- 模糊的链接文本 -->
<a href="/products">点击这里</a>  <!-- 文本不明确 -->
```

**关键记忆**：
- 外部链接必加安全属性
- 链接文本要有意义  
- 下载用download属性
- 锚点用#id跳转
