## 1. 🖼️ 图片标签


### 1.1 `<img>` 基本用法


**🔸 基础语法**
```html
<img src="image.jpg" alt="图片描述">
```

**📋 核心特性**
```
元素类型：行内替换元素
必需属性：src（图片路径）、alt（替代文本）
单标签：自闭合
```

### 1.2 重要属性


```html
<!-- 基础属性 -->
<img src="photo.jpg" 
     alt="风景照片" 
     width="300" 
     height="200"
     title="鼠标悬停提示">

<!-- 懒加载 -->
<img src="image.jpg" alt="描述" loading="lazy">

<!-- 响应式图片 -->
<img src="small.jpg"
     srcset="small.jpg 300w, medium.jpg 600w, large.jpg 1200w"
     sizes="(max-width: 600px) 300px, 600px"
     alt="响应式图片">
```

### 1.3 图片格式选择


| **格式** | **特点** | **适用场景** |
|---------|---------|-------------|
| **JPEG** | `有损压缩，小文件` | `照片、复杂图像` |
| **PNG** | `无损压缩，支持透明` | `图标、简单图形` |
| **WebP** | `现代格式，高压缩` | `现代浏览器优化` |
| **SVG** | `矢量图，可缩放` | `图标、简单图形` |

### 1.4 响应式处理


```html
<!-- 方法1：srcset -->
<img src="default.jpg"
     srcset="small.jpg 480w, 
             medium.jpg 768w, 
             large.jpg 1200w"
     sizes="(max-width: 480px) 100vw,
            (max-width: 768px) 50vw,
            33vw"
     alt="响应式图片">

<!-- 方法2：picture元素 -->
<picture>
    <source media="(max-width: 480px)" srcset="mobile.jpg">
    <source media="(max-width: 768px)" srcset="tablet.jpg">
    <img src="desktop.jpg" alt="图片描述">
</picture>
```

## 2. 🎵 音频标签


### 2.1 `<audio>` 基本用法


```html
<!-- 基础音频 -->
<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    您的浏览器不支持音频播放。
</audio>
```

### 2.2 音频属性


```html
<audio controls 
       autoplay 
       loop 
       muted
       preload="metadata">
    <source src="music.mp3" type="audio/mpeg">
    <source src="music.wav" type="audio/wav">
</audio>
```

**📋 属性说明**
```
controls：显示播放控件
autoplay：自动播放（需用户交互）
loop：循环播放
muted：静音
preload：预加载（none/metadata/auto）
```

### 2.3 多格式支持


```html
<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    <source src="audio.wav" type="audio/wav">
    <p>您的浏览器不支持HTML5音频。<a href="audio.mp3">下载音频</a></p>
</audio>
```

## 3. 📹 视频标签


### 3.1 `<video>` 基本用法


```html
<!-- 基础视频 -->
<video controls width="640" height="360">
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    您的浏览器不支持视频播放。
</video>
```

### 3.2 视频属性


```html
<video controls 
       width="800" 
       height="450"
       poster="thumbnail.jpg"
       preload="metadata"
       muted
       loop>
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
</video>
```

**📋 视频专有属性**
```
poster：视频封面图
width/height：尺寸
controls：显示控件
autoplay：自动播放（移动端限制）
muted：静音（autoplay必需）
loop：循环播放
preload：预加载策略
```

### 3.3 字幕支持


```html
<video controls>
    <source src="video.mp4" type="video/mp4">
    <track kind="subtitles" 
           src="subtitles-zh.vtt" 
           srclang="zh" 
           label="中文字幕"
           default>
    <track kind="subtitles" 
           src="subtitles-en.vtt" 
           srclang="en" 
           label="English">
</video>
```

### 3.4 响应式视频


```html
<!-- CSS响应式 -->
<style>
.video-container {
    position: relative;
    width: 100%;
    height: 0;
    padding-bottom: 56.25%; /* 16:9宽高比 */
}

.video-container video {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
}
</style>

<div class="video-container">
    <video controls>
        <source src="video.mp4" type="video/mp4">
    </video>
</div>
```

## 4. 🖼️ 嵌入框架


### 4.1 `<iframe>` 基本用法


```html
<!-- 基础iframe -->
<iframe src="https://example.com" 
        width="800" 
        height="600"
        title="外部网页">
</iframe>
```

### 4.2 常见应用


```html
<!-- YouTube视频 -->
<iframe width="560" 
        height="315" 
        src="https://www.youtube.com/embed/VIDEO_ID"
        title="YouTube视频"
        frameborder="0"
        allowfullscreen>
</iframe>

<!-- 地图嵌入 -->
<iframe src="https://maps.google.com/embed?..."
        width="100%"
        height="300"
        style="border:0;"
        allowfullscreen=""
        loading="lazy">
</iframe>

<!-- 文档嵌入 -->
<iframe src="document.pdf" 
        width="100%" 
        height="500px"
        title="PDF文档">
</iframe>
```

### 4.3 安全属性


```html
<iframe src="https://trusted-site.com"
        sandbox="allow-scripts allow-same-origin"
        referrerpolicy="no-referrer"
        loading="lazy">
</iframe>
```

**🔒 sandbox属性值**
```
allow-scripts：允许脚本执行
allow-forms：允许表单提交
allow-same-origin：允许同源访问
allow-popups：允许弹窗
allow-top-navigation：允许顶层导航
```

## 5. 📋 核心要点总结


### 5.1 必知要点


```
🔸 图片：<img src="" alt=""> 必需src和alt属性
🔸 音频：<audio controls><source></audio> 多格式支持
🔸 视频：<video controls><source></video> 需要poster封面
🔸 嵌入：<iframe src=""> 注意安全sandbox属性
🔸 响应式：使用srcset或picture元素
🔸 懒加载：loading="lazy"提升性能
```

### 5.2 媒体格式建议


**🎯 推荐格式**
```
图片：
WebP > JPEG（照片）
PNG（透明图）
SVG（图标）

音频：
MP3（主流）+ OGG（备选）

视频：
MP4（主流）+ WebM（备选）
```
