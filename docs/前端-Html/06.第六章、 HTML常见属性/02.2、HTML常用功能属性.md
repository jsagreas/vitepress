## 1. 📝 表单相关属性


### 1.1 `name` 表单字段名称


**🔸 基本语法**
```html
<input type="text" name="username">
<select name="country">
    <option value="cn">中国</option>
</select>
```

**📋 核心作用**
```
表单提交：后端通过name获取数据
JavaScript：通过forms API访问
单选组：相同name的radio组成一组
```

### 1.2 `value` 表单字段值


**🔸 基本语法**
```html
<input type="text" name="username" value="张三">
<input type="radio" name="gender" value="male">男
<option value="bj">北京</option>
```

**📋 核心作用**
```
默认值：页面加载时的初始值
提交数据：表单提交时发送的值
动态修改：JavaScript可读取和修改
```

### 1.3 `placeholder` 占位符文本


**🔸 基本语法**
```html
<input type="text" placeholder="请输入用户名">
<input type="email" placeholder="user@example.com">
<textarea placeholder="请输入意见..."></textarea>
```

**📋 核心特性**
```
提示作用：为用户提供输入格式提示
自动消失：用户开始输入时消失
不提交：不会作为表单数据提交
```

### 1.4 `required` 必填验证


**🔸 基本语法**
```html
<input type="text" name="username" required>
<select name="country" required>
    <option value="">请选择</option>
    <option value="cn">中国</option>
</select>
```

**📋 核心特性**
```
布尔属性：只需要属性名
原生验证：浏览器自动验证，阻止提交
错误提示：显示默认或自定义错误信息
```

### 1.5 `disabled` 禁用控件


**🔸 基本语法**
```html
<input type="text" name="username" disabled>
<button type="submit" disabled>提交</button>
```

**📋 核心特性**
```
完全禁用：不能编辑、不能获得焦点
不提交：禁用字段不会提交到服务器
动态控制：JavaScript可启用/禁用
```

**🔸 JavaScript控制**
```html
<input type="text" id="email" disabled>
<script>
// 启用/禁用
document.getElementById('email').disabled = false;
document.getElementById('email').disabled = true;
</script>
```

## 2. 🔗 链接属性


### 2.1 `href` 链接地址


**🔸 基本语法**
```html
<a href="https://www.example.com">外部链接</a>
<a href="/about.html">站内链接</a>
<a href="#section1">锚点链接</a>
<a href="mailto:user@email.com">邮件链接</a>
<a href="tel:138-0000-0000">电话链接</a>
```

**📋 链接类型**
```
绝对URL：https://example.com
相对URL：/page.html、../file.pdf
锚点：#header
协议：mailto:、tel:、ftp://
```

### 2.2 `target` 打开方式


**🔸 基本语法**
```html
<a href="https://example.com" target="_blank">新窗口</a>
<a href="/about.html" target="_self">当前窗口</a>
```

**📋 常用值**
```
_blank：新窗口或新标签页
_self：当前窗口（默认）
_parent：父框架
_top：顶层窗口
```

### 2.3 `rel` 链接关系


**🔸 基本语法**
```html
<a href="https://external.com" target="_blank" rel="noopener noreferrer">安全外链</a>
<a href="/ads" rel="nofollow">广告链接</a>
```

**📋 常用值**
```
noopener：防止新页面访问window.opener
noreferrer：不发送referrer信息
nofollow：搜索引擎不跟踪此链接
external：外部链接标记
```

## 3. 📷 媒体属性


### 3.1 `src` 资源地址


**🔸 基本语法**
```html
<img src="/images/photo.jpg" alt="照片">
<script src="/js/app.js"></script>
<audio src="/music.mp3" controls></audio>
<iframe src="/page.html"></iframe>
```

**📋 核心特性**
```
资源路径：指向文件的URL
支持类型：相对路径、绝对路径、CDN链接
动态修改：JavaScript可修改src属性
```

### 3.2 `alt` 替代文本


**🔸 基本语法**
```html
<img src="logo.png" alt="公司Logo">
<img src="chart.jpg" alt="2024年销售数据图表">
<img src="avatar.jpg" alt="用户张三的头像">
```

**📋 核心作用**
```
可访问性：屏幕阅读器朗读内容
SEO优化：搜索引擎理解图片内容
加载失败：图片无法显示时的替代文本
HTML规范：img标签的必需属性
```

**✅ 良好的alt示例**
```html
<!-- 描述内容，不写"图片" -->
<img src="sunset.jpg" alt="海边日落景色">

<!-- 功能性图片 -->
<img src="search.svg" alt="搜索">

<!-- 装饰性图片用空alt -->
<img src="decoration.png" alt="">
```

### 3.3 `width` 和 `height` 尺寸控制


**🔸 基本语法**
```html
<img src="photo.jpg" width="300" height="200" alt="照片">
<iframe src="/page.html" width="100%" height="400"></iframe>
<canvas width="800" height="600"></canvas>
```

**📋 核心特性**
```
HTML属性：设置元素的显示尺寸
单位：默认为像素，可用CSS覆盖
性能优化：提前设置尺寸避免页面重排
响应式：通常用CSS控制，HTML设置初始值
```

**🎯 最佳实践**
```html
<!-- 固定尺寸 -->
<img src="thumbnail.jpg" width="150" height="150" alt="缩略图">

<!-- 响应式图片 -->
<img src="banner.jpg" 
     width="1200" 
     height="400" 
     style="max-width: 100%; height: auto;" 
     alt="横幅">

<!-- CSS优先 -->
<style>
.responsive-img {
    max-width: 100%;
    height: auto;
}
</style>
<img src="photo.jpg" class="responsive-img" alt="照片">
```

## 4. ⚡ 事件属性


### 4.1 `onclick` 点击事件


**🔸 基本语法**
```html
<button onclick="alert('Hello!')">点击我</button>
<div onclick="showMenu()">显示菜单</div>
<a href="#" onclick="return confirm('确定删除?')">删除</a>
```

**📋 核心特性**
```
事件触发：用户点击时执行JavaScript
返回值：return false可阻止默认行为
事件对象：可通过event参数访问
```

**✅ 实用示例**
```html
<!-- 简单操作 -->
<button onclick="this.style.display='none'">隐藏自己</button>

<!-- 确认操作 -->
<button onclick="if(confirm('确定删除?')) deleteItem()">删除</button>

<!-- 表单提交 -->
<button onclick="document.getElementById('form1').submit()">提交表单</button>
```

### 4.2 `onchange` 值改变事件


**🔸 基本语法**
```html
<select onchange="handleChange(this.value)">
    <option value="1">选项1</option>
    <option value="2">选项2</option>
</select>
<input type="text" onchange="validateInput(this)">
```

**📋 核心特性**
```
触发时机：元素值改变且失去焦点时
常用元素：input、select、textarea
this指向：事件发生的元素
```

**✅ 实用示例**
```html
<!-- 级联选择 -->
<select onchange="loadCities(this.value)">
    <option value="">选择省份</option>
    <option value="bj">北京</option>
    <option value="sh">上海</option>
</select>

<!-- 实时验证 -->
<input type="email" onchange="if(!this.checkValidity()) alert('邮箱格式错误')">

<!-- 自动保存 -->
<textarea onchange="autoSave(this.value)"></textarea>
```

### 4.3 `onload` 加载完成事件


**🔸 基本语法**
```html
<body onload="initPage()">
<img src="photo.jpg" onload="console.log('图片加载完成')" alt="照片">
<iframe src="/page.html" onload="hideLoading()"></iframe>
```

**📋 核心特性**
```
触发时机：元素或页面加载完成时
常用元素：body、img、iframe、script
页面初始化：body onload常用于页面初始化
```

**✅ 实用示例**
```html
<!-- 页面初始化 -->
<body onload="showWelcomeMessage(); loadUserData();">

<!-- 图片加载处理 -->
<img src="large-image.jpg" 
     onload="hideSpinner()" 
     onerror="showErrorImage()" 
     alt="大图片">

<!-- 脚本加载回调 -->
<script src="/api.js" onload="initAPI()"></script>
```

**🎯 现代化建议**
```html
<!-- 推荐：使用addEventListener -->
<script>
document.addEventListener('DOMContentLoaded', function() {
    // 页面DOM加载完成
    initPage();
});

window.addEventListener('load', function() {
    // 所有资源加载完成
    hideLoading();
});
</script>

<!-- HTML事件属性适合简单操作 -->
<button onclick="toggleMenu()">菜单</button>
```

## 5. 📋 核心要点总结


### 5.1 属性分类总览


| **类别** | **属性** | **核心作用** | **常用元素** |
|---------|---------|-------------|-------------|
| **表单** | `name` | `字段名称，数据提交` | `input, select, textarea` |
| **表单** | `value` | `字段值，默认内容` | `input, option, button` |
| **表单** | `placeholder` | `输入提示，格式说明` | `input, textarea` |
| **表单** | `required` | `必填验证，阻止提交` | `input, select, textarea` |
| **表单** | `disabled` | `禁用控件，不可操作` | `input, button, select` |
| **链接** | `href` | `链接地址，跳转目标` | `a, link` |
| **链接** | `target` | `打开方式，新窗口等` | `a` |
| **链接** | `rel` | `链接关系，安全控制` | `a, link` |
| **媒体** | `src` | `资源地址，文件路径` | `img, script, audio` |
| **媒体** | `alt` | `替代文本，可访问性` | `img` |
| **媒体** | `width/height` | `尺寸控制，布局优化` | `img, iframe, canvas` |
| **事件** | `onclick` | `点击响应，用户交互` | `button, div, a` |
| **事件** | `onchange` | `值改变，表单交互` | `input, select` |
| **事件** | `onload` | `加载完成，初始化` | `body, img, iframe` |

### 5.2 使用要点


**✅ 最佳实践**
```html
<!-- 完整的表单字段 -->
<input type="email" 
       name="email" 
       placeholder="user@example.com" 
       required>

<!-- 安全的外部链接 -->
<a href="https://external.com" 
   target="_blank" 
   rel="noopener noreferrer">外部链接</a>

<!-- 优化的图片 -->
<img src="photo.jpg" 
     width="300" 
     height="200" 
     alt="具体描述" 
     loading="lazy">
```

**⚠️ 常见注意点**
```
name属性：表单字段必需，用于数据提交
alt属性：img标签必需，不能省略
rel="noopener"：外部链接安全必需
事件属性：简单操作可用，复杂逻辑用addEventListener
```

**关键记忆**：
- 表单三要素：name、value、required
- 外链安全：target="_blank" + rel="noopener"
- 图片必需：src、alt缺一不可
- 事件简化：onclick最常用，onchange验证，onload初始化
