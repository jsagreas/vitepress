## 1. 💪 语义化强调标签


### 1.1 `<strong>` 重要性强调


**🔸 基本语法**
```html
<p>这是<strong>非常重要</strong>的信息。</p>
```

**📋 核心特性**
```
语义含义：表示重要性、严重性或紧急性
默认样式：粗体显示
SEO价值：搜索引擎理解重要内容
屏幕阅读器：重音朗读
```

**✅ 推荐使用场景**
```html
<p><strong>注意：</strong>本产品不支持退换货。</p>
<p>账户余额不足，请<strong>及时充值</strong>。</p>
<p><strong>重要提醒：</strong>系统将于今晚维护。</p>
```

### 1.2 `<em>` 语气强调


**🔸 基本语法**
```html
<p>我<em>真的</em>很喜欢这个产品。</p>
```

**📋 核心特性**
```
语义含义：表示语气强调、重音
默认样式：斜体显示
语音效果：改变语调朗读
使用场景：强调语气、对比重点
```

**✅ 推荐使用场景**
```html
<p>这<em>不是</em>一个好主意。</p>
<p>请在<em>明天之前</em>提交报告。</p>
<p>我们需要<em>更多</em>的时间。</p>
```

### 1.3 `<mark>` 高亮标记


**🔸 基本语法**
```html
<p>搜索结果：包含<mark>HTML</mark>的相关文档。</p>
```

**📋 核心特性**
```
语义含义：标记相关性、高亮显示
默认样式：黄色背景高亮
使用场景：搜索结果、重点标记
HTML5标签：现代浏览器支持
```

**✅ 实际应用**
```html
<!-- 搜索结果高亮 -->
<p>关于<mark>JavaScript</mark>的教程资料</p>

<!-- 重点内容标记 -->
<p>请记住这个<mark>关键概念</mark>。</p>

<!-- 更新内容标记 -->
<p>价格已更新为<mark>￥299</mark>。</p>
```

## 2. 🎨 视觉化格式标签


### 2.1 `<b>` 视觉加粗


**🔸 基本语法**
```html
<p>产品名称：<b>iPhone 15</b></p>
```

**📋 核心特性**
```
语义含义：无特殊语义，纯视觉效果
默认样式：粗体显示
使用场景：关键词、产品名称等
SEO影响：无特殊权重
```

### 2.2 `<i>` 视觉斜体


**🔸 基本语法**
```html
<p>这是一本<i>经典著作</i>。</p>
```

**📋 核心特性**
```
语义含义：无特殊语义，纯视觉效果
默认样式：斜体显示
现代用法：图标字体、特殊术语
替代方案：CSS font-style
```

### 2.3 标签对比表


| **标签** | **语义** | **默认样式** | **SEO权重** | **推荐度** |
|---------|---------|-------------|-------------|-----------|
| `<strong>` | `重要性强调` | `粗体` | `高` | `★★★★★` |
| `<b>` | `无语义` | `粗体` | `无` | `★★☆☆☆` |
| `<em>` | `语气强调` | `斜体` | `中` | `★★★★★` |
| `<i>` | `无语义` | `斜体` | `无` | `★★☆☆☆` |

## 3. ✨ 特殊文本标签


### 3.1 `<small>` 细小文本


**🔸 基本语法**
```html
<p>原价：￥399 <small>现价：￥299</small></p>
<p>版权所有 <small>© 2024 公司名称</small></p>
```

**📋 使用场景**
```
常见用途：
- 版权信息
- 法律声明
- 旁注说明
- 免责条款
```

### 3.2 `<del>` 删除标记


**🔸 基本语法**
```html
<p>原价：<del>￥399</del> 现价：￥299</p>
```

**📋 核心特性**
```
语义含义：表示已删除或过时的内容
默认样式：删除线效果
配合使用：常与<ins>标签搭配
时间属性：可添加datetime属性
```

### 3.3 `<ins>` 插入标记


**🔸 基本语法**
```html
<p>会议时间：<del>上午9点</del> <ins>上午10点</ins></p>
```

**📋 核心特性**
```
语义含义：表示新增或插入的内容
默认样式：下划线效果
版本控制：适用于文档修订
时间记录：支持datetime属性
```

### 3.4 上标与下标


**🔸 `<sup>` 上标**
```html
<p>E = mc<sup>2</sup></p>
<p>日期：12<sup>th</sup> December</p>
```

**🔸 `<sub>` 下标**
```html
<p>化学式：H<sub>2</sub>O</p>
<p>数学：log<sub>2</sub>8 = 3</p>
```

**📋 实际应用**
```
上标用途：
- 数学指数
- 序数词
- 脚注引用

下标用途：
- 化学分子式
- 数学下标
- 变量索引
```

## 4. 🤔 语义化vs视觉化选择


### 4.1 选择原则


**📊 决策流程图**
```
是否需要表达特定含义？
├─ 是 → 选择语义化标签
│   ├─ 重要性强调 → <strong>
│   ├─ 语气强调 → <em>  
│   └─ 内容标记 → <mark>
└─ 否 → 考虑视觉化标签
    ├─ 纯样式需求 → CSS
    ├─ 兼容性考虑 → <b>, <i>
    └─ 快速实现 → 视觉标签
```

### 4.2 实践建议


**✅ 推荐做法**
```html
<!-- 语义优先 -->
<p><strong>警告：</strong>系统即将重启</p>
<p>请<em>仔细</em>阅读说明</p>

<!-- 样式用CSS -->
<style>
.keyword { font-weight: bold; color: blue; }
.highlight { background: yellow; }
</style>
<p>关键词：<span class="keyword">HTML</span></p>
```

**❌ 不推荐做法**
```html
<!-- 仅为样式使用语义标签 -->
<strong>普通文本</strong> <!-- 无实际重要性 -->

<!-- 语义不当 -->
<em>产品型号</em> <!-- 产品型号不需要语气强调 -->
```

### 4.3 可访问性考虑


**🔸 屏幕阅读器效果**
```
语义标签的朗读效果：
<strong>重要</strong>    → 重音朗读
<em>强调</em>          → 语调变化  
<b>粗体</b>           → 正常朗读
<i>斜体</i>           → 正常朗读

用户体验：
语义化标签 → 更好的无障碍访问
视觉化标签 → 仅视觉效果
```

## 5. 📋 核心要点总结


### 5.1 必知要点


```
🔸 语义优先：<strong>, <em> 优于 <b>, <i>
🔸 重要性：<strong> 表示内容重要性
🔸 语气强调：<em> 表示语音语气变化
🔸 高亮标记：<mark> 用于搜索结果或重点标记
🔸 特殊文本：<small>, <del>, <ins>, <sup>, <sub> 各有特定用途
```

### 5.2 选择指南


**🎯 快速选择**
```
表达重要性 → <strong>
强调语气   → <em>
搜索高亮   → <mark>
删除内容   → <del>
新增内容   → <ins>
纯视觉效果 → CSS优先
```

### 5.3 最佳实践


**✅ 现代化建议**
```html
<!-- 结合CSS的现代写法 -->
<style>
.important { font-weight: bold; color: red; }
.emphasis { font-style: italic; color: blue; }
.highlight { background: yellow; padding: 2px; }
</style>

<p><strong class="important">重要通知</strong></p>
<p><em class="emphasis">请注意</em></p>
<p><mark class="highlight">关键信息</mark></p>
```

**关键记忆**：
- 语义优先于视觉
- strong表重要，em表强调
- 现代开发多用CSS控制样式
- 考虑SEO和可访问性
