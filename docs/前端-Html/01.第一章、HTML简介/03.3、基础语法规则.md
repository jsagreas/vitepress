## 1. 🏷️ 标签语法规则


### 1.1 标签基本结构


**🔸 开始标签与闭合标签**
```html
<!-- 基本语法格式 -->
<标签名>内容</标签名>

<!-- 实际示例 -->
<h1>这是标题</h1>
<p>这是段落内容</p>
<div>这是容器</div>
```

**🔸 自闭合标签**
```html
<!-- HTML5推荐写法 -->
<img src="image.jpg" alt="图片">
<br>
<hr>
<input type="text">
<meta charset="UTF-8">

<!-- XHTML风格（可选） -->
<img src="image.jpg" alt="图片" />
<br />
<hr />
```

### 1.2 标签嵌套规则


**✅ 正确嵌套**
```html
<div>
  <p>段落内容 <strong>加粗文字</strong> 继续内容</p>
</div>
```

**❌ 错误嵌套**
```html
<!-- 标签交叉，语法错误 -->
<p>段落内容 <strong>加粗文字</p> 继续内容</strong>
```

### 1.3 标签大小写规范


**🔸 HTML5推荐：小写标签**
```html
<!-- ✅ 推荐写法 -->
<html>
<head>
  <title>页面标题</title>
</head>
<body>
  <h1>主标题</h1>
  <p>段落内容</p>
</body>
</html>

<!-- ❌ 不推荐 -->
<HTML>
<HEAD>
  <TITLE>页面标题</TITLE>
</HEAD>
```

## 2. 📝 属性使用格式


### 2.1 属性语法规则


**🔸 基本格式**
```html
<标签名 属性名="属性值">内容</标签名>
```

**🔸 双引号规范**
```html
<!-- ✅ 标准写法：双引号 -->
<img src="image.jpg" alt="图片描述" width="300">
<a href="https://example.com" target="_blank">链接</a>

<!-- ⚠️ 可以但不推荐：单引号 -->
<img src='image.jpg' alt='图片描述'>

<!-- ❌ 错误：无引号（特殊情况除外） -->
<img src=image.jpg alt=图片>
```

### 2.2 属性值处理


**🔸 空格和特殊字符**
```html
<!-- 属性值包含空格必须用引号 -->
<p class="main content">文本</p>
<img alt="这是一张 风景图片">

<!-- 多个属性之间用空格分隔 -->
<input type="text" name="username" id="user" class="form-input">
```

### 2.3 布尔属性


**🔸 布尔属性写法**
```html
<!-- HTML5推荐写法 -->
<input type="checkbox" checked>
<input type="text" disabled>
<script src="app.js" defer></script>

<!-- 完整写法（也正确） -->
<input type="checkbox" checked="checked">
<input type="text" disabled="disabled">
```

## 3. 📐 代码格式规范


### 3.1 缩进规范


**🔸 缩进标准**
```html
<!-- 推荐：2空格缩进 -->
<html>
  <head>
    <title>页面标题</title>
  </head>
  <body>
    <div class="container">
      <h1>主标题</h1>
      <p>段落内容</p>
    </div>
  </body>
</html>

<!-- 也可接受：4空格缩进 -->
<html>
    <head>
        <title>页面标题</title>
    </head>
    <body>
        <div class="container">
            <h1>主标题</h1>
        </div>
    </body>
</html>
```

**❌ 避免混用**
```html
<!-- 不要混用Tab和空格 -->
<!-- 不要混用2空格和4空格 -->
```

### 3.2 换行规范


**🔸 块级元素换行**
```html
<!-- ✅ 推荐：块级元素独占一行 -->
<div>
  <h1>标题</h1>
  <p>段落内容</p>
  <ul>
    <li>列表项1</li>
    <li>列表项2</li>
  </ul>
</div>

<!-- ✅ 可接受：简短内联元素同行 -->
<p>文本内容 <strong>加粗</strong> 和 <em>斜体</em></p>
```

## 4. 💬 注释规范


### 4.1 注释语法


**🔸 基本格式**
```html
<!-- 这是HTML注释 -->
```

### 4.2 注释最佳实践


**🔸 结构注释**
```html
<!DOCTYPE html>
<html>
<head>
  <!-- 页面元信息 -->
  <meta charset="UTF-8">
  <title>页面标题</title>
</head>
<body>
  <!-- 页面头部 -->
  <header>
    <h1>网站标题</h1>
  </header>
  
  <!-- 主要内容区域 -->
  <main>
    <article>
      <!-- 文章内容 -->
      <h2>文章标题</h2>
      <p>文章内容...</p>
    </article>
  </main>
  
  <!-- 页面底部 -->
  <footer>
    <p>&copy; 2024 版权信息</p>
  </footer>
</body>
</html>
```

**🔸 功能注释**
```html
<!-- TODO: 添加搜索功能 -->
<div class="search-box">
  <!-- 搜索表单将在这里实现 -->
</div>

<!-- 临时隐藏的内容 -->
<!--
<div class="feature-banner">
  <p>新功能即将上线</p>
</div>
-->
```

**⚠️ 注释注意事项**
```html
<!-- ❌ 避免嵌套注释 -->
<!-- 外层注释 <!-- 内层注释 --> 结束 -->

<!-- ❌ 避免在注释中使用 -- -->
<!-- 这里不要用 -- 双横线 -->
```

## 5. 📄 文件扩展名


### 5.1 扩展名对比


| 扩展名 | **使用场景** | **推荐程度** | **说明** |
|--------|-------------|-------------|----------|
| **`.html`** | `现代开发` | `⭐⭐⭐⭐⭐` | `标准推荐，广泛支持` |
| **`.htm`** | `旧系统兼容` | `⭐⭐` | `DOS时代遗留，现在少用` |

### 5.2 选择建议


**🔸 推荐使用 `.html`**
```
文件命名示例：
├── index.html        (首页)
├── about.html        (关于页面)  
├── contact.html      (联系页面)
└── products.html     (产品页面)
```

**🔸 命名规范**
```html
<!-- ✅ 推荐：小写字母，连字符分隔 -->
product-list.html
user-profile.html
contact-us.html

<!-- ❌ 避免：空格和特殊字符 -->
product list.html
user@profile.html
```

## 6. 📋 核心要点总结


### 6.1 必知语法要点


```
🔸 标签规则：正确嵌套，小写标签，合理闭合
🔸 属性规范：双引号包围，布尔属性简写
🔸 代码格式：统一缩进(2/4空格)，合理换行
🔸 注释规范：结构清晰，避免嵌套注释
🔸 文件命名：使用.html扩展名，小写命名
```

### 6.2 记忆要点


**🔹 标签三要素**
```
开始标签 + 内容 + 结束标签 = 完整元素
自闭合标签 = 无内容元素
嵌套规则 = 后开先闭
```

**🔹 属性核心**
```
属性名="属性值"    (双引号标准)
布尔属性可简写     (checked, disabled)
多属性空格分隔     (class="a b" id="c")
```

**🔹 格式原则**
```
缩进统一：选择2空格或4空格并保持一致
换行合理：块级元素换行，内联元素可同行
注释清晰：标注结构和功能，避免嵌套
```
