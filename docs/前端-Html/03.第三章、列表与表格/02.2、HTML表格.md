## 1. 📊 表格基础结构


### 1.1 基本标签组合


**🔸 核心标签**
```html
<table>
    <tr>
        <th>表头1</th>
        <th>表头2</th>
        <th>表头3</th>
    </tr>
    <tr>
        <td>数据1</td>
        <td>数据2</td>
        <td>数据3</td>
    </tr>
</table>
```

**📋 标签含义**
```
table：表格容器
tr：表格行（Table Row）
th：表头单元格（Table Header）
td：数据单元格（Table Data）
```

### 1.2 简单表格示例


**✅ 学生成绩表**

## 2. 🏗️ 表格语义化标签


### 2.1 表格分区标签


**🔸 完整语义结构**
```html
<table>
    <caption>2024年第一季度销售报告</caption>
    
    <thead>
        <tr>
            <th>月份</th>
            <th>销售额</th>
            <th>增长率</th>
        </tr>
    </thead>
    
    <tbody>
        <tr>
            <td>1月</td>
            <td>100万</td>
            <td>+5%</td>
        </tr>
        <tr>
            <td>2月</td>
            <td>120万</td>
            <td>+20%</td>
        </tr>
        <tr>
            <td>3月</td>
            <td>110万</td>
            <td>-8%</td>
        </tr>
    </tbody>
    
    <tfoot>
        <tr>
            <th>总计</th>
            <th>330万</th>
            <th>+5.7%</th>
        </tr>
    </tfoot>
</table>
```

### 2.2 语义标签详解


**📋 各部分作用**
```
caption：表格标题，描述表格内容
thead：表格头部，包含列标题
tbody：表格主体，包含数据行
tfoot：表格底部，包含汇总信息
```

| **标签** | **语义含义** | **位置** | **样式特点** |
|---------|-------------|---------|-------------|
| `<caption>` | `表格标题` | `table开始后` | `通常居中显示` |
| `<thead>` | `表格头部` | `表格顶部` | `常用粗体字` |
| `<tbody>` | `表格主体` | `中间部分` | `主要数据区域` |
| `<tfoot>` | `表格底部` | `表格底部` | `汇总信息` |

## 3. 🔗 单元格合并与属性


### 3.1 跨列合并 `colspan`


**🔸 基本用法**
```html
<table>
    <tr>
        <th colspan="3">学生信息表</th>
    </tr>
    <tr>
        <th>姓名</th>
        <th>年龄</th>
        <th>班级</th>
    </tr>
    <tr>
        <td>张三</td>
        <td>18</td>
        <td>高三(1)班</td>

</table>
```

```html

        <tr>
```

```html
            <td>12999元</td>
            <td>20台</td>
        </tr>
    </tbody>

```

**🔸 headers属性关联**
```html
<table>

        <tr>

            <td headers="iphone q2">1200台</td>
        </tr>
```

## 4. 🎨 表格样式与响应式


### 4.1 基础表格样式


**🔸 CSS样式控制**
```html
<style>
/* 基础表格样式 */
.basic-table {
    width: 100%;
    border-collapse: collapse;
    margin-bottom: 20px;
}

.basic-table th,
.basic-table td {
    border: 1px solid #ddd;
    padding: 12px;
    text-align: left;
}

.basic-table th {
    background-color: #f5f5f5;
    font-weight: bold;
}

.basic-table tr:nth-child(even) {
    background-color: #f9f9f9;
}

.basic-table tr:hover {
    background-color: #f0f0f0;
}

/* 表格标题样式 */
.basic-table caption {
    font-size: 1.2em;
    font-weight: bold;
    margin-bottom: 10px;
    text-align: center;
}
</style>

<table class="basic-table">
    <caption>产品销售数据</caption>
    <!-- 表格内容 -->
</table>
```

### 4.2 响应式表格设计


**📱 移动端适配方案1：水平滚动**
```html
<style>
.table-container {
    overflow-x: auto;
    margin-bottom: 20px;
}

.responsive-table {
    min-width: 600px;
    width: 100%;
    border-collapse: collapse;
}

@media (max-width: 768px) {
    .table-container {
        border: 1px solid #ddd;
        border-radius: 4px;
    }
}
</style>

<div class="table-container">
    <table class="responsive-table">
        <!-- 表格内容 -->
    </table>
</div>
```

## 5. 📋 核心要点总结


### 5.1 必知要点


```
🔸 基础结构：table > tr > th/td 的嵌套关系
🔸 语义标签：caption、thead、tbody、tfoot增强语义
🔸 单元格合并：colspan跨列，rowspan跨行
🔸 可访问性：scope和headers属性提升无障碍
🔸 响应式设计：移动端需要特殊处理
```

### 5.2 表格标签层次


**🏗️ 标签嵌套关系**
```
table
├── caption (可选)
├── thead (可选)
│   └── tr
│       └── th
├── tbody (可选，但推荐)
│   └── tr
│       └── td
└── tfoot (可选)
    └── tr
        └── th/td
```

### 5.3 使用场景判断


**🎯 何时使用表格**
```
✅ 适合使用表格：
- 数据对比
- 财务报表
- 课程表
- 产品规格
- 统计数据

❌ 不适合使用表格：
- 页面布局
- 纯装饰性排列
- 简单的键值对
- 响应式要求高的内容
```

### 5.4 最佳实践总结


**✅ 语义化表格结构**
```html
<!-- 推荐的完整结构 -->
<table>
    <caption>表格描述</caption>
    <thead>
        <tr>
            <th scope="col">列标题</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <th scope="row">行标题</th>
            <td>数据</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <th>汇总</th>
            <td>汇总数据</td>
        </tr>
    </tfoot>
</table>
```

**⚠️ 常见错误**
```html
<!-- 错误：用表格做布局 -->
<table>
    <tr>
        <td>导航</td>
        <td>内容</td>
        <td>侧边栏</td>
    </tr>
</table>

<!-- 错误：缺少必要的语义标签 -->
<table>
    <tr><td>标题</td></tr>  <!-- 应该用th -->
    <tr><td>数据</td></tr>
</table>
```

**关键记忆**：
- 表格用于数据展示，不用于布局
- thead、tbody、tfoot增强语义
- th用于标题，td用于数据
- 合并单元格要计算好行列数
- 移动端需要响应式处理
