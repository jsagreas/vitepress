## 1. 📋 无序列表


### 1.1 `<ul>` + `<li>` 基本结构


**🔸 基本语法**
```html
<ul>
    <li>列表项1</li>
    <li>列表项2</li>
    <li>列表项3</li>
</ul>
```

**📋 核心特性**
```
ul：无序列表容器（Unordered List）
li：列表项（List Item）
默认样式：项目符号（通常是圆点）
语义含义：项目顺序不重要的列表
```

### 1.2 实际应用场景


**✅ 典型用法**
```html
<!-- 导航菜单 -->
<nav>
    <ul>
        <li><a href="#home">首页</a></li>
        <li><a href="#products">产品</a></li>
        <li><a href="#contact">联系我们</a></li>
    </ul>
</nav>

<!-- 功能特性列表 -->
<ul>
    <li>响应式设计</li>
    <li>跨浏览器兼容</li>
    <li>SEO优化</li>
    <li>快速加载</li>
</ul>

<!-- 标签云 -->
<ul class="tags">
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
    <li>React</li>
</ul>
```

### 1.3 项目符号控制


**🎨 CSS样式控制**
```html
<style>
/* 默认圆点 */
.default { list-style-type: disc; }

/* 空心圆 */
.circle { list-style-type: circle; }

/* 方块 */
.square { list-style-type: square; }

/* 无符号 */
.none { list-style-type: none; }

/* 自定义符号 */
.custom::before {
    content: "→ ";
    color: blue;
}
</style>

<ul class="square">
    <li>方块符号</li>
    <li>列表项</li>
</ul>
```

## 2. 🔢 有序列表


### 2.1 `<ol>` + `<li>` 基本结构


**🔸 基本语法**
```html
<ol>
    <li>第一步</li>
    <li>第二步</li>
    <li>第三步</li>
</ol>
```

**📋 核心特性**
```
ol：有序列表容器（Ordered List）
li：列表项（List Item）
默认样式：数字编号（1, 2, 3...）
语义含义：项目顺序重要的列表
```

### 2.2 编号类型与起始值


**🔸 type属性控制编号样式**
```html
<!-- 数字编号（默认） -->
<ol type="1">
    <li>第一项</li>
    <li>第二项</li>
</ol>

<!-- 字母编号 -->
<ol type="A">
    <li>A项</li>
    <li>B项</li>
</ol>

<!-- 小写字母 -->
<ol type="a">
    <li>a项</li>
    <li>b项</li>
</ol>

<!-- 罗马数字 -->
<ol type="I">
    <li>I项</li>
    <li>II项</li>
</ol>

<!-- 小写罗马数字 -->
<ol type="i">
    <li>i项</li>
    <li>ii项</li>
</ol>
```

**🔸 start属性控制起始值**
```html
<!-- 从5开始编号 -->
<ol start="5">
    <li>第五项</li>
    <li>第六项</li>
    <li>第七项</li>
</ol>

<!-- 倒序编号 -->
<ol reversed>
    <li>最后一项</li>
    <li>倒数第二项</li>
    <li>倒数第三项</li>
</ol>
```

### 2.3 实际应用场景


**✅ 操作步骤**
```html
<h3>安装教程</h3>
<ol>
    <li>下载安装包</li>
    <li>运行安装程序</li>
    <li>选择安装路径</li>
    <li>点击安装按钮</li>
    <li>等待安装完成</li>
</ol>
```

**✅ 排行榜**
```html
<h3>销量排行榜</h3>
<ol>
    <li>iPhone 15 Pro</li>
    <li>Samsung Galaxy S24</li>
    <li>Google Pixel 8</li>
</ol>
```

**✅ 法律条款**
```html
<h3>用户协议</h3>
<ol>
    <li>用户注册
        <ol type="a">
            <li>填写真实信息</li>
            <li>验证邮箱地址</li>
            <li>设置安全密码</li>
        </ol>
    </li>
```

## 3. 📖 定义列表


### 3.1 `<dl>` + `<dt>` + `<dd>` 结构


**🔸 基本语法**
```html
<dl>
    <dt>术语1</dt>
    <dd>术语1的定义或描述</dd>
    
    <dt>术语2</dt>
    <dd>术语2的定义或描述</dd>
</dl>
```

**📋 核心特性**
```
dl：定义列表容器（Definition List）
dt：定义术语（Definition Term）
dd：定义描述（Definition Description）
语义关系：术语与其定义的对应关系
```

### 3.2 灵活的对应关系


**🔸 一对多关系**
```html
<dl>
    <dt>HTML</dt>
    <dd>超文本标记语言</dd>
    <dd>用于创建网页的标准标记语言</dd>
    <dd>由W3C组织维护和发展</dd>
</dl>
```

**🔸 多对一关系**
```html
<dl>
    <dt>前端</dt>
    <dt>Frontend</dt>
    <dt>客户端</dt>
    <dd>网站或应用程序中用户直接交互的部分</dd>
</dl>
```

**🔸 复杂对应关系**
```html
<dl>
    <dt>JavaScript</dt>
    <dd>动态编程语言</dd>
    <dd>主要用于网页开发</dd>
    
    <dt>TypeScript</dt>
    <dd>JavaScript的超集</dd>
    <dd>添加了静态类型系统</dd>
</dl>
```

## 5. 📋 核心要点总结


### 5.1 必知要点


```
🔸 无序列表：<ul> + <li>，顺序不重要的项目
🔸 有序列表：<ol> + <li>，顺序重要的项目
🔸 定义列表：<dl> + <dt> + <dd>，术语与定义对应
🔸 嵌套规则：li内可嵌套新的列表结构
🔸 语义选择：根据内容语义选择合适的列表类型
```

### 5.2 列表选择指南


**🎯 快速决策**
```
导航菜单     → <ul>
操作步骤     → <ol>
功能特性     → <ul>
排行榜       → <ol>
术语解释     → <dl>
FAQ问答     → <dl>
标签云       → <ul>
面包屑导航   → <ol>
```

### 5.3 列表类型对比


| **列表类型** | **标签组合** | **默认样式** | **适用场景** | **语义含义** |
|-------------|-------------|-------------|-------------|-------------|
| **无序列表** | `<ul><li>` | `圆点符号` | `导航菜单、特性列表` | `顺序不重要` |
| **有序列表** | `<ol><li>` | `数字编号` | `步骤说明、排行榜` | `顺序重要` |
| **定义列表** | `<dl><dt><dd>` | `术语定义格式` | `词汇表、FAQ` | `术语解释` |

### 5.4 最佳实践


**✅ 语义化使用**
```html
<!-- 正确：根据语义选择列表类型 -->
<h3>安装步骤</h3>
<ol>
    <li>下载软件</li>
    <li>运行安装程序</li>
    <li>完成安装</li>
</ol>

<h3>产品特性</h3>
<ul>
    <li>快速响应</li>
    <li>安全可靠</li>
    <li>易于使用</li>
</ul>
```

**⚠️ 常见错误**
```html
<!-- 错误：仅为缩进使用列表 -->
<ul>
    <li>这不是真正的列表项</li>
</ul>

<!-- 错误：空列表项 -->
<ul>
    <li></li>  <!-- 空内容 -->
    <li>有内容的项</li>
</ul>
```

**关键记忆**：
- ul无序，ol有序，dl定义
- 根据语义选择列表类型
- li必须在ul或ol内使用
- 嵌套时注意层级关系
