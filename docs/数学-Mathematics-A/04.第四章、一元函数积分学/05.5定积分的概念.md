##  一、曲边梯形的面积


对于连续函数$f(x)$在区间$[a, b]$上的定积分：

$$\int_{a}^{b} f(x) d x$$


其几何意义可以解释为函数曲线$y=f(x)$与$x$轴之间的 有向面积：

  - 当$f(x) \geq 0$  时，积分值表示曲线与$x$轴之间的 实际面积。

  - 当$f(x) \leq 0$  时，积分值表示 负的面积，即它是曲线下方区域的面积但带负号。

  - 当$f(x)$在$[a, b]$上部分为正，部分为负时，定积分的值是正区域面积减去负区域面积的结果，而非总面积。

##  二、定积分的概念


定义 :设函数$f(x)$在$[a,b]$上有界，在$[a,b]$中任意插入若干个分点:
$$a=x_{0}<x_{1}<x_{2}<\cdots<x_{n-1}<x_{n}=b$$
把区间$[a,b]$分成$n$个小区间:
$$[x_{0},x_{1}],[x_{1},x_{2}],\cdots,[x_{n-1},x_{n}]$$
各个小区间的长度依次为:
$$\Delta x_{1}=x_{1}-x_{0},\Delta x_{2}=x_{2}-x_{1},\cdots,\Delta x_{n}=x_{n}-x_{n-1}$$
在每个小区间$[x_{i-1},x_{i}]$上任取一点$\xi_{i}(x_{i-1}\leqslant \xi_{i}\leqslant x_{i})$，作函数值$f(\xi_{i})$与小区间长度$\Delta x_{i}$的乘积$f(\xi_{i})\Delta x_{i}(i=1,2,\cdots,n)$，并作出和：
$$S=\sum_{i=1}^{n}f(\xi_{i})\Delta x_{i}$$
记$\lambda=\max\{\Delta x_{1},\Delta x_{2},\cdots,\Delta x_{n}\}$，如果当$\lambda \to  0$时，这和的极限总存在，且与闭区间
$[a,b]$的分法及点$\xi_{i}$的取法无关，那么称这个极限$I$为函数$f(x)$在区间$[a,b]$上的定积分（简称积分），记作$\int_{a}^{b}f(x)dx$，即：
$$\int_{a}^{b}f(x)dx=\lim_{\lambda \to  0}\sum_{i=1}^{n}f(\xi_{i})\Delta x_{i}$$

在这里，$\lambda \to 0$的意思是 划分区间的最大子区间长度趋于$0$ ，更具体地说：设  $[a, b]$  被分成$n$个子区间，每个子区间的长度为$\Delta x_{i}=x_{i}-x_{i-1}$。定义$\lambda=\max \{\Delta x_{1}, \Delta x_{2}, \ldots, \Delta x_{n}\}$，即划分中最长的子区间的长度。当$\lambda arrow 0$时，意味着所有子区间的长度都变得非常小，即区间划分越来越精细。

##  三、定积分的存在定理



- 连续函数的可积性：如果函数  $f(x)$  在闭区间  $[a, b]$  上连续，则$f(x)$在$[a, b]$上黎曼可积。这是因为连续函数在闭区间上的一致连续性保证了当分割的模趋近于零时，振幅也会趋近于零，从而上和与下和的差趋近于零。

- 单调函数可积性：若函数$f(x)$在闭区间$[a, b]$上单调（递增或递减），则$f(x)$在$[a, b]$上黎曼可积。单调函数的不连续点集合是可数的（零测集），因此满足可积条件。

- 间断点有限的有界函数必定可积：如果函数$f(x)$在闭区间$[a, b]$上有界，并且只有有限个不连续点，则$f(x)$在$[a, b]$上黎曼可积。这是因为有限个不连续点构成的集合是零测集，从而满足更一般的可积条件。

- $Lebesgue$定理：有界函数$f(x)$ 在闭区间$[a, b]$上黎曼可积的充分必要条件是，其不连续点构成的集合的$Lebesgue$测度为零。例如，可数无穷个不连续点构成的集合是零测集，因此这样的函数也是黎曼可积的，也就是说间断点集合测度为零的有界函数必定可积。



##  四、测度的含义


**测度相关概念**


- 测度为零：若一个集合测度为零，那么在积分计算时，它的影响可以忽略。它意味着这个集合在数轴上＂占据的空间＂可以忽略不计，即它的＂长度＂是零，尽管集合可能有无穷多个点，但它们的总＂大小＂仍然是 $0$ 。

- 测度为零举例：
$$\begin{array}{|l|l|}
\hline \text { 集合类型 } & \text { 测度 } \\
\hline \text { 有限集合（如 }\{1,2,3\} \text { ）} & 0 \\
\hline \text { 区间（如 }[1,5] \text { ）} & b-a \\
\hline \text { 康托尔集（不可数但零测度）} & 0 \\
\hline \text { 无穷多个点的集合 } & \text { 视情况而定 } \\
\hline \text { 可数无穷多个点（如 } \mathbb{Q} \cap[0,1] \text { ）} & 0 \\
\hline
\end{array}$$




- 如果函数的 间断点集合测度为零，那么它在该区间上仍然 黎曼可积，即定积分存在。例如：
  - 有限个间断点的函数可积（因为有限集合的测度是 $0$ ）。
  - 可数无穷个间断点的函数可积（例如有理数点上的间断不会影响积分）。
  - 不可数无穷多个间断点，但测度为零的函数仍可积（如康托尔集 这样的集合）。
  - 但狄利克雷函数不可积，因为它的间断点集合是整个区间  $[0,1]$ （测度为 $1$ ）。




- 狄利克雷函数在区间$[0,1]$可积情况：


$$ \begin{array}{|c|c|c|c|c|}
\hline 函数定义 & 黎曼可积  & 勒贝格测度    & 勒贝格积分 \\
\hline  D(x)=1 （有理）， D(x)=0  （无理） & 
\begin{array}{l}
\text { 不可积 } \\
\end{array}
& 可积 & 0 \\
\hline  D(x)=1 （无理）， D(x)=0  （有理） & 
\begin{array}{l}
\text { 不可积 } \\
\end{array}
 &   可积 & 1 \\
\hline
\end{array}$$



**可数无穷相关概念**


日常语言中，“可数”通常指“能数得完”（有限），中文“可数”容易误解为“能被数尽”，但英文“$countable$”更强调“可被逐个标记”（如贴上自然数标签）。

“可数”$≠$有限：术语中的“可数”并非日常意义的“可以数完”，而是数学上“可列”（$listable$）的含义。即使无限，只要存在系统性的排列方式（如自然数的顺序），就属于可数无穷。

“可数无穷”（$countably \quad  infinite$）是数学中的一个精确术语，它与“不可数无穷”（$uncountably \quad infinite$）共同构成了对无限集合的区分，关键在于集合的元素能否与自然数建立一一对应关系。

- ​可数无穷：当且仅当它的元素可以按某种顺序排列成一个无限序列（即能与自然数集$N$一一对应）。
    - ​自然数​$（1, 2, 3, ...）$本身显然是可数无穷的。
    - ​整数$​（... -2, -1, 0, 1, 2, ...）$可以通过排列如 $0, 1, -1, 2, -2, ...$ 的方式与自然数对应。
    - ​有理数​（分数）也可以通过对角线枚举法排列成序列。
    - 可数无穷 $=$ 无限但可编号（如自然数、整数、有理数）。

- ​不可数无穷：若集合的无限元素无法与自然数一一对应，则称为不可数无穷。
    - ​实数​（$0$到1之间的所有小数）无法被逐一列出，康托尔对角线论证证明了这一点。
    - ​复数的不可数子集​（如实数轴上的区间）。
    - 不可数无穷 $=$ 无限且无法编号（如实数、复数）。



- 直观例子：
  - ​有限集合：{苹果, 香蕉, 橘子}，元素有限，显然可数。
  - ​可数无穷：想象所有书籍的$ISBN$号，虽然无限多，但每个号码都能被唯一列出。
  - ​不可数无穷：沙滩上所有沙粒的位置坐标，无法按顺序穷举。



##  五、积分的种类



**黎曼积分（$Riemann  \quad Integral$）**

基于区间划分：把区间$[a, b]$划分成$n$个小区间$[x_{i-1}, x_{i}]$，计算每个区间上某个点的函数值$f(\xi_{i})$，然后求和：
$$S_{n}=\sum_{i=1}^{n} f(\xi_{i}) \Delta x_{i}$$
最后取极限：
$$\int_{a}^{b} f(x) d x=\lim _{\lambda  \to   0} S_{n}$$

这里  $\lambda=\max \{\Delta x_{1}, \Delta x_{2}, \ldots, \Delta x_{n}\}$  是最大子区间长度，当划分越来越细  $(\lambda \to 0)$  时，若极限存在，则定义黎曼积分。

关键点：割的是自变量$x$，每个小区间选择一个点计算$f(x)$进行求和，适用于连续函数或有限间断点的函数。

**勒贝格积分（$Lebesgue \quad  Integral$）**

基于函数值的分层求和：勒贝格积分不是从$x$轴划分，而是先把$f(x)$的取值范围（$\mathbf{y}$轴）划分成多个区间，然后计算这些函数值对应的测度。
$$\int_{a}^{b} f(x) d x=\sum_{\text {取某个值的区域 }} \text { 函数值 } \times \text { 该值的测度 }$$

关键点：分割的是函数值$f(x)$的范围，再看这些值在哪些$x$上取到，求这些$x$的测度，适用于更广泛的函数，比如到处间断的函数。


**适用范围的区别**

黎曼积分：适用于连续函数或有限间断点的函数，对有限个间断点没影响，但如果间断点太多就不行（例如狄利克雷函数不可积）。

勒贝格积分：适用于更一般的函数，包括到处间断的函数，只要间断点的集合测度为 0，积分仍然存在（如狄利克雷变种函数可积）。

简言之，黎曼积分从"区间"出发分割定义域，而勒贝格积分从"函数值"出发分割值域，这种本质区别使得勒贝格积分能够处理更广泛的函数类。
