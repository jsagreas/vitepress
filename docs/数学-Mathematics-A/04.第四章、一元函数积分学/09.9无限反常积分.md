## 一、无穷区间上的反常积分


- 无限反常积分 是指 积分区间无穷 的不定积分，即被积函数在无穷远处的积分值。

- 无限反常积分的计算通常需要判断其 收敛性，即是否存在有限的极限值。 

- 先把无穷上限$+\infty$或无穷下限$-\infty$ 换成有限值$t$，计算 普通定积分，再令$t \rightarrow \pm \infty$取极限。

- 如果极限存在且有限，则积分收敛；否则积分发散。

<br/>

- 上限无界：
$$\int_{a}^{+\infty} f(x) d x=\lim _{b \rightarrow+\infty} \int_{a}^{b} f(x) d x=\left.F(x)\right|_{a} ^{+\infty}=F(+\infty)-F(a) $$



- 下限无界：
$$\int_{-\infty}^{b} f(x) d x=\lim _{a \rightarrow-\infty} \int_{a}^{b} f(x) d x $$


- 上下限无界：
$$  \left \{ \begin{aligned}   
&   \quad F(+\infty)=\lim _{x \rightarrow+\infty} F(x)   \qquad F(-\infty)=\lim _{x \rightarrow-\infty} F(x)  \\ \newline 
&  \quad  \int_{-\infty}^{+\infty} f(x) d x =\left.F(x)\right|_{-\infty} ^{+\infty}=F(+\infty)-F(-\infty)\\ \newline 
&  \quad \int_{-\infty}^{+\infty} f(x) d x  =\int_{-\infty}^{c} f(x) d x+\int_{c}^{+\infty} f(x) d x \\ \newline 
&  \quad \int_{-\infty}^{+\infty} f(x) d x =\lim _{a \rightarrow-\infty} \int_{a}^{c} f(x) d x+\lim _{b \rightarrow+\infty} \int_{c}^{b} f(x) dx 
\end{aligned} \right.$$


## 二、有奇点的反常积分

- 有奇点的反常积分是指被积函数在积分区间内某点或区间端点处无定义或无界的积分。这些积分不能用普通的黎曼积分直接计算，需要采用极限的方法处理，当被积函数
在点$c$处有奇点时：其中，当$\varepsilon \rightarrow 0$时取极限。

 - 第一个公式：这个公式处理的是区间$[a, c]$上的积分，其中$c$是右端点的奇点。由于函数在$c$处无定义或无界，我们不能直接积分到$c$，而是积分到$c$的左侧一个很小的距离$\varepsilon$，然后让$\varepsilon$趋近于 $0$ 。
 $$\int_{a}^{c} f(x) d x=\lim _{\varepsilon \rightarrow 0^{+}} \int_{a}^{c-\varepsilon} f(x) d x$$

- 第二个公式：这个公式处理的是区间$[c, b]$上的积分，其中  $c$  是左端点的奇点。同理，我们从$c$右侧的一个很小距离$\varepsilon$开始积分，然后让  $\varepsilon$  趋近于 $0$ 。

$$\int_{c}^{b} f(x) d x=\lim _{\varepsilon \rightarrow 0^{+}} \int_{c+\varepsilon}^{b} f(x) d x$$
- 合并理解：当函数  $f(x)$  在区间 $[a, b]$的内部点$c$处有奇点时，完整的积分可以表示为：
  $$\int_{a}^{b} f(x) d x=\int_{a}^{c} f(x) d x+\int_{c}^{b} f(x) d x=\lim _{\varepsilon \rightarrow 0^{+}} \int_{a}^{c-\varepsilon} f(x) d x+\lim _{\varepsilon \rightarrow 0^{+}} \int_{c+\varepsilon}^{b} f(x) d x$$
  这种处理方法可以看作是在奇点$c$处＂挖掉＂一个小区间  $(c-\varepsilon, c+\varepsilon)$，然后考察当这个小区间长度趋于$0$ 时的极限。如果这两个极限都存在且有限，我们就说这个反常积分是收敛的；如果至少有一个极限不存在或无穷大，则积分发散。

## 三、常用求解方法


- 判别法总结：

$$\begin{array}{l}
\begin{array}{|l|l|l|}
\hline \text { 方法 } & \text { 适用范围 } & \text { 结论 } \\
\hline \text { 直接计算极限 } & \text { 计算明确的积分 } & \text { 通过极限判断收敛或发散 } \\
\hline \text { 比较判别法 } & \text { 适用于正函数 } & f(x) \leq g(x) \text { ，若 } g(x) \text { 收敛，则 } f(x) \text { 也收敛 } \\
\hline \text { 极限比较法 } & \text { 适用于复杂函数 } & \text { 若 } f(x) / g(x) \rightarrow C \neq 0 \text { ，两者收敛性相同 } \\
\hline
\end{array}
\end{array}$$

- 绝对收敛与条件收敛:

  绝对收敛：如果级数$\sum_{n=1}^{\infty}\left|a_{n}\right|$收敛，则称原级数$\sum_{n=1}^{\infty} a_{n}$绝对收敛，这意味着即使将每一项取绝对值后，级数仍然收敛。绝对收敛是一种非常强的收敛性。
 
  条件收敛：如果级数$\sum_{n=1}^{\infty} a_{n}$收敛，但是级数$\sum_{n=1}^{\infty}\left|a_{n}\right|$发散，则称原级数$\sum_{n=1}^{\infty} a_{n}$条件收敛。这意味着级数的收敛性依赖于正项与负项之间的相互抵消。

  $$  \left \{ \begin{aligned}   
  &  绝对收敛：  \int_{a}^{b}|f(x)| d x  收敛  \Longrightarrow \int_{a}^{b} f(x) d x  收敛  \\ \newline 
  &  条件收敛： \int_{a}^{b} f(x) d x  收敛但  \int_{a}^{b}|f(x)| d x  发散 \\  
  \end{aligned} \right.$$


- 比较判别法：

   若$|f(x)| \leq g(x)$且 $\int_{a}^{b} g(x) d x$收敛，则$\int_{a}^{b} f(x) d x$绝对收敛。

   $p$积分法：利用已知收敛性的参考积分（如  $\large \int_{0}^{1} \frac{1}{x^{p}} d x$  或  $\large \int_{1}^{\infty} \frac{1}{x^{p}} d x$  ）。


- 极限比较法：

  若两个函数$f(x)$和$g(x)$满足：$\lim _{x \rightarrow+\infty} \frac{f(x)}{g(x)}=C \neq 0$，那么$\int_{a}^{+\infty} f(x) d x$和$\int_{a}^{+\infty} g(x) d x$的收敛性是 相同的。 

- 分部积分法：

  若$\lim _{x \rightarrow+\infty} u(x) v(x)$和$\lim _{x \rightarrow a} u(x) v(x)$存在，则：
  $$\int_{a}^{+\infty} u d v=\left.u v\right|_{a} ^{+\infty}-\int_{a}^{+\infty} v d u$$

- 对称性应用：偶函数积分可简化为：
 $$\int_{-\infty}^{+\infty} f(x) d x=2 \int_{0}^{+\infty} f(x) d x \quad  若 f(x)  为偶函数 $$



## 四、常见函数和公式


- 幂函数积分:
$$\int_{a}^{+\infty} \frac{1}{x^{p}} d x(a>0) \quad \left \{\begin{array}{ll}  发散， & p \leq 1 \\  收敛， & p>1 \end{array}\right.$$ 

- 对数幂积分:
$$\int_{e}^{+\infty} \frac{1}{x \ln ^{p} x} d x \quad   \left\{\begin{array}{ll} 发散， & p \leq 1 \\  收敛， & p>1 \end{array}\right.$$ 

- 指数函数积分：当$k>0$时收敛，结果为$\frac{1}{k}$（当$a=0$时结果为$\frac{1}{k}$）。
$$\int_{a}^{+\infty} e^{-k x} d x=\frac{1}{k} e^{-k a} \quad(k>0)$$

- 高斯积分：收敛，通过极坐标换元法证明，是概率论和统计学中的核心积分。
$$\int_{-\infty}^{\infty} e^{-x^{2}} d x=\sqrt{\pi}$$

- 伽玛函数（Г函数）：性质： $\Gamma(n)=(n-1)！$（当  $n$  为正整数）。
$$\Gamma(n)=\int_{0}^{+\infty} x^{n-1} e^{-x} d x \quad(n>0)$$
