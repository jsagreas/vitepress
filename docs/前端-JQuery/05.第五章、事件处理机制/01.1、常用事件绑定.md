## 1. 🎯 事件绑定基础概念


### 1.1 什么是事件绑定

**简单理解**：就是告诉浏览器"当用户做某个动作时，执行我写的代码"

```
用户操作        jQuery事件        执行结果
点击按钮   →   .click()    →   弹出提示框
输入内容   →   .keyup()    →   实时显示字数
鼠标悬停   →   .hover()    →   显示下拉菜单
```

### 1.2 事件绑定语法

**核心语法**：`元素选择器.事件方法(处理函数)`

```javascript
// 基本语法格式
$('选择器').事件方法(function() {
    // 事件发生时执行的代码
});

// 实际例子
$('#btn').click(function() {
    alert('按钮被点击了！');
});
```

## 2. 🖱️ 点击事件处理


### 2.1 click() 点击事件

**最常用的事件**，几乎每个网页都会用到

```javascript
// 按钮点击
$('#submit-btn').click(function() {
    alert('表单提交中...');
});

// 列表项点击
$('.menu-item').click(function() {
    console.log('你点击了：' + $(this).text());
});
```

### 2.2 实际应用场景

```javascript
// 场景1：按钮功能
$('.delete-btn').click(function() {
    if(confirm('确定要删除吗？')) {
        $(this).closest('tr').remove(); // 删除当前行
    }
});

// 场景2：选项卡切换
$('.tab-btn').click(function() {
    $('.tab-btn').removeClass('active');     // 所有按钮取消激活
    $(this).addClass('active');              // 当前按钮激活
    
    let tabId = $(this).data('tab');         // 获取对应内容ID
    $('.tab-content').hide();                // 隐藏所有内容
    $('#' + tabId).show();                   // 显示对应内容
});
```

## 3. 🐭 鼠标悬停事件


### 3.1 hover() 悬停事件

**用途**：鼠标移入移出时的效果，常用于导航菜单

```javascript
// 基本用法
$('.menu').hover(
    function() {
        // 鼠标移入时执行
        $(this).find('.submenu').show();
    },
    function() {
        // 鼠标移出时执行
        $(this).find('.submenu').hide();
    }
);
```

### 3.2 实用应用示例

```javascript
// 图片悬停效果
$('.product-img').hover(
    function() {
        $(this).css('opacity', '0.8');
    },
    function() {
        $(this).css('opacity', '1');
    }
);

// 提示信息显示
$('.help-icon').hover(
    function() {
        $('.tooltip').text($(this).data('tip')).show();
    },
    function() {
        $('.tooltip').hide();
    }
);
```

## 4. 👁️ 焦点事件控制


### 4.1 focus() 和 blur() 事件

**focus**：输入框获得焦点时  
**blur**：输入框失去焦点时

```javascript
// 输入框获得焦点
$('#username').focus(function() {
    $(this).css('border-color', '#007bff');
    $('.username-tip').show();
});

// 输入框失去焦点
$('#username').blur(function() {
    $(this).css('border-color', '#ccc');
    $('.username-tip').hide();
});
```

### 4.2 表单体验优化

```javascript
// 输入框占位符效果
$('.form-input').focus(function() {
    $(this).siblings('label').addClass('active');
}).blur(function() {
    if($(this).val() === '') {
        $(this).siblings('label').removeClass('active');
    }
});

// 密码强度提示
$('#password').focus(function() {
    $('.password-strength').show();
}).blur(function() {
    $('.password-strength').hide();
});
```

## 5. 📝 表单相关事件


### 5.1 change() 值改变事件

**触发时机**：输入框内容改变并失去焦点时

```javascript
// 下拉框选择
$('#city').change(function() {
    let city = $(this).val();
    console.log('选择的城市：' + city);
    
    // 根据城市加载区域
    loadDistricts(city);
});

// 复选框状态改变
$('#agree').change(function() {
    if($(this).is(':checked')) {
        $('#submit-btn').prop('disabled', false);
    } else {
        $('#submit-btn').prop('disabled', true);
    }
});
```

### 5.2 submit() 表单提交事件

**重要**：可以在提交前进行验证

```javascript
$('#login-form').submit(function(e) {
    let username = $('#username').val();
    let password = $('#password').val();
    
    if(username === '' || password === '') {
        alert('用户名和密码不能为空！');
        e.preventDefault(); // 阻止表单提交
        return false;
    }
    
    // 验证通过，继续提交
});
```

## 6. ⌨️ 键盘事件处理


### 6.1 keydown() 和 keyup() 事件

**keydown**：按键按下时  
**keyup**：按键松开时

```javascript
// 实时搜索
$('#search').keyup(function() {
    let keyword = $(this).val();
    if(keyword.length > 2) {
        searchProducts(keyword);
    }
});

// 回车提交
$('#message').keydown(function(e) {
    if(e.keyCode === 13) { // 回车键
        sendMessage();
    }
});
```

### 6.2 常用键盘代码

```javascript
// 常用按键对应的keyCode
const KEY_CODES = {
    ENTER: 13,    // 回车
    ESC: 27,      // 退出
    SPACE: 32,    // 空格
    UP: 38,       // 上箭头
    DOWN: 40      // 下箭头
};

// 使用示例
$('#search').keydown(function(e) {
    switch(e.keyCode) {
        case KEY_CODES.ENTER:
            doSearch();
            break;
        case KEY_CODES.ESC:
            $(this).val('').blur();
            break;
    }
});
```

## 7. 📋 核心要点总结


### 7.1 必须掌握的事件


```
🔸 click() - 点击事件，使用频率最高
🔸 hover() - 鼠标悬停，常用于菜单和提示
🔸 focus()/blur() - 焦点事件，优化表单体验
🔸 change() - 值改变事件，处理表单选择
🔸 submit() - 表单提交，数据验证必备
🔸 keyup() - 按键事件，实现实时交互
```

### 7.2 事件使用技巧


**🔹 this的使用**
```javascript
$('.btn').click(function() {
    $(this).text('已点击');  // this指向当前被点击的元素
});
```

**🔹 阻止默认行为**
```javascript
$('a').click(function(e) {
    e.preventDefault();  // 阻止链接跳转
    // 执行自定义操作
});
```

**🔹 事件对象的使用**
```javascript
$('input').keydown(function(e) {
    console.log('按键代码：' + e.keyCode);
    console.log('是否按下Ctrl：' + e.ctrlKey);
});
```

### 7.3 实际应用场景


| 事件类型 | **常见应用** | **典型场景** |
|---------|-------------|-------------|
| 🖱️ **click** | `按钮操作、菜单切换` | `提交表单、删除数据、选项卡` |
| 🐭 **hover** | `菜单展开、提示显示` | `导航下拉、图片预览、帮助提示` |
| 👁️ **focus/blur** | `输入框美化、验证提示` | `表单样式、错误提示、占位符效果` |
| 📝 **change** | `表单联动、状态切换` | `省市联动、开关控制、实时计算` |
| 📋 **submit** | `表单验证、数据提交` | `登录验证、注册检查、搜索提交` |
| ⌨️ **keyup** | `实时搜索、快捷键` | `搜索建议、字数统计、快捷操作` |

### 7.4 事件绑定最佳实践


**✅ 推荐写法**
```javascript
// 简洁明了
$('#btn').click(function() {
    doSomething();
});

// 功能封装
function handleClick() {
    // 处理逻辑
}
$('#btn').click(handleClick);
```

**❌ 避免的写法**
```javascript
// 过于复杂的内嵌逻辑
$('#btn').click(function() {
    // 几十行代码...
});
```

**核心记忆**：
- 事件绑定就是"用户做什么，我们干什么"
- click最常用，hover做特效，focus优化体验
- 善用this获取当前元素，善用e.preventDefault()阻止默认行为
- 代码简洁，逻辑清晰，功能单一
