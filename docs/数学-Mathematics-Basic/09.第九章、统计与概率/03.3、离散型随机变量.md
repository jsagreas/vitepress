**离散型随机变量:** 

如果随机变量$X$只可能取有限个或至多可列个值，则称$X$为离散型随机变量。

**离散型随机变量的分布列**

一般地,设离散型随机变量$X$可能取的值为$x_1,x_2,...,x_i,...,x_n$,$X$取每一个值$x_i (i=1,2,...,n)$的概率$P(X=x_i)=P_i$则称如下表为随机变量$X$的概率分布列,简称为$X$的分布列。

分布列中的概率$P$具有以下性质：
  -  $P_i≥0,i=1,2,...,n$
  -  $P_1+P_2+...+P_n=1$

则以下公式称为数学期望,其中$x_i$是随机变量$X$的可能取值,$p_i$是概率,$p_i≥0,i=1,2,...n,p_1+p_2+...+p_n=1$ 

$$EX=x_1p_1+x_2p_2+x_3p_3+...+x_np_n$$
   
   
数学期望的性质：
  - $EC=C \quad$($C$为常数)。
  - $E(aX+b)=aEX+b \quad$($a,b$为常数)。
  - $E(X_1+X_2)=EX_1+EX_2$。
  - 如果$X_1,X_2$,相互独立,那么$E(X_1·X_2)=EX_1·EX_2$。

关于随机变量$X$的数学期望有以下公式：  

  - 当$X$服从两点分布$B(1,p)$,$E(x)=p$
  - 当$X$服从二项分布$B(n,p)$,$E(x)=np$
  - 当$X$服从超几何分布$H(N,M,n)$,$E(x)=n\frac{M}{N}$



**离散型随机变量的均值与方差**

 方差:若离散型随机变量$X$的概率分布如下表所示:

则$(x_i-\mu)^2$描述了$x_i(i=1,2,...n)$相对于均值$\mu$的偏离程序,故以下公式:刻画了随机变量$X$与其均值$\mu$的偏离程序,我们将其称为离散型随机变量$X$的方差,记为$D(X)$或者$\sigma^2$ 

$$(x_1-\mu)^2p_1+(x_2-\mu)^2p_2+...+(x_n-\mu)^2p_n  \quad(p_i≥0，i=1,2,...,n,p_1+p_2+...+p_n=1)$$


方差的性质：
  - $DX=(x_1-EX)^2p_1+(x_2-EX)^2p_2+...+(x_n-EX)^2p_n$
  - $DX=EX^2-(EX)^2$
  - $D(aX+b)=a^2DX \quad$($a、b$是常数)
  - $D(C)=0 \quad$($C$是常数)；
  - 若$X_1,X_2,...,X_n$两两独立,则$D(X_1+X_2+...+X_n)=DX_1+DX_2+...+DX_n$


关于随机变量$X$,有以下方差计算公式：
  - 当$X$服从两点分布$B(1,p)$,$D(x)=p(1-p)$
  - 当$X$服从二项分布$B(n,p)$,$D(x)=np(1-p)$
  - 当$X$服从超几何分布$H(N,M,n)$,$D(x)=n\frac{M}{N}·(1-\frac{M}{N})·\frac{N-n}{N-1}$
