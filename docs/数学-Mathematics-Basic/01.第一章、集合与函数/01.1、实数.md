## 一、实数


- 实数（Real Numbers）：是有理数和无理数的集合，可以表示整数、分数、小数以及无限不循环小数等。

- 有理数（rational number）:有理数是指可以用两个整数的比（分母不为0）表示的数。

- 无理数（irrational number）:无理数是指不能用两个整数的比表示的数。例如$\large \pi$、$\sqrt{2}$等都是无理数。

- 超越数（Transcendental Numbers）：不能作为任何有理系数（整系数）代数方程的解的数。例如，$e$和$\large \pi$。

## 二、数轴


- 零点（Origin）：数轴的中心点通常表示为零点，代表实数0。

- 正方向（Positive Direction）：数轴的右侧代表正实数，实数值逐渐增加，符号为“+”。

- 负方向（Negative Direction）：数轴的左侧代表负实数，实数值逐渐减小，符号为“-”。

- 刻度线（Tick Marks）：数轴上会有一系列刻度线，用来表示不同实数值。

- 单位刻度（Unit Scale）：相邻刻度线之间的距离通常表示一个单位，可以是1、2、5等，具体取决于数轴的标度选择。

- 标记数字（Numerical Labels）：刻度线上通常标有数字，表示对应的实数值，以便读者理解数轴上各点的值。

- 无限延伸（Infinite Extent）：数轴在正方向和负方向都可以无限延伸，以包含所有的实数。


## 三、区间


- 开区间（Open Interval）：开区间由两个端点定义，不包括端点的值。例如，开区间$(a, b)$表示包含所有大于$a$且小于$b$的实数，但不包括$a$和$b$本身。

- 闭区间（Closed Interval）：闭区间同样由两个端点定义，但包括端点的值。例如，闭区间$[a, b]$表示包含所有大于等于$a$且小于等于$b$的实数。

- 半开半闭区间（Half-Open, Half-Closed Interval）：这种区间只包含其中一个端点的值。例如，半开半闭区间$[a, b)$表示包含所有大于等于$a$且小于$b$的实数，但不包括$b$本身。

- 无限区间（Infinite Interval）：无限区间可以是左无限、右无限或两者同时无限。例如，左无限区间$(-∞, a)$表示包含所有小于$a$的实数，而右无限区间$(b, ∞)$表示包含所有大于$b$的实数。






>区间可以使用集合表示法来表示，集合表示法通常使用大括号${}$表示集合，并使用特定的符号和条件来描述区间。以下是不同类型的区间的集合表示法：

- 开区间：一个开区间由两个实数$a$和$b$定义，其中$a < b$。它的集合表示法如下：( 这表示集合包含所有大于$a$且小于$b$的实数$x$。)

   $$\left \{ x | a < x < b \right \}$$

- 闭区间：一个闭区间由两个实数$a$和$b$定义，其中$a ≤ b$。它的集合表示法如下：(这表示集合包含所有大于等于$a$且小于等于$b$的实数$x$。)

  $$\left \{  x | a ≤ x ≤ b  \right\}$$

- 半开半闭区间：左半开半闭区间$[a, b)$的集合表示法如下：(这表示集合包含所有大于等于$a$且小于$b$的实数$x$。)

   $$\left \{  x | a ≤ x < b  \right  \}$$

- 无限区间：左无限区间$(-∞, a)$的集合表示法如下：( 这表示集合包含所有小于 $a$ 的实数 $x_0$右无限区间 (a, ∞) )

  $$\left \{  x | x < a  \right   \}$$

在集合表示法中，大括号${}$内的表达式描述了集合中的元素，竖线$|$后面的条件规定了元素必须满足的条件。

## 四、邻域


邻域（Neighborhood）是一个与某个给定点或数值相关的集合，通常是一个包含该点附近的一组数的集合。



具体来说，一个点$x$的邻域是包含$x$在内的一个集合，通常表示为$N(x)$，该集合满足以下特点：

- 存在一个正实数$\large \epsilon$（通常$\large \epsilon > 0$），使得对于所有与$x$的距离小于$\large \epsilon$的点$y$，$y$都属于邻域$N(x)$。


- 邻域$N(x)$可以用不等式来表示，通常表示为$\left \{ y | |y - x| < \large \epsilon  \right\}$(其中$|y - x|$表示$y$和$x$之间的距离)

- 换句话说，$N(x)$包含了距离$x$不超过$\large \epsilon$的所有点。


## 五、绝对值


对于实数 $x$，它的绝对值通常表示为$|x|$或$||x||$，定义如下：

- 如果$x$是正数或零，那么$|x| = x$。

- 如果$x$是负数，那么$|x| = -x$。
