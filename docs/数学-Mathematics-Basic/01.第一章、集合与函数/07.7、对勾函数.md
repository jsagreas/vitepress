## 一、对勾函数

- 定义：是指形如$f(x) = ax + \frac{b}{x}$的函数。
- 图像：对勾函数的图像是由两条渐近线$y = ax$和$x = 0$围成的两个双曲线，形似对勾。

## 二、对勾函数性质


- 定义域:$x ≠ 0$或$(-∞, 0) ∪ (0, +∞)$
- 渐近线：$y = ax$和$x = 0$。
- 奇偶性：对勾函数是奇函数。
- 值域:

   $a、b>0$,$\left ( \sqrt{\frac{b}{a} } ,+∞ \right )∪ \left ( -\sqrt{\frac{b}{a} } ,-∞  \right )$

   $a、b<0$,$\left ( \sqrt{\frac{b}{a} } ,-∞  \right ) ∪\left ( -\sqrt{\frac{b}{a} } ,+∞ \right )$

- 单调性：

  $a、b>0$,令$k=\sqrt{\frac{b}{a}}$，增区间：$\left \{   x|x≤-k  \right \}$和$\left \{   x|x≥k  \right \}$；减区间：$\left \{   x|-k≤x<0  \right \}$和$\left \{   x| 0<x ≤k \right \}$
  
  $a、b<0$,令$k=\sqrt{\frac{b}{a}}$，增区间：$\left \{   x|0<x≤k  \right \}$和$\left \{    x|-k≤x<0 \right \}$；减区间：$\left \{  x|x≥k   \right \}$和$\left \{   x| x|x≤-k  \right \}$


- 拐点：

   $a、b>0$,在第一象限内，其转折点为$\left ( \sqrt{\frac{b}{a} } ,2\sqrt{ab}  \right )$；在第三象限内，其转折点为$\left ( -\sqrt{\frac{b}{a} } ,-2\sqrt{ab}  \right )$。

   $a、b<0$,在第二象限内，其转折点为$\left ( -\sqrt{\frac{b}{a} } ,2\sqrt{ab}  \right )$；在第四象限内，其转折点为$\left ( \sqrt{\frac{b}{a} } ,-2\sqrt{ab}  \right )$。
 


1. **拐点推导过程：**
   - 当$x\in \left ( 0,+\infty  \right )$时,由均值不等式$a+b\ge 2\sqrt{ab} \left ( a>0,b>0 \right )$

   - 将$ax+\frac{b}{x}$中$ax( a>0 )$看做$a$,$\frac{b}{x}$看做$b$代入上式，得：
   $ax+\frac{b}{x} \ge 2\sqrt{ax\cdot \frac{b}{x} } =2\sqrt{ab}$
   - 当且仅当$ax=\frac{b}{x}$，即$x=\sqrt{\frac{b}{a}}$时等号成立。
   - 故当$x>0$时，对勾函数的转折点横坐标为$\frac{b}{a}$。
