## 一、二次函数的性质

- 定义：一元二次函数是指形式为$f(x)=ax^2+bx+c$的函数，其中$a、b$和$c$是实数且$a≠0$。

- 抛物线的开口方向：当$a>0$时，抛物线开口朝上；当$a<0$时，抛物线开口朝下。
- 开口大小：系数$a$决定了抛物线的开口方向和抛物线的开口程度。
- 系数$b$的影响：$b$控制抛物线的对称轴位置。对称轴的$x$坐标为$−\frac{b}{2a}$。
- 常数项$c$的影响：$c$控制抛物线与$y$轴的交点，即函数的纵截距。
-$a、b$影响：共同决定抛物线对称轴的位置$x=−\frac{b}{2a}$
- 顶点坐标：抛物线的顶点坐标为${\large  (−\frac{b}{2a},\frac{4ac-b^2}{4a} )}$。
- 交点式：$y=a(x−x_1 )(x−x_2 )$（仅限于与$x$轴有交点的抛物线）
- 顶点式：$y=a(x-h)²+k(a≠0,a、h、k$为常数$)$顶点坐标为$（h,k)$，对称轴为直线$x=h$。



## 二、韦达定理


韦达定理:
$$x_1 + x_2 = -\frac{b}{a}$$
$$x_1 \times x_2 = \frac{c}{a}$$


- 假设有两个点$(x_1, y_1)$和$(x_2, y_2)$，我们要找到一个二次函数，使得它经过这两个点。

- 首先，我们可以假设该二次函数的一般形式为$f(x) = ax^2 + bx + c$。
- 然后，我们使用这两个点的坐标，将它们带入二次函数中得到两个方程：
$$y_1 = ax_1^2 + bx_1 + c   \longrightarrow ax_1^2 + bx_1 + c - y_1 = 0$$
$$y_2 = ax_2^2 + bx_2 + c   \longrightarrow ax_2^2 + bx_2 + c - y_2 = 0$$
$$a(x-x_1)( x- x_2) = 0$$
$$a[x^2−(x_1​+x_2​)x+x_1​x_2]​=0$$
- 与原方程$ax^2+bx+c=0$进行比较，可以得到：

$x_1 + x_2 = -\frac{b}{a}$

$x_1 \times x_2 = \frac{c}{a}$
