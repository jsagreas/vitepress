## 一、函数的性质

函数在数学中有许多不同的特性和性质，以下是一些常见的函数特性：
- 定义域（Domain）：函数的定义域是指函数可以接受的输入值的集合。

- 值域（Range）：函数的值域是指函数可能产生的输出值的集合。
- 奇偶性（Odd and Even）：函数可以是奇函数、偶函数或既不奇也不偶。
- 周期性（Periodicity）：周期性函数具有周期，这意味着函数在一定间隔内重复自身。
- 单调性（Monotonicity）：单调函数是指在其定义域内，函数值随着自变量的增加而单调增加或单调减少。
- 连续性（Continuity）：连续函数是指在其定义域内没有突变或断裂的函数。
- 最大值和最小值（Maximum and Minimum）：函数可能在其定义域内具有最大值和最小值。
- 渐近线（Asymptotes）：渐近线是指函数图形在无穷远处逼近的直线或曲线。
- 对称性（Symmetry）：某些函数具有对称性，如轴对称、中心对称或旋转对称，这影响函数的图形和性质。
- 零点（Zeroes）：函数的零点是指使函数等于零的输入值，也称为根或解。零点通常用来求解方程。


## 二、函数的奇偶性


函数的奇偶性是指函数在镜像对称性质方面的特性。具体来说，一个函数的奇偶性可以通过以下方式定义：

- 奇函数（Odd Function）：如果对于函数中的每个$x$，都有$f(-x) = -f(x)$，则该函数被称为奇函数。
- 偶函数（Even Function）：如果对于函数中的每个 x，都有$f(-x) = f(x)$，则该函数被称为偶函数。
- 非奇非偶函数：如果一个函数既不是奇函数也不是偶函数，那么它被称为非奇非偶函数。

奇偶性是函数的一个重要性质，它有以下一些性质：

- 奇函数和奇函数的乘积是偶函数，奇函数和偶函数的乘积是奇函数。
- 偶函数和偶函数的乘积是偶函数。
- 奇函数和偶函数的和（或差）可能是奇函数、偶函数，或非奇非偶函数，具体取决于它们的性质和形式。



**奇函数特点**

- 中心对称性：奇函数具有中心对称性，也就是说，它的图形关于坐标原点 (0, 0) 对称。
- $f(-x) = -f(x)$：这是奇函数的定义特点。对于奇函数中的任何$x$，都满足关系式$f(-x) = -f(x)$。
- 积分性质：如果将奇函数在对称区间$[a,-a]$上积分，结果总是零，即${\Large\int} _{-a}^{a} f(x)dx=0$。
- 导数性质：奇函数的导数通常是偶函数。如果$f(x)$是奇函数，那么$f'(x)$通常是偶函数。


**偶函数特点**

- 轴对称性：偶函数具有轴对称性。
- $f(-x) = f(x)$：偶函数的定义特点是$f(-x) = f(x)$。
- 导数性质：偶函数的导数通常是奇函数。如果$f(x)$是偶函数，那么$f'(x)$通常是奇函数。


**奇偶函数运算律**
- 奇函数与奇函数的加和差仍然是奇函数。

- 偶函数与偶函数的加和差仍然是偶函数。
- 偶函数与奇函数的加和差一般情况下是非奇非偶函数。
- 两个偶函数的乘法或除法的结果通常是一个偶函数或一个常数函数。
- 当两个奇函数相乘或相除时，它们的结果通常是一个偶函数。
- 奇函数与偶函数相乘或相除时，它们的结果通常是一个奇函数。


**常见奇偶函数**
- 以下是一些常见的奇函数的示例： 
    - 正弦函数：$f(x)=sin(x)$是最典型的奇函数，因为它满足$sin(−x)=−sin(x)$
    - 立方函数：$f(x)=x^3$也是一个奇函数，因为它满足$f(−x)=−f(x)$
    - 正切函数：$tan(x)$在其周期内是奇函数，因为它满足$tan(−x)=−tan(x)$
    - 反正弦函数：$f(x)=arcsin(x)$也是一个奇函数，因为它满足$arcsin(−x)=−arcsin(x)$
    - 反正切函数：反正切函数$arctan(x)$是正切函数的反函数，也是一个奇函数。
    - 余切函数： 余切函数$cot(−x) = sin(−x)/cos(−x) ​= −sin(x)/cos(x) ​= −cot(x)$

- 以下是一些常见的偶函数的示例： 
    - 余弦函数：$cos(x)$是最典型的偶函数，因为它满足$cos(−x)=cos(x)$
    - 平方函数：$f(x)=x^2$是一个典型的偶函数，因为它满足$f(−x)=f(x)$
    - 绝对值函数：$∣x∣$也是一个偶函数，因为它满足$∣(−x)∣  =∣x∣$
    - 指数函数的平方：$e^{x^2}$也是一个偶函数，因为它满足$e^{-x^2}=e^{x^2}$
    - 正弦函数的平方：$sin ^2 (x)$是一个偶函数，因为它满足$sin^2 (−x)=sin^2 (x)$
    - 余切函数的平方：$cot ^2 (x)$是一个偶函数，因为它满足$cot ^2 (−x)=cot^2 (x)$



## 三、函数的单调性

- 定义：函数的单调性是指函数在定义域内的取值随自变量的增减而保持一致的性质。
- 唯一性：在一个给定的定义域上，一个函数要么是单调递增的，要么是单调递减的，或者是非单调的。
- 局部性：函数的单调性可以在整个定义域上成立，也可以仅在某个区间上成立。
- 连续性和可导性：函数的单调性与函数的连续性和可导性不一定相关。
- 导数的符号：函数的单调性与导数的符号有关。
- 判断方法： 导数法,图像法,定义法。


- **单调性运算律：**

  |    函数     |$f(x)$|$g(x)$|$f(x)+g(x)$|$f(x)-g(x)$|$f(x)*g(x)$|
  | :---------: | :----: | :----: | :---------: | :---------: | :---------: |
  | 单调性（1） |   ↗    |   ↗    |      ↗      |      -      |      ↗      |
  | 单调性（2） |   ↘    |   ↘    |      ↘      |      -      |      ↗      |
  | 单调性（3） |   ↗    |   ↘    |      -      |      ↗      |      ↘      |
  | 单调性（4） |   ↘    |   ↗    |      -      |      ↘      |      ↘      |

  

## 四、函数的周期性

 
- 定义：如果存在一个正数$T$，使得对于函数$f(x)$，对于任意实数$x$，满足$f(x + T) = f(x)$，则函数$f(x)$是周期函数。
- 最小正周期：如果存在一个正数$T$，使得对于任意$x∈R$，都有$f(x+T)=f(x)$，那么称$T$是$f(x)$的一个周期。
- **周期运算律：**

   如果$f(x)$和$g(x)$都是周期函数：
  - 若 $f(x)$在某个区间上的最小正周期为$T$，那么函数$f(\omega x)$在相同区间上的最小正周期为${\LARGE\frac{T}{∣\omega∣} }$。

  - 若最小正周期分别为$T1$和$T2$，则它们的和差$f(x) \pm  g(x)$也是周期函数，并且它的最小正周期为最小公倍数。
  - 若它们的周期相同（或是周期的整数倍关系），那么$f(x) * g(x)$也是一个周期函数，其周期与原函数相同。
  - 若函数$f(x)$的周期是$T$，函数$g(x)$的周期是$S$，则函数$f(x)*g(x)$的周期是$T⋅S$。
  - 非周期函数与周期函数的加减乘除都是非周期函数。
- **常见周期函数：**
  - 正弦函数（sine function）：$f(x) = A \times sin(B \times x + C)$，正弦函数以$T = 2\pi/B$为周期。
  
  - 余弦函数（cosine function）：$f(x) = A \times cos(B \times x + C)$，余弦函数以$T = 2\pi/B$为周期。
  - 正切函数（tangent function）：$f(x) = A \times tan(B\times x + C)$，正切函数以$T = \pi/B$为周期。
  - 余切函数（cotangent function）： 余切函数$f(x)=cot(x)$的周期是$\pi$。即$f(x+\pi)=f(x)$。
  - 反正弦函数/反余弦函数（arcsine/arccosine）： $f(x)=arcsin(x)$和$f(x)=arccos(x)$的周期都是$2\pi$。

## 五、函数的有界性

  - 如果存在正数$M$，使得$|f(x)|≤M$对任意$x∈D$都成立，则称函数在$D$上有界。

  - 如果这样的$M$不存在，就称函数$f(x)$在$D$上无界。
  - 此外，函数$f(x)$在$D$上有界的充分必要条件是它在$D$上既有上界也有下界。
  - 有界函数在定义域上存在上界和下界。这意味着存在两个常数$M$和$N$，对于定义域上的任意$x$，有$N ≤ f(x) ≤ M$。

- **常见有界函数：**
  - 常值函数：$f(x) = C$，其中$C$是常数。常值函数是一个有界函数，因为它的取值始终等于常数$C$。
  - 三角函数：正弦函数、余弦函数、正切函数等在有界区间上是有界函数。
  - 指数函数：指数函数在任何有限区间上都是有界的。
