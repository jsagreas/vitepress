##  一、指数概念


- 指数: 表示一个数的乘方次数,指数通常由一个底数和一个指数组成。例如，$a^n$中的$a$是底数，$n$是指数。

- 指数函数: 是以指数为自变量的函数，$y=a^x$函数($a$为常数且以$a>0，a≠1$)叫做指数函数，函数的定义域是$R$。

##  二、指数幂及其运算

  - 同底数幂相乘除:
$$a^m×a^n=a^{m+n}$$
$$a^m÷a^n=a^{m−n}$$


  - 指数为$0$幂法则:
$$a^0=a^{1−1}=\frac{a^1}{a^1} =1 \qquad  a≠0$$
  - 负整数指数幂法则:
$$a^{−k}=a^{0−k}=\frac{a^0}{a^k}=\frac{1}{a^k}   \qquad  a≠0,k∈N^∗$$
  - 正分数指数幂法则:
  $$a^{\frac{m}{n} } =\sqrt[n]{a^{m} }   \qquad  n≠0,\frac{m}{n}>0,m、n∈N^∗$$
      推导过程：

  $$ a^\frac{m}{n} =(a^m)^\frac{1}{n} =b →   [(a^m)^\frac{1}{n}  ]^n=b^n  →  b=(b^n )^\frac{1}{n}=\sqrt[n]{b^n}    \qquad  a,b>0,m、n∈Z,n>0$$

  - 负分数指数幂法则:
$$a^{-(\frac{m}{n} )} =a^{(0 -\frac{m}{n}) } =\frac{a^{0} }{a^{\frac{m}{n} } }   =\frac{1}{a^{\frac{m}{n} } }=\frac{1}{\sqrt[n]{a^{m} } } \qquad  \\  a^{m}≠0 (\sqrt[n]{a^{m} }≠0、a≠0 ), \frac{m}{n} >0,n≠0,m、n∈N^∗$$
  - 平方差公式:
$$(a+b)×(a−b)=a^2−b^2$$
  - 分数的乘方法则:
$$(\frac{a}{b} )^k=a^k×b^{-k} =\frac{a^k}{b^k}$$
  - 幂的乘方法则:
$$(a^m )^n=a^{m×n} 、 a^{m^{n} } =a^{m×m^{n-1}}  =(a^{m})^{m^{n-1} }$$
  - 积的乘方:
$$(a×b)^n=a^n×b^n  、(a×b×c)^n=a^n×b^n ×c^n$$
  - 同指数幂相乘:
$$a^n×b^n=(ab)^n$$
  - 完全平方公式:
$$(a±b)^2=a^2±2ab+b^2$$
  - 立方差公式:
$$a^3−b^3=(a−b)(a^2+ab+b^2 )$$
$$a^3−a^2 b−b^3+a^2 b=a^2 (a−b)+b(a^2−b^2 )=a^2 (a−b)+b(a+b)(a−b)$$

  - 立方和公式:
$$a^3+b^3=(a+b)(a^2−ab+b^2 )$$
$$a^3+a^2 b+b^3−a^2 b=a^2 (a+b)−b(a^2−b^2 )=a^2 (a+b)−b(a+b)(a−b)=(a+b)[a^2−b(a−b)]$$

  - 多项式平方:
$$(a+b+c)^2=a^2+b^2+c^2+2ab+2bc+2ac$$
  - 公式法:

     - 平方差公式：$\qquad \qquad \qquad  \qquad \qquad \qquad     a^2−b^2=(a+b)(a−b)$

     - 完全平方公式：$(a±b)^2=a^2±2ab+b^2$
     - 立方和公式：$a^3+b^3=(a+b)(a^2−ab+b^2 )$
     - 立方差公式：$a^3−b^3=(a−b)(a^2+ab+b^2 )$
     - 完全立方公式：
        - $(a+b)^3=a^3+b^3+3a^2 b+3ab^2=(a+b)(a^2+2ab+b^2 )$
        - $(a−b)^3=a^3−3a^2 b+3ab^2−b^3=(a−b)(a^2−2ab+b^2 )$
        - $a^3+b^3+c^3−3abc=(a+b+c)(a^2+b^2+c^2−ab−bc−ac)$


##  三、指数函数的性质与图像

  - 定义域和值域：指数函数的定义域是实数集合，即$x ∈ R$，值域是正实数集合。

  - 奇偶性：函数性质既不是奇函数，也不是偶函数。
  - 递增性和递减性：指数函数的递增性取决于底数的大小。
  - 指数函数的反函数：指数函数和对数函数是互为逆函数。
