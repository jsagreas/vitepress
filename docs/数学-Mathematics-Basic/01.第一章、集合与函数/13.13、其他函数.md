## 一、分段函数



**分段函数：**
就是对于自变量 $x$ 的不同的取值范围，有着不同的解析式的函数。它是一个函数，而不是几个函数；分段函数的定义域是各段函数定义域的并集，值域也是各段函数值域的并集。

**常见的分段函数：**
  - 绝对值函数：$f(x) = |x|$，在不同的区间上，绝对值函数的表达式不同。
  - 符号函数：$f(x)$表示自变量的符号。符号函数$\text{sgn}(x)$定义如下：
$$\text{sgn}(x) =\begin{cases} -1, & \text{如果 } x < 0 \\ 0, & \text{如果 } x = 0 \\ 1, & \text{如果 } x > 0 \end{cases}$$


# 二、幂指函数



幂指函数是指幂底数和指数都含有自变量的函数。其表达式为$\ f(x) = u(x)^{v(x)}$,例如：
$$\Large f(x) =x^x$$

# 三、隐函数


- **概念解读：**

     一般来说，一个显式函数可以通过直接解方程来表示，其中因变量可以表示为自变量的函数。例如，对于方程$y=f(x)$，我们可以通过解方程得到显式函数$y$

     然而，有时候方程可能不容易解出因变量，或者方程中包含多个变量之间的关系。在这些情况下，我们可以使用隐函数来描述自变量和因变量之间的关系。

     隐函数的表示形式通常是$F(x,y)=0$其中$F$是一个函数表达式，它将自变量和因变量组合在一起，并使等式成立。通过这个方程，我们可以用自变量表示因变量，或者用因变量表示自变量，但不能直接将因变量表示为自变量的函数形式。
  
     设$F(x,y)=0$是一个含有两个变量$x$和$y$的隐函数方程，如果在$F(x,y)=0$所确定的曲线上，$y$作为$x$的函数是单值的，则称$y$是$x$的隐函数。

- **显函数和隐函数:**

     显函数是指可以直接用一个变量表示另一个变量的函数形式的函数。在显函数中，因变量可以明确地表示为自变量的函数。例如，$y = f(x)$。

     隐函数是指因变量和自变量在同一个方程中，但并不直接表示。需要通过求解方程来“揭示”或“显示”它们之间的关系。在某些情况下，可以使用变量的参数化或换元等方法将隐函数转化为显函数。
     

- **常见的隐函数：**
  - 圆的方程：
$$x^2+y^2=r^2$$
  - 椭圆的方程：
$$\frac{x^2}{a^2}  + \frac{y^2}{b^2} =1$$
  - 求解三角方程：
$$sin(x)+cos(y)=1$$
# 四、参数函数



   - 参数函数是指自变量和因变量都含有参数的函数。设$x$和$y$都是参数$t$的函数，则称以下表达式为参数方程：
$$y=f(x(t),y(t))$$

   - 常见的参数函数：圆的参数方程是：(其中，$a$是圆的半径，$t$是参数。)
$$x=acost,y=asint$$
