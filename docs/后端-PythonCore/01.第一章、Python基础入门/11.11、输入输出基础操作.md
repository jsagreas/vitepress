## 1. 🌐 输入输出概述


### 1.1 什么是输入输出


**💡 简单理解**
```
输入(Input)：程序从外部获取数据
- 就像你跟朋友聊天时，朋友说话给你听(输入)

输出(Output)：程序向外部展示结果  
- 就像你回应朋友的话(输出)

Python程序：
用户输入 → Python处理 → 结果输出
```

**🔸 为什么需要输入输出**
- **交互性**：让程序能和用户对话
- **实用性**：获取用户需求，返回处理结果
- **灵活性**：同一程序可以处理不同的输入数据

### 1.2 Python中的输入输出函数


```
核心函数：
📤 print() → 输出函数，把结果显示给用户看
📥 input() → 输入函数，获取用户输入的内容

简单类比：
print() = 喇叭(向外广播)
input() = 话筒(接收声音)
```

## 2. 📤 print()函数详解


### 2.1 print()基本用法


**🔸 最简单的使用**
```python
print("Hello, World!")  # 输出：Hello, World!
print(123)              # 输出：123
print(3.14)             # 输出：3.14
```

**💭 理解要点**
- `print()`会自动把内容显示在屏幕上
- 可以输出文字(字符串)、数字、各种数据类型
- 每次`print()`后会自动换行

### 2.2 输出多个内容


```python
# 一次输出多个值
print("姓名:", "张三")           # 输出：姓名: 张三
print("年龄:", 25)              # 输出：年龄: 25
print("姓名:", "张三", "年龄:", 25)  # 输出：姓名: 张三 年龄: 25
```

**🔹 工作原理**
- 多个值用逗号`,`分隔
- Python会自动在每个值之间加空格
- 就像说话时的自然停顿

### 2.3 print()重要参数详解


#### 🎯 sep参数 - 分隔符


**作用**：控制多个值之间的分隔符

```python
# 默认用空格分隔
print("苹果", "香蕉", "橘子")                    # 输出：苹果 香蕉 橘子

# 用-分隔  
print("苹果", "香蕉", "橘子", sep="-")           # 输出：苹果-香蕉-橘子

# 用其他符号分隔
print("2025", "09", "21", sep="/")              # 输出：2025/09/21
print("姓名", "张三", sep=":")                   # 输出：姓名:张三
```

**💡 实用场景**
```python
# 格式化日期
year, month, day = 2025, 9, 21
print(year, month, day, sep="-")                # 输出：2025-9-21

# 创建分隔线
print("*", "*", "*", "*", "*", sep="")          # 输出：*****
```

#### 🎯 end参数 - 结尾符


**作用**：控制print()结束时添加什么

```python
# 默认会换行
print("第一行")
print("第二行")
# 输出：
# 第一行
# 第二行

# 不换行，在同一行输出
print("Hello", end=" ")
print("World")          # 输出：Hello World

# 用其他符号结尾
print("加载中", end="...")
print("完成")           # 输出：加载中...完成
```

**💡 实用场景**
```python
# 创建进度条效果
print("下载进度: ", end="")
for i in range(5):
    print("█", end="")  # 输出：下载进度: █████

# 同一行显示多个相关信息
print("温度: 25°C", end=" | ")
print("湿度: 60%", end=" | ")  
print("风速: 3m/s")    # 输出：温度: 25°C | 湿度: 60% | 风速: 3m/s
```

#### 🎯 file参数 - 输出目标


**作用**：控制输出到哪里

```python
# 输出到文件
with open("log.txt", "w", encoding="utf-8") as f:
    print("这是日志信息", file=f)

# 输出到错误流
import sys
print("这是错误信息", file=sys.stderr)
```

#### 🎯 flush参数 - 立即输出


**作用**：强制立即显示内容

```python
import time

# 正常情况可能有延迟
for i in range(3):
    print(f"倒计时: {3-i}", end=" ")
    time.sleep(1)

# 使用flush立即显示
for i in range(3):
    print(f"倒计时: {3-i}", end=" ", flush=True)
    time.sleep(1)  # 每秒立即显示一个数字
```

### 2.4 print()格式化输出


**🔸 f-string格式化（推荐）**
```python
name = "小明"
age = 18
score = 95.5

# 清晰直观的格式化
print(f"姓名: {name}")                    # 输出：姓名: 小明
print(f"年龄: {age}岁")                   # 输出：年龄: 18岁
print(f"{name}今年{age}岁，考了{score}分")  # 输出：小明今年18岁，考了95.5分

# 格式控制
print(f"分数: {score:.1f}")               # 输出：分数: 95.5 (1位小数)
print(f"百分比: {score:.0f}%")            # 输出：百分比: 96% (整数)
```

**🔸 .format()方法**
```python
print("姓名: {}，年龄: {}".format(name, age))
print("姓名: {0}，年龄: {1}".format(name, age))
```

## 3. 📥 input()用户输入


### 3.1 input()基本用法


**🔸 获取用户输入**
```python
# 基本用法
user_input = input()        # 等待用户输入
print("你输入的是:", user_input)

# 带提示信息
name = input("请输入你的姓名: ")
print(f"你好, {name}!")
```

**💭 工作流程**
```
程序运行到input() → 暂停等待 → 用户输入内容 → 按回车确认 → 继续执行
```

### 3.2 input()重要特点


**🔸 返回值永远是字符串**
```python
# 即使输入数字，也是字符串类型
age_str = input("请输入年龄: ")  # 用户输入：18
print(type(age_str))            # 输出：<class 'str'>
print(age_str)                  # 输出：18 (字符串的18)
```

**🔸 提示信息的重要性**
```python
# 不好的提示
data = input()  # 用户不知道要输入什么

# 好的提示
name = input("请输入姓名: ")
age = input("请输入年龄(数字): ")
email = input("请输入邮箱地址: ")
```

### 3.3 处理不同类型的输入


**🔸 多行输入提示**
```python
print("请输入个人信息:")
name = input("姓名: ")
age = input("年龄: ")
city = input("城市: ")

print("\n=== 信息确认 ===")
print(f"姓名: {name}")
print(f"年龄: {age}")  
print(f"城市: {city}")
```

**🔸 输入验证提示**
```python
# 给用户明确的输入要求
phone = input("请输入手机号(11位数字): ")
password = input("请输入密码(至少6位): ")
```

## 4. 🔄 数据类型转换与验证


### 4.1 为什么需要类型转换


**💭 理解核心问题**
```python
# 问题演示
age_str = input("请输入年龄: ")  # 用户输入：18
next_year = age_str + 1          # ❌ 错误！字符串不能和数字相加

# 正确做法
age_int = int(age_str)          # 转换为数字
next_year = age_int + 1         # ✅ 正确！数字可以相加
```

### 4.2 常用类型转换


**🔸 转换为整数**
```python
# 基本转换
age_str = input("请输入年龄: ")
age = int(age_str)
print(f"明年你就{age + 1}岁了")

# 一行完成
age = int(input("请输入年龄: "))
```

**🔸 转换为浮点数**
```python
# 处理小数
height_str = input("请输入身高(米): ")  # 用户输入：1.75
height = float(height_str)
print(f"你的身高是{height}米")
```

**🔸 类型转换对照表**

| **函数** | **作用** | **示例输入** | **转换结果** | **用途** |
|---------|---------|-------------|-------------|---------|
| `int()` | 转整数 | `"18"` | `18` | 年龄、数量等 |
| `float()` | 转小数 | `"3.14"` | `3.14` | 价格、身高等 |
| `str()` | 转字符串 | `18` | `"18"` | 显示、拼接等 |
| `bool()` | 转布尔值 | `"True"` | `True` | 判断条件等 |

### 4.3 输入验证与异常处理


**🔸 基本异常处理**
```python
# 安全的数字输入
try:
    age = int(input("请输入年龄: "))
    print(f"你的年龄是{age}岁")
except ValueError:
    print("❌ 请输入有效的数字！")
```

**🔸 循环验证输入**
```python
# 持续验证直到输入正确
while True:
    try:
        age = int(input("请输入年龄(数字): "))
        if age > 0:
            print(f"✅ 输入成功！你的年龄是{age}岁")
            break
        else:
            print("❌ 年龄必须大于0，请重新输入")
    except ValueError:
        print("❌ 请输入有效的数字！")
```

**🔸 输入范围验证**
```python
# 验证年龄范围
def get_valid_age():
    while True:
        try:
            age = int(input("请输入年龄(1-120): "))
            if 1 <= age <= 120:
                return age
            else:
                print("❌ 年龄范围应在1-120之间")
        except ValueError:
            print("❌ 请输入有效的数字")

age = get_valid_age()
print(f"年龄验证通过: {age}岁")
```

## 5. 🔄 标准流与重定向


### 5.1 理解标准流


**💡 什么是标准流**
```
标准输入(stdin)：程序接收输入的通道
标准输出(stdout)：程序正常输出的通道  
标准错误(stderr)：程序错误信息的通道

类比：
stdin  = 耳朵(听)
stdout = 嘴巴(说)
stderr = 警报器(报错)
```

### 5.2 标准输出流详解


**🔸 默认输出**
```python
import sys

# 这些是等价的
print("Hello")
print("Hello", file=sys.stdout)  # 明确指定输出到标准输出
```

**🔸 错误输出流**
```python
import sys

# 输出错误信息
print("这是正常信息")                    # 标准输出
print("这是错误信息", file=sys.stderr)   # 错误输出

# 实际应用
def divide(a, b):
    if b == 0:
        print("❌ 错误：除数不能为0", file=sys.stderr)
        return None
    return a / b
```

### 5.3 输出重定向


**🔸 重定向到文件**
```python
# 输出到文件而不是屏幕
with open("output.txt", "w", encoding="utf-8") as f:
    print("这行会写入文件", file=f)
    print("这行也会写入文件", file=f)

# 同时输出到屏幕和文件
def log_print(message):
    print(message)  # 屏幕显示
    with open("log.txt", "a", encoding="utf-8") as f:
        print(message, file=f)  # 同时记录到文件
```

**🔸 命令行重定向**
```bash
# 在命令行中运行Python时
python script.py > output.txt      # 重定向标准输出到文件
python script.py 2> error.txt      # 重定向错误输出到文件
python script.py > all.txt 2>&1    # 重定向所有输出到文件
```

def collect_user_info():

### 6.2 简单计算器


```python
def simple_calculator():
    """简单的计算器程序"""
    print("=== 简单计算器 ===")
    print("支持运算: + - * /")
    
    try:
        # 获取输入
        num1 = float(input("请输入第一个数字: "))
        operator = input("请输入运算符(+, -, *, /): ")
        num2 = float(input("请输入第二个数字: "))
        
        # 计算结果
        if operator == '+':
            result = num1 + num2
        elif operator == '-':
            result = num1 - num2
        elif operator == '*':
            result = num1 * num2
        elif operator == '/':
            if num2 == 0:
                print("❌ 错误：除数不能为0")
                return
            result = num1 / num2
        else:
            print("❌ 不支持的运算符")
            return
        
        # 输出结果
        print(f"计算结果: {num1} {operator} {num2} = {result}")
        
    except ValueError:
        print("❌ 请输入有效的数字")

simple_calculator()
```

### 6.3 进度显示程序


```python
import time

def show_progress():
    """显示进度条"""
    print("开始下载文件...")
    
    total = 50
    for i in range(total + 1):
        # 计算进度百分比
        percent = (i / total) * 100
        
        # 创建进度条
        bar_length = 20
        filled_length = int(bar_length * i // total)
        bar = '█' * filled_length + '-' * (bar_length - filled_length)
        
        # 显示进度(不换行)
        print(f'\r进度: |{bar}| {percent:.1f}%', end='', flush=True)
        
        # 模拟下载时间
        time.sleep(0.1)
    
    print("\n✅ 下载完成！")

show_progress()
```

## 7. 📋 核心要点总结


### 7.1 必须掌握的核心概念


```
🔸 print()函数：向用户显示信息的工具
🔸 input()函数：获取用户输入的工具
🔸 类型转换：input()返回字符串，需要转换为需要的类型
🔸 异常处理：防止用户输入错误导致程序崩溃
🔸 标准流：程序与外界交互的三个通道
```

### 7.2 关键理解要点


**🔹 print()参数的实际作用**
```
sep参数：控制多个值之间用什么分隔
end参数：控制每次print()结尾加什么
file参数：控制输出到哪里(屏幕/文件)
flush参数：控制是否立即显示
```

**🔹 input()的核心特点**
```
返回类型：永远是字符串(str)
用户体验：一定要有清晰的提示信息
数据处理：通常需要类型转换和验证
```

**🔹 类型转换的必要性**
```
为什么需要：input()返回字符串，但计算需要数字
如何转换：int()转整数，float()转小数
如何安全：使用try-except处理转换错误
```

### 7.3 实际应用指导


**✅ 最佳实践**
- 给用户清晰的输入提示
- 对用户输入进行验证
- 使用异常处理防止程序崩溃
- 合理使用print()参数美化输出

**❌ 常见错误**
- 忘记类型转换，直接用字符串做数值运算
- 没有输入验证，程序容易出错
- 提示信息不清楚，用户不知道怎么输入

### 7.4 记忆要点


**💡 简单记忆法**
```
print() = 说话(输出)
input() = 听话(输入) 
input()的结果 = 字符串(需要转换)
异常处理 = 防护网(防止出错)
```

**🎯 核心流程**
```
1. 用print()给用户提示
2. 用input()获取用户输入  
3. 转换数据类型(如果需要)
4. 处理和计算数据
5. 用print()显示结果
```

**📝 实践建议**
- 多练习不同类型的输入输出
- 学会处理用户的错误输入
- 重视用户体验，提供友好的交互界面
- 掌握格式化输出，让结果更美观
