## 1. 🎯 输入输出函数概述


### 1.1 什么是输入输出


**🔸 简单理解**
```
输出(Output)：程序向外界展示信息
就像你对朋友说话 → 朋友听到了

输入(Input)：程序从外界接收信息  
就像朋友对你说话 → 你听到了

Python程序 ↔ 用户交互
```

**💡 为什么需要输入输出**
- **输出**：程序需要告诉用户运行结果
- **输入**：程序需要用户提供数据才能工作
- **交互**：让程序变得有用，不再是"哑巴"程序

### 1.2 Python的标准输入输出


**🔸 核心概念**
```
标准输出：print() → 在屏幕上显示内容
标准输入：input() → 从键盘获取用户输入

这两个函数是Python最基础、最常用的交互方式
```

## 2. 📺 print()函数详解


### 2.1 print()的基本作用


**🔸 最简单的理解**
`print()` 就是让Python"说话"，把内容显示在屏幕上

```python
print("Hello World")  # Python对你说：Hello World
```

### 2.2 print()的基本用法


**📝 基础语法**
```python
print(值1, 值2, 值3, ...)
```

**💻 简单示例**
```python
# 打印文字
print("你好，Python!")

# 打印数字
print(123)
print(3.14)

# 打印多个内容
print("我的年龄是", 25)
```

### 2.3 print()的重要参数


#### 🔹 sep参数 - 连接符


**作用**：控制多个值之间用什么连接

```python
# 默认用空格连接
print("苹果", "香蕉", "橘子")
# 输出：苹果 香蕉 橘子

# 用逗号连接
print("苹果", "香蕉", "橘子", sep=",")
# 输出：苹果,香蕉,橘子

# 用横线连接
print("2024", "09", "21", sep="-")
# 输出：2024-09-21
```

#### 🔹 end参数 - 结尾符


**作用**：控制打印完成后在末尾加什么

```python
# 默认换行
print("第一行")
print("第二行")
# 输出：
# 第一行
# 第二行

# 不换行，用空格结尾
print("姓名:", end=" ")
print("张三")
# 输出：姓名: 张三

# 用其他符号结尾
print("加载中", end="...")
# 输出：加载中...
```

### 2.4 print()输出格式化


**🔸 为什么需要格式化**
有时候我们需要把变量的值嵌入到句子中，让输出更自然

#### 📋 方法1：f字符串（推荐）


```python
name = "小明"
age = 20
score = 85.5

# f字符串：在引号前加f，变量用{}包围
print(f"学生{name}今年{age}岁，考试得了{score}分")
# 输出：学生小明今年20岁，考试得了85.5分
```

#### 📋 方法2：format()方法


```python
name = "小红"
age = 18

# 用{}作占位符，然后用format填入值
print("学生{}今年{}岁".format(name, age))
# 输出：学生小红今年18岁
```

#### 📋 方法3：%格式化（了解即可）


```python
name = "小李"
age = 22

# %s表示字符串，%d表示整数
print("学生%s今年%d岁" % (name, age))
# 输出：学生小李今年22岁
```

## 3. ⌨️ input()函数详解


### 3.1 input()的基本作用


**🔸 简单理解**
`input()` 让程序"听用户说话"，等待用户输入内容

```python
name = input("请输入您的姓名：")
print(f"你好，{name}!")
```

### 3.2 input()的工作机制


**🔄 执行流程**
```
1. 程序执行到input()时会暂停
2. 显示提示信息（如果有的话）
3. 等待用户输入并按回车
4. 把用户输入的内容作为字符串返回
5. 程序继续执行
```

**💡 重要提醒**
> ⚠️ **注意**：input()获取的**总是字符串**，即使用户输入数字！

### 3.3 处理不同类型的输入


#### 🔸 获取文字


```python
name = input("请输入您的姓名：")
print(f"欢迎您，{name}!")
```

#### 🔸 获取数字


```python
# 错误方式 - 直接使用
age = input("请输入您的年龄：")  # 这里age是字符串
# 如果用户输入"20"，age的值是字符串"20"，不是数字20

# 正确方式 - 转换类型
age = int(input("请输入您的年龄："))  # 转换为整数
height = float(input("请输入您的身高(米)："))  # 转换为小数
```

#### 🔸 输入验证的安全处理


```python
# 安全的数字输入
try:
    age = int(input("请输入您的年龄："))
    print(f"您{age}岁了")
except ValueError:
    print("请输入有效的数字!")
```

### 3.4 input()的高级用法


#### 📝 带默认值的输入


```python
name = input("请输入您的姓名(默认为'匿名用户')：") or "匿名用户"
print(f"你好，{name}!")
```

#### 📝 多行输入处理


```python
print("请输入您的个人信息：")
name = input("姓名：")
age = int(input("年龄："))
city = input("城市：")

print(f"\n您的信息：\n姓名：{name}\n年龄：{age}\n城市：{city}")
```

## 5. 🔧 常见问题与解决方案


### 5.1 类型转换问题


**❌ 常见错误**
```python
age = input("年龄：")  # 用户输入"20"
next_year = age + 1    # 错误！不能把字符串和数字相加
```

**✅ 正确做法**
```python
age = int(input("年龄："))  # 转换为整数
next_year = age + 1        # 正确！
print(f"明年您就{next_year}岁了")
```

### 5.2 输入验证


**🛡️ 防止用户输入错误**
```python
while True:
    try:
        age = int(input("请输入年龄(数字)："))
        if age > 0:
            break
        else:
            print("年龄必须大于0")
    except ValueError:
        print("请输入有效的数字")

print(f"您的年龄是{age}岁")
```

### 5.3 打印格式问题


**📋 对比不同的输出效果**

| 需求 | 代码 | 输出效果 |
|------|------|----------|
| 普通拼接 | `print("姓名:" + name)` | 姓名:张三 |
| 带空格 | `print("姓名:", name)` | 姓名: 张三 |
| f字符串 | `print(f"姓名:{name}")` | 姓名:张三 |
| 格式化对齐 | `print(f"姓名:{name:>10}")` | 姓名:      张三 |

## 6. 📋 核心要点总结


### 6.1 必须掌握的核心概念


```
🔸 print()：程序向用户输出信息的窗口
🔸 input()：程序从用户获取信息的入口
🔸 字符串转换：input()总是返回字符串，需要时转换类型
🔸 格式化输出：f字符串是最推荐的方式
🔸 参数控制：sep和end参数可以控制输出格式
```

### 6.2 实用技巧总结


**💡 print()使用技巧**
- 使用 `f字符串` 进行格式化输出
- 用 `sep参数` 控制多个值的连接方式
- 用 `end参数` 控制行尾的处理

**💡 input()使用技巧**
- 记住input()返回的总是字符串
- 需要数字时用int()或float()转换
- 重要输入要做错误处理

**💡 安全编程**
- 对用户输入进行验证
- 使用try-except处理转换错误
- 提供清晰的提示信息
