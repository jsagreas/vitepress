## 1. 🎯 运算符概念入门


### 1.1 什么是运算符


**简单理解**：运算符就像数学中的加减乘除符号，用来对数字进行各种计算操作。

```
日常生活中的运算：
买苹果：3个 + 2个 = 5个  (这里的"+"就是运算符)
分蛋糕：8块 ÷ 4人 = 2块  (这里的"÷"就是运算符)

Python中的运算：
price = 10 + 5    # "+"是加法运算符
result = 20 / 4   # "/"是除法运算符
```

### 1.2 运算符的基本构成


**🔸 运算符的三要素**
```
运算符表达式构成：
操作数1  运算符  操作数2  =  结果

例子解析：
5    +    3    =   8
↓    ↓    ↓        ↓
左   运算  右      计算
操作数 符  操作数   结果
```

**🔸 运算符分类一览**
```
按功能分类：
🔢 算术运算符：+、-、*、/、//、%、**
📊 比较运算符：>、<、>=、<=、==、!=
🔗 逻辑运算符：and、or、not
📝 赋值运算符：=、+=、-=、*=、/=

今天重点学习：算术运算符 🎯
```

### 1.3 为什么要学运算符


**💡 实际应用场景**
```
🛒 购物场景：
total_price = item1_price + item2_price + tax

🎮 游戏场景：
remaining_hp = current_hp - damage_taken

📊 数据分析：
average_score = total_score / student_count

🏗️ 工程计算：
area = length * width
```

## 2. ➕ 基本算术运算符


### 2.1 加法运算符（+）


**🔸 基本用法**
```python
# 数字相加
result = 5 + 3        # 结果：8
price = 12.5 + 7.8    # 结果：20.3

# 字符串"相加"（实际是拼接）
greeting = "Hello" + " World"    # 结果："Hello World"
name = "张" + "三"              # 结果："张三"
```

**💡 新手易错点**
```python
# ❌ 错误：不同类型不能直接相加
# result = 5 + "3"    # 会报错！

# ✅ 正确：先转换类型再相加
result = 5 + int("3")       # 结果：8
result = str(5) + "3"       # 结果："53"
```

### 2.2 减法运算符（-）


**🔸 基本用法**
```python
# 普通减法
difference = 10 - 3      # 结果：7
balance = 1000 - 250     # 结果：750

# 负数表示
negative = -5            # 表示负5
temp = -(10 + 3)        # 结果：-13
```

**🎯 实际应用**
```python
# 计算年龄
current_year = 2025

age = current_year - birth_year    # 结果：35

# 计算剩余数量
total_items = 100
sold_items = 35
remaining = total_items - sold_items    # 结果：65
```

### 2.3 乘法运算符（*）


**🔸 基本用法**
```python
# 数字相乘
area = 5 * 3           # 结果：15

# 字符串重复
repeat_str = "Hi! " * 3    # 结果："Hi! Hi! Hi! "
```

**🔧 实用技巧**
```python
# 创建分隔线
print("=" * 50)

# 创建空格缩进
```

**🔸 基本用法**

## 3. 🔧 特殊算术运算符


### 3.1 整除运算符（//）


**🔸 什么是整除**

整除就是"只要整数部分，小数部分直接丢掉"的除法。

```python
# 普通除法 vs 整除
print(7 / 2)      # 结果：3.5
print(7 // 2)     # 结果：3（丢掉小数部分）

print(15 / 4)     # 结果：3.75  
print(15 // 4)    # 结果：3
```

**💡 直观理解**
```
想象分苹果：
15个苹果分给4个人
- 普通除法：每人3.75个（不现实）
- 整除：每人3个（实际能分到的完整苹果）
```

**🎯 实际应用**
```python
# 计算需要多少页显示数据
total_items = 25
items_per_page = 10
full_pages = total_items // items_per_page    # 结果：2页

# 计算小时数
total_minutes = 150
hours = total_minutes // 60    # 结果：2小时
```

### 3.2 取余运算符（%）


**🔸 什么是取余**

取余就是"除法后剩下的部分"，也叫求余数。

```python
# 基本取余操作
print(7 % 2)      # 结果：1（7÷2=3余1）
print(15 % 4)     # 结果：3（15÷4=3余3）
print(10 % 3)     # 结果：1（10÷3=3余1）
```

**📊 取余运算图解**
```
15 ÷ 4 的过程：
┌─────────────────┐
│ 15 ÷ 4 = 3 余 3│
│                 │
│ 15 = 4×3 + 3   │
│      ↑     ↑   │
│    商数   余数  │
└─────────────────┘

所以：15 % 4 = 3
```

**🎯 取余的神奇用途**

**判断奇偶数**
```python
def is_even(number):
    """判断数字是否为偶数"""
    return number % 2 == 0

print(is_even(4))    # True（偶数）
print(is_even(5))    # False（奇数）
```

**循环索引**
```python
# 轮转显示（比如轮播图）
items = ["图片1", "图片2", "图片3"]
current_index = 5
show_item = items[current_index % 3]    # 显示"图片3"
```

### 3.3 幂运算符（**）


**🔸 什么是幂运算**

幂运算就是"连续相乘"，也叫乘方。

```python
# 基本幂运算
print(2 ** 3)     # 结果：8（2×2×2）
print(5 ** 2)     # 结果：25（5×5）
print(3 ** 4)     # 结果：81（3×3×3×3）
```

**🧮 幂运算的直观理解**
```
2 ** 3 的含义：
2 ** 3 = 2 × 2 × 2 = 8
         ↑   ↑   ↑
      第1个 第2个 第3个

数学表示：2³ = 8
```

**💡 特殊情况**
```python
# 任何数的0次方都是1
print(5 ** 0)     # 结果：1
print(100 ** 0)   # 结果：1

# 1的任何次方都是1
print(1 ** 100)   # 结果：1

# 平方根（分数次方）
print(9 ** 0.5)   # 结果：3.0（9的平方根）
print(8 ** (1/3)) # 结果：2.0（8的立方根）
```

### 3.4 divmod() 函数


**🔸 一次性获取商和余数**

`divmod()` 函数可以同时计算整除和取余的结果。

```python
# 基本用法
quotient, remainder = divmod(15, 4)
print(f"商：{quotient}, 余数：{remainder}")
# 输出：商：3, 余数：3

# 等价于分别计算
quotient = 15 // 4    # 3
remainder = 15 % 4    # 3
```

**🎯 实际应用场景**
```python
# 时间转换：将秒数转换为分钟和秒
total_seconds = 150
minutes, seconds = divmod(total_seconds, 60)
print(f"{minutes}分{seconds}秒")    # 输出：2分30秒

# 金额转换：将分转换为元和分
total_cents = 1537
yuan, cents = divmod(total_cents, 100)
print(f"{yuan}元{cents}分")         # 输出：15元37分
```

## 4. ⚖️ 运算符优先级与结合性


### 4.1 什么是运算符优先级


**🎯 简单理解**

优先级就是"谁先算"的规则，就像数学中"先乘除后加减"一样。

```python
# 没有括号的情况
result = 2 + 3 * 4    # 结果：14，不是20
# 计算过程：2 + (3 * 4) = 2 + 12 = 14

# 用括号改变优先级
result = (2 + 3) * 4  # 结果：20
# 计算过程：(2 + 3) * 4 = 5 * 4 = 20
```

### 4.2 Python算术运算符优先级表


| **优先级** | **运算符** | **名称** | **结合性** |
|-----------|-----------|---------|-----------|
| `最高` | `**` | 幂运算 | **右结合** |
| `高` | `-x`, `+x` | 正负号 | **左结合** |
| `中高` | `*`, `/`, `//`, `%` | 乘除取余 | **左结合** |
| `中低` | `+`, `-` | 加减 | **左结合** |

**🔍 优先级记忆口诀**
```
幂运算最高要先算，
正负号次之不能忘，
乘除取余排中间，
加减运算垫后方。
```

### 4.3 结合性详解


**🔸 左结合性（大多数运算符）**
```python
# 加法的左结合性
result = 10 - 5 - 2
# 计算过程：(10 - 5) - 2 = 5 - 2 = 3

# 除法的左结合性  
result = 20 / 4 / 2
# 计算过程：(20 / 4) / 2 = 5.0 / 2 = 2.5
```

**🔸 右结合性（幂运算）**
```python
# 幂运算的右结合性
result = 2 ** 3 ** 2
# 计算过程：2 ** (3 ** 2) = 2 ** 9 = 512
# 不是：(2 ** 3) ** 2 = 8 ** 2 = 64

# 为什么是右结合？
# 数学中：2^(3^2) 是常见写法
# 表示：2的(3的2次方)次方
```

### 4.4 复杂表达式计算步骤


**🧮 计算示例**
```python
result = 2 + 3 * 4 ** 2 - 10 // 3
```

**📝 一步步计算过程**
```
原式：2 + 3 * 4 ** 2 - 10 // 3

第1步：先算幂运算（最高优先级）
= 2 + 3 * 16 - 10 // 3

第2步：再算乘法和整除（同级，从左到右）
= 2 + 48 - 3

第3步：最后算加减（从左到右）
= 50 - 3
= 47
```

**🔧 验证计算**
```python
# Python计算验证
result = 2 + 3 * 4 ** 2 - 10 // 3
print(result)    # 输出：47

# 分步验证
step1 = 4 ** 2         # 16
step2 = 3 * step1      # 48  
step3 = 10 // 3        # 3
final = 2 + step2 - step3    # 47
```

## 5. 🧮 实用数学函数


### 5.1 abs() - 绝对值函数


**🔸 什么是绝对值**

绝对值就是"数字到0的距离"，总是正数。

```python
# 基本用法
print(abs(5))     # 结果：5
print(abs(-5))    # 结果：5
print(abs(-3.7))  # 结果：3.7
print(abs(0))     # 结果：0
```

**💡 生活中的绝对值**
```python
# 计算温度差（不关心谁大谁小）
temp1 = 25
temp2 = 18
difference = abs(temp1 - temp2)    # 结果：7度

# 计算距离（不关心方向）
position1 = 10
position2 = 3
distance = abs(position1 - position2)    # 结果：7
```

### 5.2 round() - 四舍五入函数


**🔸 基本四舍五入**
```python
# 四舍五入到整数
print(round(3.7))     # 结果：4
print(round(3.3))     # 结果：3
print(round(3.5))     # 结果：4

# 指定小数位数
print(round(3.14159, 2))    # 结果：3.14
print(round(3.14159, 3))    # 结果：3.142
```

**⚠️ 银行家舍入规则**
```python
# Python使用"银行家舍入"
print(round(2.5))     # 结果：2（不是3！）
print(round(3.5))     # 结果：4
print(round(4.5))     # 结果：4（不是5！）

# 规则：.5时舍入到最近的偶数
```

**🎯 实际应用**
```python
# 金额计算
price = 19.996
final_price = round(price, 2)    # 结果：20.0

# 成绩计算
score = 87.6666
final_score = round(score, 1)    # 结果：87.7
```

### 5.3 pow() - 幂运算函数


**🔸 两种幂运算方式**
```python
# 方式1：使用 ** 运算符
result1 = 2 ** 3      # 结果：8

# 方式2：使用 pow() 函数
result2 = pow(2, 3)   # 结果：8

# 两种方式结果相同
print(result1 == result2)    # True
```

**🔧 pow() 的特殊功能**
```python
# 带模运算的幂（加密算法中常用）
result = pow(2, 3, 5)    # 等价于 (2 ** 3) % 5
print(result)            # 结果：3

# 计算过程：
# 2 ** 3 = 8
# 8 % 5 = 3
```

### 5.4 数学函数组合应用


**🎯 综合计算示例**


## 6. 🎯 运算符实战应用


### 6.1 购物车价格计算


```python
def calculate_shopping_cart():
    """购物车价格计算系统"""
    
    # 商品信息
    items = [
        {"name": "苹果", "price": 6.8, "quantity": 3},
        {"name": "牛奶", "price": 12.5, "quantity": 2},
        {"name": "面包", "price": 8.0, "quantity": 1}
    ]
    
    total = 0
    print("📦 购物清单：")
    print("-" * 30)
    
    for item in items:
        subtotal = item["price"] * item["quantity"]
        total += subtotal
        print(f"{item['name']: <6} {item['quantity']}个 × {item['price']}元 = {subtotal}元")
    
    # 计算折扣
    if total >= 50:
        discount = total * 0.1  # 9折
        final_total = total - discount
        print(f"\n💰 原价：{round(total, 2)}元")
        print(f"🎉 满50减免：{round(discount, 2)}元")
        print(f"✅ 实付：{round(final_total, 2)}元")
    else:
        print(f"\n💰 总计：{round(total, 2)}元")

# 运行示例
calculate_shopping_cart()
```

### 6.2 时间格式转换器


```python
def format_seconds(total_seconds):
    """将秒数转换为时分秒格式"""
    
    # 计算小时、分钟、秒
    hours, remainder = divmod(total_seconds, 3600)  # 1小时=3600秒
    minutes, seconds = divmod(remainder, 60)        # 1分钟=60秒
    
    # 格式化输出
    if hours > 0:
        return f"{hours}小时{minutes}分{seconds}秒"
    elif minutes > 0:
        return f"{minutes}分{seconds}秒"
    else:
        return f"{seconds}秒"

# 测试示例
test_cases = [45, 125, 3661, 7325]
for seconds in test_cases:
    formatted = format_seconds(seconds)
    print(f"{seconds}秒 = {formatted}")

# 输出：
# 45秒 = 45秒
# 125秒 = 2分5秒  
# 3661秒 = 1小时1分1秒
# 7325秒 = 2小时2分5秒
```

### 6.3 分页器实现


```python
def create_paginator(total_items, items_per_page):
    """创建分页器"""
    
    # 计算总页数
    total_pages = (total_items + items_per_page - 1) // items_per_page
    # 或者使用：total_pages = -(-total_items // items_per_page)
    
    print(f"📄 分页信息:")
    print(f"总记录数: {total_items}")
    print(f"每页显示: {items_per_page}")
    print(f"总页数: {total_pages}")
    print("-" * 25)
    
    # 显示每页的记录范围
    for page in range(1, total_pages + 1):
        start_item = (page - 1) * items_per_page + 1
        end_item = min(page * items_per_page, total_items)
        print(f"第{page}页: 记录{start_item}-{end_item}")

# 测试示例
create_paginator(total_items=47, items_per_page=10)

# 输出：
# 📄 分页信息:
# 总记录数: 47
# 每页显示: 10
# 总页数: 5
# -------------------------
# 第1页: 记录1-10
# 第2页: 记录11-20
# 第3页: 记录21-30
# 第4页: 记录31-40
# 第5页: 记录41-47
```

### 6.4 简单密码强度检测


```python
def check_password_strength(password):
    """检测密码强度"""
    
    score = 0
    feedback = []
    
    # 长度检查
    length = len(password)
    if length >= 8:
        score += 2
        feedback.append("✅ 长度充足")
    else:
        feedback.append("❌ 长度至少8位")
    
    # 字符类型检查
    has_digit = any(char.isdigit() for char in password)
    has_lower = any(char.islower() for char in password)  
    has_upper = any(char.isupper() for char in password)
    
    if has_digit:
        score += 1
        feedback.append("✅ 包含数字")
    
    if has_lower and has_upper:
        score += 2
        feedback.append("✅ 大小写混合")
    elif has_lower or has_upper:
        score += 1
        feedback.append("⚠️ 建议大小写混合")
    
    # 强度评级
    if score >= 5:
        strength = "强"
        emoji = "🔒"
    elif score >= 3:
        strength = "中"
        emoji = "🔐"
    else:
        strength = "弱"
        emoji = "🔓"
    
    print(f"{emoji} 密码强度: {strength} (得分: {score}/5)")
    for tip in feedback:
        print(f"  {tip}")

# 测试示例
test_passwords = ["123456", "password", "Password123", "StrongPass2024!"]
for pwd in test_passwords:
    print(f"\n密码: {'*' * len(pwd)}")
    check_password_strength(pwd)
```

## 7. 📋 核心要点总结


### 7.1 运算符使用要点


**🔸 基本算术运算符记忆**
```
+ 加法：数字相加，字符串拼接
- 减法：数字相减，也可表示负数
* 乘法：数字相乘，字符串重复
/ 除法：总是返回浮点数
// 整除：只保留整数部分
% 取余：返回除法的余数
** 幂运算：连续相乘
```

**🔸 特殊运算符的核心用途**

| 运算符 | **最佳用途** | **记忆方法** |
|-------|-------------|-------------|
| `//` | 分页计算、时间转换 | "要整数商就用它" |
| `%` | 判断奇偶、循环索引 | "要余数就用它" |
| `**` | 面积体积计算、科学计算 | "要乘方就用它" |
| `divmod()` | 同时要商和余数 | "一次性解决两个问题" |

### 7.2 优先级记忆技巧


**🧠 简单记忆法**
```
📝 优先级口诀：
括号最优先，
幂运算第二，
乘除取余中间站，
加减运算最后算。

🔧 实用技巧：
不确定的时候就加括号！
(a + b) * c  比  a + b * c  更清晰
```

### 7.3 常见错误与避免方法


**❌ 新手常见错误**
```python
# 错误1：类型混用
# age = "20" + 5    # TypeError!
age = int("20") + 5  # ✅ 正确

# 错误2：整除理解错误  
# 以为 7 // 2 = 3.5  # ❌ 实际是 3
print(7 // 2)        # ✅ 正确理解：3

# 错误3：优先级搞错
# 以为 2 + 3 * 4 = 20  # ❌ 实际是 14
print((2 + 3) * 4)    # ✅ 想要20就加括号
```

**✅ 最佳实践建议**
```python
# 1. 复杂表达式要加括号
total = (price + tax) * quantity

# 2. 浮点数比较要小心
if abs(a - b) < 0.0001:  # 而不是 a == b

# 3. 除零要检查
if denominator != 0:
    result = numerator / denominator

# 4. 使用合适的函数
distance = abs(x1 - x2)      # 绝对值
rounded = round(price, 2)    # 四舍五入
```
